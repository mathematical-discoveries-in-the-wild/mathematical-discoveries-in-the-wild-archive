# Literature-implied answer: resolvent metastability subcase for arXiv:1310.6874

Status: literature_implied_answer (uniformly convex and uniformly smooth subcase; arbitrary uniformly smooth case remains open).

Source paper: Daniel Kornlein, *Quantitative results for Halpern iterations of nonexpansive mappings*, arXiv:1310.6874; J. Math. Anal. Appl. 428 (2015), no. 2, 1161--1172, DOI `10.1016/j.jmaa.2015.03.020`.

Supporting paper: Ulrich Kohlenbach and Andrei Sipos, *The finitary content of sunny nonexpansive retractions*, arXiv:1812.04940; Comm. Contemp. Math. 23 (2021), no. 3, 1950093.

## Source Signal

In the introduction of arXiv:1310.6874, Kornlein extracts a rate of metastability for Halpern iterations in uniformly smooth Banach spaces relative to a rate of metastability for the resolvent path `(z_t)`. The source then states that extracting such a rate for `(z_t)` in the uniformly smooth case is still open, and explicitly notes that this would include `L_p` spaces for `1<p<infty`, `p != 2`.

## Later Identification

Kohlenbach--Sipos extract a uniform rate of metastability for the Reich approximating/resolvent curve in Banach spaces that are both uniformly convex and uniformly smooth. They explicitly say this covers all `L_p` spaces for `1<p<infty`, and that their result gives the first explicit rate of metastability for the Shioji--Takahashi extension of Wittmann's Halpern theorem to this class. In Section 7, they spell out the Halpern iteration consequence using the earlier Kohlenbach--Leustean transformation.

Thus the source open problem is answered for the important `L_p` subcase and, more generally, for uniformly convex plus uniformly smooth Banach spaces. This is an agent-identified literature implication, not a new proof.

## Scope

The arbitrary uniformly smooth case from arXiv:1310.6874 is not fully settled by this packet, because Kohlenbach--Sipos assume uniform convexity in addition to uniform smoothness. The packet records the partial literature answer and prevents future agents from treating the `L_p` / uniformly convex-and-smooth subcase as open.
