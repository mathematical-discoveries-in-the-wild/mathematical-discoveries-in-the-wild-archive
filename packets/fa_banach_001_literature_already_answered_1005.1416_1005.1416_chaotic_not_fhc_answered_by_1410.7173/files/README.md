# Literature already answered: chaotic does not imply frequently hypercyclic

status: literature_already_answered

source: Manuel De la Rosa, Leonhard Frerick, Sophie Grivaux, and Alfredo
Peris, *Frequent hypercyclicity, chaos, and unconditional Schauder
decompositions*, arXiv:1005.1416.

supporting answer: Quentin Menet, *Linear chaos and frequent hypercyclicity*,
arXiv:1410.7173.

packet: `runs/fa_banach_001/solutions/literature_already_answered/1005.1416_chaotic_not_fhc_answered_by_1410.7173/`

## Identification

The source paper states in the introduction that it is still open whether every
chaotic operator on a Banach space is automatically frequently hypercyclic.

Menet's later paper explicitly formulates the same problem and answers it
negatively: Theorem 1.2 constructs a chaotic operator on `ell_1` which is not
`U`-frequently hypercyclic, hence not frequently hypercyclic.

## Scope

This packet records only the chaotic-implies-frequently-hypercyclic subquestion
mentioned by the source. It does not answer the source's broader Question 1.1
about characterizing complex separable Banach spaces supporting frequently
hypercyclic or chaotic operators, and it does not answer the real Gowers-space
frequent-hypercyclicity question at the end of the source paper.
