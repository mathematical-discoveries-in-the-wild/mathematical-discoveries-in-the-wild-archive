# Question 4.3 is answered by arXiv:2104.02714

Status: `literature_already_answered`

## Source question

Braga and Chávez-Domínguez, *Completely coarse maps are
\(\mathbb R\)-linear*, arXiv:2006.00948, Question 4.3, ask whether almost
complete coarse embeddability is
strictly weaker than almost complete real-isomorphic embeddability: do there
exist operator spaces \(X,Y\) such that \(X\) almost completely coarsely
embeds into \(Y\), but \(X\) does not almost completely
\(\mathbb R\)-isomorphically embed into \(Y\)?

## Supporting answer

Braga, Chávez-Domínguez, and Sinclair, *Lipschitz geometry of operator spaces
and Lipschitz-free operator spaces*, arXiv:2104.02714, explicitly says that
its main conceptual theorem answers Question 4.3 of the source paper.
Theorem 1.4 constructs operator spaces \(X,Y\) such that \(X\) almost
completely isometrically embeds into \(Y\), while \(X\) does not even
\(\mathbb R\)-isomorphically embed into \(Y\) at the Banach-space level.

An almost complete isometric embedding is, after the evident common linear
control, an almost complete coarse embedding.  Failure of every real-linear
Banach-space embedding is stronger than failure of an almost complete
real-isomorphic embedding.  Thus this is an affirmative and strictly stronger
answer to the exact source question.

The supporting authors explicitly identify their theorem as answering the
source question, so this is an `already answered` relation, not a new result
of this run.  The 2021 follow-up arXiv:2106.15022 independently summarizes
the same implication and cites Theorem 1.4 for these examples.

## Scope

The unrestricted question is answered.  The supporting example is
nonseparable; the same paper proves a positive linearization theorem for
separable domains under the stronger hypothesis of almost complete
isometric embeddability.  This separable refinement does not limit the
affirmative answer to the source's existential question.
