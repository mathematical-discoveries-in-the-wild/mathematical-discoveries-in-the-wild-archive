# Literature-implied partial answer: Bendova's Haydon-space example

## Status

`literature_implied_answer (partial subcase)`.

This is not a new proof produced by the run. It records that a later paper
settles one named family from Bendova's final open-ended question.

## Source question

- Source paper: Hana Bendova, *Quantitative Grothendieck property*,
  arXiv:1309.4684.
- Location: Section 4, final paragraph after the quotient results for
  `ell_infty(Gamma)`.
- Source signal: Bendova asks whether the other Grothendieck spaces mentioned
  in the introduction enjoy the quantitative version. The introduction lists
  `C(K)` for compact F-spaces, weak `L^p` spaces, `H^\infty`, and Haydon's
  Grothendieck space without `ell_infty`.

## Supporting answer

- Supporting paper: Jindrich Lechner, *1-Grothendieck C(K) spaces*,
  arXiv:1511.02202.
- Main supporting result: Theorem 1.2 says that if `S` is a Haydon space, then
  `C(S)` is 1-Grothendieck.
- Consequence in the same paper: the class includes Haydon's constructed
  Grothendieck space without an isomorphic copy of `ell_infty`; Lechner also
  records sigma-Stonean and extremally disconnected compact cases.

## Scope

This answers the Haydon-space/Haydon-example part of Bendova's final question
affirmatively, and with the sharp constant `1`.

It does not settle the remaining examples named by Bendova: general compact
F-space `C(K)` spaces, weak `L^p` spaces, or `H^\infty`. Lechner explicitly
leaves other introductory examples open after proving the Haydon-space theorem.
