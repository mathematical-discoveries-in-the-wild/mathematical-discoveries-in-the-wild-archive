# Historical Strong Partial Packet: State-Norming Units

Status at the time: `strong_partial_result_likely_valid`.

This was the second promoted form of the result for arXiv:1305.6420. It showed
that if `E` has a norm-one element `e` whose state face norms `E`, equivalently
a spear-vector style unit, then BPBp for `(X, E \hat{\otimes}_\epsilon Y)`
implies BPBp for `(X,Y)` with the same modulus.

This packet is now superseded by the parent full packet, which proves
qualitative descent for every nonzero `E`. The state-norming-unit theorem
remains as the same-modulus refinement.
