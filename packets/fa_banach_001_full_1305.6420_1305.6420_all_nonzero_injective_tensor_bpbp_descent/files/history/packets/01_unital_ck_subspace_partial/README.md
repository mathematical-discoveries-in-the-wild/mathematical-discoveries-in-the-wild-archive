# Historical Partial Packet: Unital `C(K)` Subspaces

Status at the time: `partial_result_likely_valid`.

This was the first promoted form of the result for arXiv:1305.6420. It showed
that if `E` is a closed linear subspace of `C(K)` containing the constant
function `1_K`, then BPBp for `(X, E \hat{\otimes}_\epsilon Y)` implies BPBp
for `(X,Y)` with the same modulus.

This packet is now superseded by the parent full packet, which proves
qualitative descent for every nonzero `E`. The result here remains a useful
same-modulus special case.
