# DSD2P coincides with the Daugavet property

Status: **literature already answered**.

Haller, Pirk, and Põldvere end arXiv:1603.08082 by asking whether the
diametral strong diameter two property (DSD2P) and the Daugavet property
coincide.

Vladimir Kadets answered this affirmatively in arXiv:2008.07258, published in
the Proceedings of the American Mathematical Society 149 (2021), 2579--2582.
Theorem 2.3 states that every Banach space with DSD2P has the Daugavet
property. The converse was already known, so the two properties are
equivalent.
