\section*{Source Problem}

The source paper is Koltai--Kunde, \emph{A Koopman--Takens theorem:
Linear least squares prediction of nonlinear time series}, arXiv:2308.02175v3.
On page 17 the authors formulate the following conjecture.

\begin{quote}
\textbf{Conjecture 25.} Let the ergodic \(T\in\MPT\) possess discrete spectrum.
Then \(\U=\U_T\) has a prevalent set of cyclic vectors in \(L^2_\mu\).
\end{quote}

\section*{Result}

\begin{theorem}
Let \(T\) be an ergodic invertible measure-preserving transformation on the
probability space considered in the source paper, and let
\(\U=\U_T\) act on the complex Hilbert space \(H=L^2_\mu\). If \(T\) has
discrete spectrum, then the set of cyclic vectors for \(\U\) is prevalent.
In fact, after writing \(H\) as an orthogonal sum of one-dimensional
eigenspaces,
\[
        H=\overline{\operatorname{span}}\{e_j:j\in J\},\qquad
        \U e_j=\lambda_j e_j,
\]
a vector \(f=\sum_{j\in J} a_j e_j\) is cyclic if and only if
\(a_j\ne 0\) for every \(j\in J\). Consequently the cyclic vectors also
contain a dense \(G_\delta\) set.
\end{theorem}

\section*{Idea of the Proof}

Discrete spectrum diagonalizes the Koopman operator. Ergodicity rules out
eigenvalue multiplicity: two eigenfunctions with the same eigenvalue have an
invariant quotient, hence are scalar multiples. Thus the problem becomes the
cyclic-vector problem for a diagonal unitary with distinct atoms on the unit
circle.

For a vector with all coordinates nonzero, the scalar spectral measure is
purely atomic and charges every eigenvalue. Kolmogorov's density theorem,
quoted in the source paper, says analytic polynomials are dense in
\(L^2\) of any singular measure on \(\T\). Therefore positive Koopman powers
of the vector recover every eigendirection. Finally, the set of vectors with
no zero coordinate is prevalent because a single probe vector with no zero
coordinate misses all coordinate hyperplanes except at countably many probe
parameters.

\section*{Proof}

Since \(T\) has discrete spectrum in the sense of the source paper,
\(H=L^2_\mu\) has a countable orthonormal basis of eigenfunctions for \(\U\).
We first note that each eigenspace is one-dimensional. Indeed, if
\(\U g=\lambda g\), then \(|g|\circ T=|g|\), so ergodicity implies that
\(|g|\) is almost everywhere constant. Thus a nonzero eigenfunction is
nonzero almost everywhere after changing it on a null set. If \(g\) and \(h\)
are eigenfunctions for the same eigenvalue \(\lambda\), then \(h/g\) is
\(T\)-invariant almost everywhere, hence constant by ergodicity. Thus
\(\operatorname{span}\{g,h\}\) is one-dimensional.

Index the eigenlines once, by a countable set \(J\), and choose normalized
eigenvectors \(e_j\) with pairwise distinct eigenvalues \(\lambda_j\in\T\).
Let
\[
        f=\sum_{j\in J} a_j e_j\in H.
\]
If \(a_k=0\) for some \(k\), then every vector in
\(\operatorname{span}\{\U^n f:n\ge 0\}\) has zero \(e_k\)-coordinate. Hence
the cyclic subspace cannot be all of \(H\), and \(f\) is not cyclic.

Conversely assume \(a_j\ne 0\) for all \(j\). Let
\[
        \nu_f=\sum_{j\in J}|a_j|^2\,\delta_{\lambda_j}
\]
be the scalar spectral measure associated to \(f\), up to the harmless
normalizing factor \(\|f\|^{-2}\) if one wants a probability measure. Under
the spectral transform for the cyclic subspace generated by \(f\), the vector
\(p(\U)f\) corresponds to the analytic polynomial \(p\) in
\(L^2(\nu_f)\). The measure \(\nu_f\) is purely atomic and therefore singular
with respect to Haar measure on \(\T\). Kolmogorov's density theorem
\cite[Theorem 2.11.5]{Simon}, also quoted as Theorem 13 in the source paper,
implies that analytic polynomials are dense in \(L^2(\nu_f)\).

Thus the closure of \(\{p(\U)f:p \text{ an analytic polynomial}\}\) is the
whole Hilbert space represented by \(L^2(\nu_f)\). Since every atom
\(\lambda_j\) has positive \(\nu_f\)-mass, this represented space is exactly
the closed span of all eigenvectors \(e_j\), namely \(H\). Hence \(f\) is
cyclic. We have proved the characterization:
\[
        f\text{ is cyclic}\quad\Longleftrightarrow\quad
        \langle f,e_j\rangle\ne 0\quad\text{for every }j\in J.
\]

It remains to prove prevalence. Enumerate \(J=\{1,2,\ldots\}\), with the
obvious finite modification if \(J\) is finite, and set
\[
        p=\sum_{j\ge 1}2^{-j}e_j\in H.
\]
Let \(E=\C p\), a one-dimensional complex probe space. For an arbitrary
\(h=\sum b_j e_j\in H\), the \(j\)-th coordinate of \(h+zp\) is
\[
        b_j+z\,2^{-j}.
\]
For each fixed \(j\) this coordinate vanishes for at most one value of
\(z\in\C\). Therefore the set of \(z\) for which at least one coordinate of
\(h+zp\) vanishes is countable, hence has two-dimensional Lebesgue measure
zero in \(E\simeq\C\). For almost every \(z\in\C\), all coordinates of
\(h+zp\) are nonzero, so \(h+zp\) is cyclic. This is exactly prevalence in
the sense used in the source paper.

The same characterization also gives a dense \(G_\delta\) set, since
\[
        \bigcap_{j\in J}\{f\in H:\langle f,e_j\rangle\ne 0\}
\]
is a countable intersection of open dense sets. This proves the conjecture.

\section*{Verification and Novelty Notes}

The proof uses only standard ingredients: diagonalization for discrete
spectrum, simple eigenspaces for ergodic transformations, Kolmogorov's
density theorem for singular spectral measures, and the definition of
prevalence via a finite-dimensional probe.

Local run indexes were searched for the arXiv id and terms including
``Koopman'', ``Takens'', ``cyclic vectors'', ``discrete spectrum'', and
``prevalent''. No exact prior packet for Conjecture 25 was found. Web searches
for exact phrases around the conjecture and its label did not reveal a later
paper explicitly resolving it. The arXiv page records arXiv:2308.02175v3 as
submitted on 2023-08-04 and last revised on 2024-03-31, with subjects
math.DS and math.FA.

Novelty confidence is moderate: the result is likely valid, but the argument
is short enough that it may be folklore or implicit in standard spectral
theory. Human review should focus on whether the use of positive powers
\(\U^n\), \(n\ge0\), is fully justified by Kolmogorov density in the purely
atomic measure.
