# Partial Result: Weak topologies have differentiable blid property

Status: `partial_result_likely_valid`

Source paper: Genrich Belitskii and Victoria Rayskin, "Extension of differentiable local mappings on linear topological spaces", arXiv:1812.11064.

## Result

Every real vector space \(X\) endowed with a weak locally convex topology \(\sigma(X,\mathcal F)\) generated by linear functionals has \(C^\infty\) differentiable blid property.

Given a weak neighborhood controlled by finitely many functionals, the proof cuts off only those finitely many coordinates using a finite-dimensional smooth local identity and a linear section. As a concrete corollary, every product \(\mathbb R^I\) with the product topology has differentiable blid property.
