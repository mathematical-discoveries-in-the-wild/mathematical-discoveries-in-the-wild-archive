# Literature already answered: Problem 3 from Moslehian's complemented subspace survey

Status: `literature_already_answered`

Source/open-problem paper: Mohammad Sal Moslehian, "A Survey on the Complemented Subspace Problem", arXiv:math/0501048.

Source question: Problem 3 in Section 12 ("Some interesting problems"), PDF page 10, asks whether every complemented
subspace of a `C(S)` space is isomorphic to some `C(S_1)`.

Supporting answer: Grzegorz Plebanek and Alberto Salguero Alarcon, "The complemented subspace problem for C(K)-spaces:
A counterexample", arXiv:2111.13860. The abstract and Theorem 1.1 construct a compact space `L` and a `1`-complemented
subspace `X` of `C(L)` which is not isomorphic to any space of continuous functions.

Identification: the supporting paper explicitly frames its result as a negative answer to the complemented subspace
problem for `C`-spaces. This is exactly the question appearing as Problem 3 in the source survey.

Scope: this clears only Problem 3 of the seven broad survey problems listed in arXiv:math/0501048. The other survey
questions are not claimed answered here.
