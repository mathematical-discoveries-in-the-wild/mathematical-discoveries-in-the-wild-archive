# Exact Asymptotic-Freeness Threshold for `ell_nu^infty(mu)`

Status: full solution candidate, subject to human review.

Source paper: Jarno Talponen, *Unconditional and bimonotone structures in high
density Banach spaces*, arXiv:1604.04408.

Target question: Section 6 asks, for infinite cardinals `nu < mu` and
`lambda < 2^nu`, for the least `kappa = kappa(lambda,nu,mu)` such that
`ell_nu^infty(mu)` is `(lambda,kappa)`-asymptotically free.

Result: for every infinite cardinal `lambda` and every uncountable regular
cardinal `kappa > lambda`,

`ell_nu^infty(mu)` is `(lambda,kappa)`-asymptotically free if and only if
`nu < kappa`.

Consequently, in the source paper's notation,

`kappa(lambda,nu,mu) = (max{lambda,nu})^+`.

This promotes the earlier upper-bound packet, which only proved sufficiency
of `2^nu < kappa`. The earlier packet is preserved in
`history/upper_bound_partial_2026_06_14/`.

Human review notes:

The positive direction is the key new simplification. For a fixed vector `x`,
one only needs to synchronize the coordinates in `supp(x)`, not the entire
coordinate band generated by a dense subset of `Z`. A Hahn-Banach correction
turns each relevant coordinate into an annihilator of `Z`; regularity of
`kappa` synchronizes these annihilators exactly when `|supp(x)| < kappa`.

The negative direction embeds `c0(kappa)` as a tail subspace when `kappa <= nu`.
The decreasing tail spaces have annihilators whose union is all of the ambient
dual, but the vector constantly `-1/2` on the `kappa` coordinates violates the
required tail inequality at every stage.
