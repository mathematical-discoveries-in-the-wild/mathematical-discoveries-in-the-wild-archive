{\large LRP/ANEP is equivalent to NEP}

\paragraph{Status.}
\textbf{full\_solution\_likely\_valid}.  This packet answers the converse
question in the remark after Proposition 4.4 of Antonin Dalet and Gilles
Lancien, \emph{Some properties of coarse Lipschitz maps between Banach
spaces}, arXiv:1612.01364.

\paragraph{Source question.}
The source defines a net in a metric space as an \((a,b)\)-net with
\(0<a\le b\): distinct net points are at distance at least \(a\), and every
ambient point is within distance \(b\) of the net.  In Section 4 the source
defines the net extension property (NEP), proves in Proposition 4.4 that NEP
implies the Lipschitz representation property (LRP), and then asks whether
the converse is true.  The same remark records that LRP is equivalent to an
almost net extension property (ANEP).

\paragraph{Result.}
The converse is true.  For Banach spaces \(X\) and \(Y\), NEP, LRP for some
constant, and ANEP for some constant are equivalent.

\begin{lemma}[Bounded data on a separated set]
Let \(X\) be a metric space, let \(Y\) be a Banach space, and let
\(\M\subset X\) be \(a\)-separated for some \(a>0\).  Every bounded map
\(e:\M\to Y\) has a Lipschitz extension \(E:X\to Y\).
\end{lemma}

\begin{proof}
Let \(B=\sup_{m\in \M}\|e(m)\|\).  If \(B=0\), take \(E=0\).  Otherwise put
\(r=a/4\) and, for \(m\in \M\),
\[
  \beta_m(x)=\max\{0,1-d(x,m)/r\}.
\]
The positive supports of the functions \(\beta_m\) are pairwise disjoint.
Define
\[
  E(x)=
  \begin{cases}
    \beta_m(x)e(m), & \text{if } \beta_m(x)>0 \text{ for the unique such }m,\\
    0, & \text{if } \beta_m(x)=0 \text{ for all }m\in\M .
  \end{cases}
\]
Then \(E(m)=e(m)\) for every \(m\in\M\).  We show that \(E\) is Lipschitz.
If \(x,y\) lie in the same bump support, then
\[
  \|E(x)-E(y)\|\le (B/r)d(x,y)= (4B/a)d(x,y).
\]
If \(x\) lies in the support of \(\beta_m\) and \(y\) lies outside that same
support with \(E(y)=0\), then \(d(y,m)\ge r\), so
\[
  \|E(x)-E(y)\|\le B(1-d(x,m)/r)
  = B(r-d(x,m))/r \le (B/r)d(x,y).
\]
Finally, if \(x\) and \(y\) lie in two different bump supports, then
\[
  d(x,y)\ge a-r-r=a/2,
\]
while \(\|E(x)-E(y)\|\le 2B\).  Hence
\(\|E(x)-E(y)\|\le (4B/a)d(x,y)\) in this case as well.  Thus \(E\) is
Lipschitz.
\end{proof}

\begin{theorem}
Let \(X\) and \(Y\) be Banach spaces.  Then the following are equivalent.
\begin{enumerate}[label=(\roman*)]
\item \((X,Y)\) has NEP.
\item \((X,Y)\) has \(\mu\)-LRP for some \(\mu\ge 1\).
\item \((X,Y)\) has \(\lambda\)-ANEP for some \(\lambda\ge 1\).
\end{enumerate}
\end{theorem}

\begin{proof}
The implication (i)\(\Rightarrow\)(ii) is exactly Proposition 4.4 of
Dalet--Lancien.  The source remark after that proposition records the
equivalence (ii)\(\Leftrightarrow\)(iii).  It remains only to prove
(iii)\(\Rightarrow\)(i).

Assume that \((X,Y)\) has \(\lambda\)-ANEP, witnessed by a net
\(\M\subset X\).  By the source definition of net, \(\M\) is an
\((a,b)\)-net for some \(0<a\le b\), hence it is \(a\)-separated.  Let
\(f:\M\to Y\) be Lipschitz.  ANEP gives a Lipschitz map \(g:X\to Y\) such
that \(f-g\) is bounded on \(\M\).  Define
\[
  e(m)=f(m)-g(m)\qquad (m\in\M).
\]
By the lemma, \(e\) has a Lipschitz extension \(E:X\to Y\).  Then
\[
  F=g+E
\]
is Lipschitz on \(X\), and for every \(m\in\M\),
\[
  F(m)=g(m)+e(m)=f(m).
\]
Thus every Lipschitz map \(\M\to Y\) admits a Lipschitz extension to \(X\).
So \((\M,X,Y)\) has the Lipschitz extension property, and therefore
\((X,Y)\) has NEP.
\end{proof}

\paragraph{Consequences.}
This proves the converse of Proposition 4.4 in the source paper.  In
particular, the ``almost'' bounded-error formulation is not weaker than NEP
for the separated nets used by Dalet--Lancien: bounded errors can be removed
by local Lipschitz bumps.

\paragraph{Limitations.}
This packet does not settle the later remark asking whether
\((CL(X,Y),\|\cdot\|_{CL})\) is always complete.  Proposition 4.5 says that
LRP, hence equivalently NEP, is sufficient for completeness, but the source
does not state that it is necessary.
