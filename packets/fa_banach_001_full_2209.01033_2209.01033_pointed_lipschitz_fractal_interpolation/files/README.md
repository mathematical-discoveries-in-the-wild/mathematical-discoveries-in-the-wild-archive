# Pointed Lipschitz fractal interpolation

Status: `full_solution_likely_valid`

This packet carries out the second future research direction in Peter R. Massopust, *Fractal Interpolation over Curves*, arXiv:2209.01033: replace the unital Lipschitz algebra by the pointed algebra `Lip_0`.

For an interval partition generated by increasing bi-Lipschitz branches `h_i`, it gives:

- the exact junction conditions under which the Read-Bajraktarevic formula defines an affine self-map of all `Lip_0([0,1])`;
- the corrected contraction constant
  `c = max_i Lip(h_i^{-1})(||s_i||_infty + Lip(s_i))`;
- existence, uniqueness, iteration convergence, and the usual a posteriori error bound;
- transport of the result to every simple bi-Lipschitz curve in a Banach space.

The junction conditions are necessary as well as sufficient. The packet also records the elementary example `s(x)=a x`, for which multiplication on `Lip_0` has norm at least `2a`; this explains why both multiplier terms must be added rather than maximized.
