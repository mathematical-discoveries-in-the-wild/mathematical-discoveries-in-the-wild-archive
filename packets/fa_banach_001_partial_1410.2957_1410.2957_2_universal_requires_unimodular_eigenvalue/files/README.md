# Partial Result: 2-Universal Operators Need a Unimodular Eigenvalue

Status: `partial_result_likely_valid`.

Source target: Sophie Grivaux, *Some new examples of universal hypercyclic operators in the sense of Glasner and Weiss*, arXiv:1410.2957.

## Source Questions

Question 5.8 asks whether there exist universal, or `2`-universal, operators admitting no unimodular eigenvalue. Question 5.9 asks whether the no-eigenvalue Kalish-type operator on `C_0([0,2pi])` is universal, or `2`-universal, for invertible ergodic systems.

## Result

Every `2`-universal operator for invertible ergodic systems on a complex separable Banach space has a unimodular eigenvalue.

Consequently, the Kalish-type operator in Question 5.9 is **not** `2`-universal for invertible ergodic systems, because the source records that it has no unimodular eigenvalue.

## Scope

This settles the finite-second/no-eigenvalue clause, but not ordinary universality without a moment condition. It also does not prove the stronger Hilbert-space conclusions about full-measure or full-circle unimodular point spectrum.
