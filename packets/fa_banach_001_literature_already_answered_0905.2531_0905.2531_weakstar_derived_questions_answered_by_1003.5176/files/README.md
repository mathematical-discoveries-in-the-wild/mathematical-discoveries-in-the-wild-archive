# Weak-Star Derived-Set Questions Answered by Ostrovskii

Status: `literature_already_answered`.

Source/open-problem paper: Domingo Garcia, Ondrej F. K. Kalenda, and Manuel Maestre, *Envelopes of open sets and extending holomorphic functions on dual Banach spaces*, arXiv:0905.2531, J. Math. Anal. Appl. 363 (2010), 663--678.

Supporting answer paper: M. I. Ostrovskii, *Weak$^*$ closures and derived sets in dual Banach spaces*, arXiv:1003.5176, Note Mat. 31 (2011), 129--138.

## Identification

The source paper's final section asks Question 6.3(a), whether for a quasireflexive Banach space `X` one has `\overline A^{w*}=A^{(1)}` for every `(absolutely) convex` subset `A` of `X*`, and Question 6.5, which Banach spaces admit a linear subspace `A subset X*` whose first weak-star derived set `A^{(1)}` is proper and norm dense in `X*`.

Ostrovskii's paper explicitly states that its main purpose is to answer these two questions from the source paper. Its Theorem 1 answers Question 6.5 exactly: such a linear subspace exists if and only if `X` is non-quasireflexive and contains an infinite-dimensional subspace with separable dual. Its Theorem 3 answers the absolutely convex part of Question 6.3(a) affirmatively for quasireflexive spaces, while Theorem 2 shows that the general convex version fails for every non-reflexive Banach space.

## Scope

This packet records only the weak-star derived-set questions, namely source Questions 6.3(a) and 6.5. It does not claim to answer the balanced-set extension question 6.2, the bounded-regularity question 6.3(b), or Question 6.4 about `A+rB_{X*}`.
