\section*{Status}

\textbf{Status: literature already answered.}
The weak-star derived-set questions in Garcia--Kalenda--Maestre,
\emph{Envelopes of open sets and extending holomorphic functions on dual Banach spaces},
arXiv:0905.2531, Questions 6.3(a) and 6.5, are explicitly answered by
M. I. Ostrovskii, \emph{Weak$^*$ closures and derived sets in dual Banach spaces},
arXiv:1003.5176.

\section*{Source Questions}

On page 19, the source paper asks:

\begin{quote}
Question 6.3(a). Let $X$ be a quasireflexive Banach space. Is
$\overline A^{w^*}=A^{(1)}$ for each (absolutely) convex set
$A\subset X^*$?
\end{quote}

\begin{quote}
Question 6.5. For which Banach spaces $X$ is there a linear subspace
$A\subset X^*$ such that $A^{(1)}$ is a proper norm-dense subset of
$X^*$? Is it true whenever $X$ is not quasireflexive?
\end{quote}

\section*{Supporting Answer}

Ostrovskii's introduction explicitly says that the paper answers these two
questions from Garcia--Kalenda--Maestre. The stated main theorems give the
answer:

\begin{itemize}
\item Theorem 1 answers Question 6.5 exactly: such a linear subspace exists
if and only if $X$ is non-quasireflexive and contains an infinite-dimensional
subspace with separable dual.
\item Theorem 3 answers the absolutely convex part of Question 6.3(a)
affirmatively: if $X$ is quasireflexive and $A\subset X^*$ is absolutely
convex, then $A^{(1)}=\overline A^{w^*}$.
\item Theorem 2 shows that the unrestricted convex version is false in every
non-reflexive Banach space, hence in non-reflexive quasireflexive spaces.
\end{itemize}

\section*{Scope Limitations}

This packet records only the literature answer to Questions 6.3(a) and 6.5.
It does not resolve Question 6.2 about balanced sets, Question 6.3(b) about
bounded regularity of open convex sets in quasireflexive duals, or Question
6.4 about the sets $A+rB_{X^*}$.
