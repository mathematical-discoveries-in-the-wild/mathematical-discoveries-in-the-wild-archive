# Literature-Implied Answer: Extreme Amenability of `Iso(L_p(L_q))`

Status: `literature_implied_answer` (full scope for Question `quest:kpt`)

Source question: Cuth--de Rancourt--Doucha, arXiv:2408.12755, `Guarded Fraisse Banach spaces`, Question `quest:kpt` in Section 7 asks whether `Iso(L_p(L_q))`, with the strong operator topology, is extremely amenable for every `1 <= p,q < infinity`, and if not whether it has a metrizable universal minimal flow.

Answer recorded here: yes, `Iso(L_p(L_q))` is extremely amenable for all `1 <= p,q < infinity`; hence its universal minimal flow is a singleton.

Identification: this is not an explicitly advertised later answer. It is an agent-identified implication from known literature: scalar `Iso(L_p)` extreme amenability, Pestov--Schneider's theorem that `L^0(G)` is extremely amenable whenever `G` is amenable, and the Guerre--Raynaud representation of `Iso(L_p(L_q))` as the corresponding measurable-fiber isometry extension.
