# Literature answer: arXiv:2006.16933 via arXiv:2206.13146

**Status:** `literature_already_answered`  
**Source:** Liran Rotem, *Surface area measures of log-concave functions*,
arXiv:2006.16933.  
**Answer:** Liran Rotem, *The anisotropic total variation and surface area
measures*, arXiv:2206.13146.

The source asks on PDF page 5 for a surface-area measure and first-variation
formula valid for all integrable upper-semicontinuous log-concave functions,
including both smooth positive functions and indicators of convex bodies.

The later paper answers this in Definition 1.2, Definition 1.4, and Theorem
1.5 (supporting PDF pages 2--3). It introduces an interior measure
`mu_f` on `R^n` and a boundary measure `nu_f` on the sphere, and proves the
two-term first-variation formula without regularity assumptions. For
`f = 1_K`, the boundary measure is exactly the classical surface-area measure
`S_K`; supporting PDF pages 4--5 explicitly treat indicator perturbations.
