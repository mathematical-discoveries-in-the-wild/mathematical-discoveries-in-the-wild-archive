# Fatou-Version Isometry Question Answered by Later Literature

Status: `literature_already_answered` for the standard strong-symmetric/Fatou-property formulation of the Fatou-norm extension question.

Source/open-problem paper: Jinghao Huang and Fedor Sukochev, "Hermitian operators and isometries on symmetric operator spaces", arXiv:2101.03667; later JEMS 26 (2024), 3287-3325. The relevant source remark appears after Example 4.12 / near source lines 2496-2502 and asks whether Theorem `th:iso` can be verified under the weaker assumption that the symmetric operator spaces have Fatou norms only.

Supporting answer paper: Kai Fang, Tianbao Guo, Jinghao Huang, and Fedor Sukochev, "Isometric Structure in Noncommutative Symmetric Spaces", arXiv:2512.20972. In the introduction, the authors explicitly say that one goal is to treat the case where the norm is not necessarily order continuous, and their first main result, Theorem `11212`, gives the elementary form of surjective isometries for strongly symmetric operator spaces having the Fatou property.

Scope note: the later theorem is phrased using "strongly symmetric operator spaces having the Fatou property" and sigma-finite von Neumann algebras. This records the target as answered in that standard formulation; it does not claim a separate resolution for any bare terminology distinction between "Fatou norm" and "Fatou property" outside that framework.
