## Claimed Contribution

The source paper asks Question 5.6:

> Is every `B`-spreading model of `T` isomorphic to `ell_1`, where `B`
> is an arbitrary barrier?

This packet gives an affirmative answer.

The key point is that a `B`-spreading model is not merely a spreading
sequence in the abstract. Choose an infinite block chain
`s_1 < s_2 < ...` inside the barrier `B`. If `X(s)` denotes the normalized
block vector over `s`, then `z_n = X(s_n)` is a normalized block sequence
in the original Banach space, and the defining asymptotic estimate for the
`B`-model says exactly that `(z_n)` generates the same basis as an ordinary
Brunel--Sucheston spreading model. Applying the source paper's cited fact
from Odell, namely that every spreading model of Tsirelson space `T` is
isomorphic to `ell_1`, answers the question.
