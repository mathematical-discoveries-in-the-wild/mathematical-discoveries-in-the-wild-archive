\section*{Source Problem}

The source paper asks the following in its Main Results section, PDF page~5.

\begin{center}
\includegraphics[width=\textwidth]{figures/open_problem_crop.png}
\end{center}

Transcribed, the open problem asks whether, for
\[
  r\in (0,1)\cup\bigl((1,\infty)\setminus\mathbb Z\bigr),
\]
every non-elliptic Fourier integral operator
\[
  T\in I^\mu_{1,0}(X;\Lambda)
\]
that belongs to the Schatten class \(S_r(L^2(X))\) must satisfy
\[
  \mu<-n/r.
\]

\section*{Definitions and Convention}

Let \(X\) be a compact smooth manifold of positive dimension \(n\), equipped
with a smooth positive density so that \(L^2(X)\) is fixed.  We write
\(\Delta\) for the diagonal canonical relation.  The standard smoothing
operators satisfy
\[
  \Psi^{-\infty}(X)\subseteq \Psi^\mu_{1,0}(X)
  = I^\mu_{1,0}(X;\Delta)
\]
for every real \(\mu\); equivalently, their kernels are \(C^\infty\) and their
finite-order principal symbols vanish.  This is the ordinary
class-membership interpretation of \(T\in I^\mu\), as opposed to saying that
\(T\) has exact order \(\mu\).

\section*{Proof Intuition}

Schatten membership cannot force a strict upper bound on a merely declared
order class, because smoothing operators belong to all finite-order classes at
once.  A rank-one smoothing operator is as compact as possible: it has only one
nonzero singular value, so it lies in every Schatten class.  But if we view the
same operator as an element of \(I^\mu_{1,0}\) for an arbitrary large
\(\mu\), it is still non-elliptic because its principal symbol at that order
is zero.  Thus the order parameter in the literal hypothesis is not an
intrinsic order of the operator.

\begin{theorem}
Let \(X\) be a compact smooth manifold of positive dimension \(n\).  For every
\(r>0\) and every \(\mu\ge -n/r\), there exists a nonzero non-elliptic
operator \(T\in I^\mu_{1,0}(X;\Delta)\) such that
\[
  T\in S_r(L^2(X)).
\]
Consequently the literal statement of the source open problem is false.
\end{theorem}

\begin{proof}
Choose nonzero functions \(u,v\in C^\infty(X)\).  Define
\[
  Tf=\langle f,v\rangle_{L^2(X)}\,u,\qquad f\in L^2(X).
\]
The Schwartz kernel is
\[
  K_T(x,y)=u(x)\overline{v(y)},
\]
which is \(C^\infty\) on \(X\times X\).  Hence \(T\) is smoothing:
\[
  T\in \Psi^{-\infty}(X).
\]
By the usual nesting of pseudodifferential symbol classes,
\(\Psi^{-\infty}(X)\subseteq \Psi^\mu_{1,0}(X)\) for every real \(\mu\).
Since pseudodifferential operators are Fourier integral operators associated
with the diagonal canonical relation, this gives
\[
  T\in I^\mu_{1,0}(X;\Delta)
\]
for every real \(\mu\).

The operator \(T\) has rank one.  More explicitly,
\[
  T = u\otimes v
\]
in the Hilbert-space rank-one notation, so its singular values are
\[
  s_1(T)=\|u\|_2\|v\|_2,\qquad s_j(T)=0\quad(j\ge2).
\]
Therefore
\[
  \sum_{j\ge1}s_j(T)^r = \bigl(\|u\|_2\|v\|_2\bigr)^r<\infty
\]
for every \(r>0\).  Thus \(T\in S_r(L^2(X))\), including the quasi-Banach
range \(0<r<1\).

Finally, \(T\) is not elliptic as an element of any finite-order class
\(I^\mu_{1,0}(X;\Delta)\).  Its finite-order principal symbol is zero because
the operator is smoothing.  In particular it is non-elliptic.  If
\(\mu\ge -n/r\), the conclusion \(\mu<-n/r\) fails.  Taking \(\mu=0\) gives a
counterexample for every \(r>0\) and every \(n\ge1\).
\end{proof}

\section{Limitations and Exact-Order Variant}\label{sec:limitations}

This counterexample uses the difference between membership in a filtered class
and exact order.  It does not address the strengthened statement:

\begin{quote}
If \(T\) is non-elliptic but has exact order \(\mu\), or has a nonzero
principal symbol at order \(\mu\), and \(T\in S_r(L^2(X))\), must
\(\mu<-n/r\)?
\end{quote}

That is the natural repaired version of the problem.  The source paper itself
uses exact-order language in the proofs of the trace-class and integer-index
results; for instance it argues with \(T\) ``having order \(\mu\)'' and with a
nonzero principal symbol.  The present packet therefore should be read as a
literal counterexample and a repair note: the stated open problem needs an
exact-order or nonzero-principal-symbol hypothesis to avoid smoothing
operators.
