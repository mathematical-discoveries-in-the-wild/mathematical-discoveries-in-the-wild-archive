# 0310023 elastic `C[0,1]` problem answered by 1502.03791

Status: literature_already_answered.

Original source: W. B. Johnson and E. Odell, *The Diameter of the
Isomorphism Class of a Banach Space*, arXiv:math/0310023.

Supporting source: Dale E. Alspach and Bunyamin Sari, *Separable Elastic
Banach spaces are universal*, arXiv:1502.03791.

## Source Problem

Johnson and Odell ask in Problem 7:

```text
Let X be elastic (and separable, say). Does C[0,1] embed into X?
```

## Answer

Alspach and Sari explicitly identify this as the Johnson-Odell conjecture
and prove it. Their Theorem 1 states that if `X` is a separable infinite
dimensional elastic Banach space, then `C[0,1]` is isomorphic to a subspace
of `X`.

Thus the separable version of Problem 7 is answered affirmatively. This
packet does not claim to settle any broader nonseparable elastic-space
variant or the nonseparable form of the original diameter problem.
