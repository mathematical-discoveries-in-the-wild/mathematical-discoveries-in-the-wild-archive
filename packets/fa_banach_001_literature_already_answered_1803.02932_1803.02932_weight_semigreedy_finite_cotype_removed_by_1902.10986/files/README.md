# Literature already answered: weighted semi-greedy finite-cotype question

Status: `literature_already_answered`

Original source: S. J. Dilworth, Denka Kutzarova, Vladimir Temlyakov, and Ben Wallis, *Weight-almost greedy bases*, arXiv:1803.02932.

Supporting answer: P. M. Berna, *Characterization of weight-semi-greedy bases*, arXiv:1902.10986.

## Target

Question 4.12 of arXiv:1803.02932 asks whether the analogue of Corollary 4.11 holds for semi-greediness. In the terminology around the question, this asks whether being `w`-semi-greedy for some weight is equivalent to being semi-greedy, beyond the finite-cotype partial answer in Corollary 4.16.

## Identification

Berná's later paper explicitly cites the original weighted semi-greedy result and says it proves the same characterization with the finite-cotype hypothesis removed. The decisive statement is Theorem 1.11: for any Schauder basis in a Banach space, `w`-semi-greediness is characterized by quasi-greediness plus weighted super-democracy, and the converse also holds. Together with the weight-almost-greedy characterization, this removes the finite-cotype hypothesis from the original question for the Schauder-basis setting of arXiv:1803.02932.
