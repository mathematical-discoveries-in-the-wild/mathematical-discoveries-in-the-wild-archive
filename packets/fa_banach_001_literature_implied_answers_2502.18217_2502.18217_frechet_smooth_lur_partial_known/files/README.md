# Partial Literature Answer: Frechet Smooth + LUR Renormings

Status: `literature_implied_answer (partial subcases)`

Source target: arXiv:2502.18217, D. Gothwal and V. Shekhar, *(Asymptotic)
uniform smoothness, ball separation and residuality results*.

Source question: Section 5, "Two open problems", asks whether a Banach space
with an equivalent Frechet smooth norm and an equivalent LUR norm must admit an
equivalent norm that is both Frechet smooth and LUR.

Identification: Hajek--Prochazka, arXiv:0901.3623, prove class-level positive
answers. Their introduction records the classical separable Asplund positive
case, and Corollary 4.4 proves that if a space admits a \(C^k\)-Frechet smooth
norm and is Vasak/WCD, WLD, or \(C(K)\) for Valdivia compact \(K\), then it
admits an equivalent \(C^1\)-Frechet smooth LUR norm.

Scope: this does not solve the full arbitrary Banach-space question and does
not solve the residuality of Frechet smooth norms. It records known positive
subcases that should not be attacked as new.
