# Literature status: canonical inclusion maps for Lipschitz-free p-spaces

Status: literature already answered.

Source paper: arXiv:1811.01265, *Lipschitz free p-spaces for 0<p<1*.

Later paper: arXiv:2603.28193, *Lipschitz extensions into p-Banach spaces, and canonical embeddings of Lipschitz-free p-spaces for 0<p<1*.

## Question identified

In Section 6 of arXiv:1811.01265, the authors ask whether, for `0<p<1` and `N subset M` two p-metric (or metric) spaces in inclusion, the canonical linear map from `F_p(N)` into `F_p(M)` is an isomorphic embedding.

The later paper arXiv:2603.28193 cites this as `AACD2020, Question 6.2`.

## Resolution by later literature

arXiv:2603.28193 explicitly states in the introduction that it gives a positive solution to this question in full generality.

The decisive theorem is arXiv:2603.28193, Theorem 3.13: for every `0<p<=1` there is a constant `C(p)` such that for every pointed p-metric space `M` and every `0 in N subset M`, the canonical map `L[N,M;p]: F_p(N) -> F_p(M)` is an isomorphic embedding with inverse norm at most `C(p)`.

This directly gives a positive answer to the source question. Ordinary metric spaces are covered by the source convention as a special case for `0<p<=1`.

## Scope notes

This packet does not claim new mathematical progress. It clears the canonical inclusion/open embedding signal from arXiv:1811.01265.

This packet does not settle the neighboring classification question in the source paper asking which p-metric spaces make the canonical map isometric for every subspace.
