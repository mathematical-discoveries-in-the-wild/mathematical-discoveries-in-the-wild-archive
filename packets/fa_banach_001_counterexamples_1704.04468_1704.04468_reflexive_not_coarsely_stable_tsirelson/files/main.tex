\section*{Source Problem}

The source is Braga--Swift, \emph{Coarse embeddings into superstable spaces},
arXiv:1704.04468 \cite{BragaSwift}.  On page 3 they recall Kalton's Problem
6.1 and state:

\begin{center}
\includegraphics[width=\textwidth]{figures/open_problem_crop.png}
\end{center}

\noindent
Thus the question is whether every separable reflexive Banach space embeds
coarsely, respectively uniformly, into a stable metric space.

\section*{Result}

\begin{theorem}\label{thm:main}
Tsirelson's original space \(T^*\) is a separable reflexive Banach space which
does not coarsely embed into any stable metric space. Consequently it gives a
negative answer to Problem 1.5 of \cite{BragaSwift}. Moreover, \(T^*\) does not
uniformly embed into any stable metric space.
\end{theorem}

\section*{Proof Intuition}

The obstruction is not linear containment of \(c_0\), since \(T^*\) is
reflexive and contains no copy of \(c_0\).  The obstruction is finite
representability.  If \(X\) finitely represents \(c_0\), then every finite part
of the integer lattice of \(c_0\) appears in \(X\) with uniformly bounded
distortion.  A coarse embedding \(X\to M\) into a stable metric space would
therefore send those finite lattice pieces into \(M\) with uniform coarse
control.  Passing to a cofinal metric ultrapower turns these finite pieces into
a genuine coarse embedding of a net in \(c_0\) into a stable metric space,
contradicting Kalton's theorem that \(c_0\) does not coarsely embed into any
stable metric space.

\section*{Proof}

We use two standard facts.  First, metric stability is preserved by metric
ultrapowers.  Second, Kalton proved that \(c_0\) does not coarsely or uniformly
embed into a stable metric space \cite{Kalton2007}; this is also quoted in
\cite[p. 3]{BragaSwift}.

\begin{lemma}\label{lem:finitely-c0}
Let \(X\) be a Banach space which finitely represents \(c_0\).  Then \(X\) does
not coarsely embed into any stable metric space.
\end{lemma}

\begin{proof}
Assume toward a contradiction that \(f:X\to M\) is a coarse embedding into a
stable metric space \(M\).  Let \(\rho,\omega:[0,\infty)\to[0,\infty)\) be
nondecreasing control functions with
\[
  \rho(\|x-y\|)\le d_M(f(x),f(y))\le \omega(\|x-y\|)
  \quad (x,y\in X),
\]
\(\rho(t)\to\infty\), and \(\omega(t)<\infty\) for every \(t\).

Let
\[
  \Lambda=\co\cap\Z^\N=c_{00}(\Z)
\]
with the \(c_0\)-metric.  This lattice is a net in \(c_0\): rounding each
coordinate of \(z\in c_0\) to a nearest integer gives an element of \(\Lambda\)
within distance \(1/2\).

Let \(I\) be the directed set of finite subsets of \(\Lambda\), ordered by
inclusion, and let \(\mathcal U\) be an ultrafilter on \(I\) extending the
cofinal filter, so that \(\{E\in I:F\subset E\}\in\mathcal U\) for every finite
\(F\subset\Lambda\).  Since \(c_0\) is finitely represented in \(X\), after
fixing a distortion constant \(D\ge1\), for each \(E\in I\) there are
\(n(E)\in\N\), containing the supports of all elements of \(E\), and a linear
map
\[
  T_E:\ell_\infty^{n(E)}\longrightarrow X
\]
such that
\[
  \|u\|_\infty\le \|T_Eu\|\le D\|u\|_\infty
  \quad (u\in\ell_\infty^{n(E)}).
\]
For \(a\in\Lambda\), put \(x_E(a)=T_E(P_{n(E)}a)\), where \(P_n\) is the first
\(n\)-coordinate projection.  The family \((f(x_E(a)))_{E\in I}\) is bounded in
\(M\), because \(\|x_E(a)\|\le D\|a\|_\infty\).  Therefore it defines a point
\[
  \Phi(a)=[\,f(x_E(a))\,]_{\mathcal U}
\]
in the metric ultrapower \(M_{\mathcal U}\).

For fixed \(a,b\in\Lambda\), the set of \(E\)'s containing both \(a\) and \(b\)
belongs to \(\mathcal U\).  On this set,
\[
  \|a-b\|_\infty
  \le \|x_E(a)-x_E(b)\|
  \le D\|a-b\|_\infty .
\]
Taking the \(\mathcal U\)-limit in \(M\), we get
\[
  \rho(\|a-b\|_\infty)
  \le d_{M_{\mathcal U}}(\Phi(a),\Phi(b))
  \le \omega(D\|a-b\|_\infty).
\]
Thus \(\Phi:\Lambda\to M_{\mathcal U}\) is a coarse embedding.  Since
\(\Lambda\) is a net in \(c_0\), choose a nearest-lattice map
\(q:c_0\to\Lambda\).  Then
\[
  \|x-y\|_\infty-1
  \le \|q(x)-q(y)\|_\infty
  \le \|x-y\|_\infty+1,
\]
so \(\Phi\circ q\) is a coarse embedding of \(c_0\) into \(M_{\mathcal U}\).
But \(M_{\mathcal U}\) is stable, contradicting Kalton's non-embeddability
theorem for \(c_0\).
\end{proof}

\begin{lemma}\label{lem:uniform}
Let \(X\) be a Banach space which finitely represents \(c_0\).  Then \(X\) does
not uniformly embed into any stable metric space.
\end{lemma}

\begin{proof}
If \(f:X\to M\) were a uniform embedding into a stable metric space, then the
usual Banach ultrapower argument would give a uniform embedding of \(c_0\) into
a stable metric ultrapower of \(M\).  Indeed, finite representability of \(c_0\)
in \(X\) gives an isomorphic embedding \(J:c_0\to X_{\mathcal U}\) into a
Banach ultrapower of \(X\).  Uniform continuity of \(f\), together with the fact
that \(X\) is a normed vector space, implies that \(f\) maps bounded subsets of
\(X\) to bounded subsets of \(M\) and induces a uniformly continuous map
\[
  f_{\mathcal U}:X_{\mathcal U}\to M_{\mathcal U}.
\]
The inverse uniform modulus of \(f\) passes to \(f_{\mathcal U}\): if
\(\|u-v\|\ge\varepsilon\) in \(X_{\mathcal U}\), then
\[
  d_{M_{\mathcal U}}(f_{\mathcal U}u,f_{\mathcal U}v)
  \ge \rho_f(\varepsilon/2)>0,
\]
where \(\rho_f(t)=\inf\{d_M(f(x),f(y)):\|x-y\|\ge t\}\).  Hence
\(f_{\mathcal U}\circ J\) is a uniform embedding of \(c_0\) into the stable
metric space \(M_{\mathcal U}\), again contradicting Kalton's theorem.
\end{proof}

We now apply the lemmas to Tsirelson's original space.  Baudier--Lancien--
Schlumprecht recall that \(T^*\) is reflexive and that all spreading models of
\(T^*\) are isomorphic to \(c_0\) \cite[p. 18]{BLS}.  Therefore \(c_0\) is
finitely represented in \(T^*\).  Lemmas \ref{lem:finitely-c0} and
\ref{lem:uniform} show that \(T^*\) neither coarsely nor uniformly embeds into
any stable metric space.  Since \(T^*\) is separable and reflexive, this proves
Theorem \ref{thm:main}. \(\square\)
