# Verifier Report

Verdict: `candidate_full_likely_valid`

## Claim Checked

The trace-only maximal nonregularity conclusion conjectured in Figure 2 of
arXiv:2302.04816 holds for all non-self-adjoint trace-class idempotents in the
Sobolev region `p>1`, `0<s<=1`, `s>=1/p`, without a uniform
`L^{2+epsilon}` assumption.

## Proof Audit

1. **Normalization transfer.** A trace-class idempotent is compact and has
   finite-dimensional range. Its nonzero eigenvalues are all one, hence its
   trace and rank agree. The orthogonal range projection therefore has the
   same normalized trace and meets the hypotheses of the source's
   self-adjoint theorem.

2. **Range-distance lemma.** For `x` in `Ran(P_0)`, the vector `P_1 x` lies in
   `Ran(P_1)`, so orthogonal projection gives
   `||(I-Q_1)x|| <= ||(P_1-P_0)x||`. Positive-operator order and min-max
   control one principal-angle block. Reversing the roles controls the other.
   The square of `Q_1-Q_0` is block diagonal relative to `Q_0`, giving the
   stated Schatten factor `2^(1/p)` exactly.

3. **Fractional Sobolev transfer.** Unitary conjugation carries `Ran(P)` to
   the corresponding translated range, so the lemma applies pointwise to
   every Weyl translation. The semiclassical scale factor and the weighted
   integral preserve the inequality. The source's self-adjoint divergence
   then forces the non-self-adjoint divergence.

4. **Order-one transfer.** Relative to
   `Ran(P) direct-sum Ran(P)^perp`, the lower-left block of `[A,P]` is
   `(I-Q)AQ`, exactly the off-diagonal block determining `[A,Q]` for a
   self-adjoint generator `A`. Applying this to position and self-adjoint
   momentum generators transfers the source's Corollary 2.1. If a domain or
   Schatten condition fails, the target seminorm is already infinite.

5. **Besov add-on.** For `0<s<1`, iterating
   `Delta_z=(Delta_(2z)-Delta_z^2)/2` converts second differences to first
   differences with the convergent geometric constant
   `1/[2(1-2^(s-1))]`. The reverse estimate is immediate from translation
   isometry. This correctly excludes the `s=1` second-difference endpoint
   from the add-on claim.

## Computational Sanity Check

commutator inequalities in 3,500 random norm/exponent cases, in dimensions

All checks passed. This is not a proof dependency.
