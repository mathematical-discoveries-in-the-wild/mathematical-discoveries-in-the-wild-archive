\section*{Source Question}

The source paper is Daws, Haydon, Schlumprecht, and White, \emph{Shift invariant preduals of \(\ell_1(\Z)\)}, arXiv:1101.5696 \cite{DHSW}.  Its Question 5 asks:
\begin{quote}
Characterise those \(a\in\ell_1(\Z)\) which occur as weak\(^*\)-limit points of
\(\{\delta_n:n\in\Z\}\).  In particular, is the condition
\(\lim_n \|a^n\|_\infty=0\) necessary as well as sufficient?

\section*{Result}

For \(a\in\ell_1(\Z)\), write
\[
  \widehat a(z)=\sum_{k\in\Z} a(k)z^k,\qquad z\in\T.
\]
The convolution power \(a^n\) has Fourier transform \(\widehat a^{\,n}\).

\begin{theorem}[Null peripheral criterion]
\label{thm:nullperipheral}
Let \(E\subseteq\ell_\infty(\Z)\) be a shift-invariant predual of \(\ell_1(\Z)\).
Suppose \(a\in\ell_1(\Z)\) is a weak\(^*\)-limit point of
\(\{\delta_n:n\in\Z\}\) for this predual.  If
\[
  m\bigl(\{z\in\T:|\widehat a(z)|=1\}\bigr)=0,
\]
where \(m\) is normalized Haar measure on \(\T\), then
\[
  \|a^n\|_\infty\longrightarrow 0.
\]
\end{theorem}

\begin{corollary}[Finitely supported candidates]
\label{cor:finite}
The necessity direction in Question 5 is affirmative for every finitely
supported weak\(^*\)-limit candidate \(a\) which is not a unimodular translate
of a point mass \(\lambda\delta_k\).
In particular, if a finitely supported \(a\) actually occurs as a weak\(^*\)-limit
point of point masses in a shift-invariant predual, then
\(\|a^n\|_\infty\to0\).
\end{corollary}

\section*{Proof Intuition}

The source semigroup machinery already gives the key algebraic fact: every
weak\(^*\)-limit point \(a\) of point masses is power-bounded for convolution.
On the Fourier side this says that the powers \(\widehat a^{\,n}\) stay
bounded in the Wiener algebra, hence \(|\widehat a|\le1\).  Once the set where
\(|\widehat a|=1\) has measure zero, dominated convergence makes
\(\widehat a^{\,n}\to0\) in \(L^2(\T)\).  Plancherel then controls every
Fourier coefficient uniformly, which is exactly the \(\ell_\infty\)-norm of
the convolution power.  The finite-support corollary follows because a
non-monomial Laurent polynomial cannot have modulus one on a positive-measure
subset of the circle.

\section*{Proof}

We first record the source-paper consequence used here.  By Proposition 4.3 of
\cite{DHSW}, every shift-invariant predual \(E\) is induced by a compact
semitopological semigroup \(\mathcal S\) and a bounded Banach-algebra
homomorphism
\[
  \Theta:M(\mathcal S)\to \ell_1(\Z)
\]
which is a projection onto the copy of \(\ell_1(\Z)\).  Proposition 4.4 of
\cite{DHSW} identifies the weak\(^*\)-closure of the point masses as
\[
  \{\Theta(\delta_\gamma):\gamma\in\overline{\Z}^{\,\mathcal S}\}.
\]
Thus, if \(a\) is a weak\(^*\)-limit point of point masses, then
\(a=\Theta(\delta_\gamma)\) for some \(\gamma\in\mathcal S\).  Since \(\Theta\)
is an algebra homomorphism,
\[
  a^n=\Theta(\delta_\gamma)^n=\Theta(\delta_{n\gamma}),
\]
and therefore
\[
  \sup_{n\ge1}\|a^n\|_1\le \|\Theta\|<\infty.
\]

Let \(f=\widehat a\in A(\T)\), the Wiener algebra.  From the preceding bound,
\[
  |f(z)|^n=|\widehat{a^n}(z)|\le \|a^n\|_1\le \|\Theta\|
  \qquad(z\in\T,\ n\ge1),
\]
so \(|f(z)|\le1\) for all \(z\in\T\).
Assume now that \(m(\{|f|=1\})=0\).  Then \(|f(z)|^{2n}\to0\) for almost every
\(z\in\T\), and the functions are dominated by \(1\).  Dominated convergence gives
\[
  \int_\T |f(z)|^{2n}\,dm(z)\longrightarrow0.
\]
By Plancherel,
\[
  \|a^n\|_\infty
  \le \|a^n\|_2
  = \left(\int_\T |f(z)|^{2n}\,dm(z)\right)^{1/2}
  \longrightarrow0.
\]
This proves Theorem~\ref{thm:nullperipheral}.

For the finite-support corollary, \(\widehat a\) is a Laurent polynomial.  If
\(\widehat a\) is not of the form \(\lambda z^k\) and the set
\(\{|\widehat a|=1\}\) had positive measure, then the real-analytic function
\(|\widehat a(e^{it})|^2-1\) would vanish on a set with an accumulation point,
hence would vanish identically.  Thus \(|\widehat a|=1\) on all of \(\T\).
Writing \(\widehat a(z)=z^{-N}p(z)\), with \(p\) a polynomial, we get
\(|p|=1\) on \(\T\).  A polynomial inner function is a unimodular monomial, so
\(\widehat a(z)=\lambda z^k\), contrary to the assumption.  Hence the
peripheral set has Haar measure zero, and the theorem applies.

Finally, Proposition 4.6 of \cite{DHSW} rules out \(\lambda\delta_0\) as a
weak\(^*\)-limit point; by shift-invariance, the same is true for every
unimodular translate \(\lambda\delta_k\).  Thus every finitely supported
weak\(^*\)-limit point falls under the non-monomial case above.

\section*{Limitations and Novelty Check}

This packet does not solve Question 5 in full.  The remaining case is a
power-bounded element \(a\in\ell_1(\Z)\) whose Fourier transform has unit
modulus on a positive-measure subset of \(\T\), while \(a\) is not a
unimodular point mass.  The previous attempt in this run isolated
power-boundedness but did not close even the null-peripheral case.

Local run indexes were searched for arXiv:1101.5696 and core terms including
``shift invariant preduals'', ``weak star'', ``ell\_1(Z)'', and
``power-bounded''.  The only direct run hit was the earlier unfinished attempt
\texttt{1101.5696\_weakstar\_limit\_power\_decay.md}; no solution packet for this subcase
was present.  A bounded web search for power-bounded Wiener-algebra and
convolution-power concentration keywords did not identify an exact full answer
during this pass.
