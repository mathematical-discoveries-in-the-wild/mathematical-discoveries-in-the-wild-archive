## Source Problem

Piotr Koszmider and Hugh Wark, *Large Banach spaces with no infinite
equilateral sets*, arXiv:2104.05330.

On page 3, after Theorem 3, the authors ask whether their spaces admit an
uncountable Auerbach system. More precisely, their Theorem 3 considers
renormings of `ell_1([0,1])` of the form

```text
||x||_T = ||x||_1 + ||T x||,
```

where `T: ell_1([0,1]) -> X` is injective and has separable range.

## Result

Every such separable-range renorming admits an uncountable Auerbach system.
In fact, injectivity is not needed for this conclusion.

## Proof Idea

Because `T` has separable range, the set `{T e_t : t in [0,1]}` is covered by
countably many balls of radius `1/2`. One ball contains uncountably many
basis images. Pair the corresponding coordinates and set
`u_i = e_{a_i} - e_{b_i}`. Then `||T u_i|| <= 1`, and on the block subspace
generated by the disjoint `u_i`, the operator `T` is dominated by the
`ell_1` norm. This makes each `u_i` attain its distance from the span of the
other blocks. Hahn-Banach then supplies norm-one functionals biorthogonal to
the normalized block vectors.

## Human Review Recommendation

Check the block-distance estimate:

```text
||u_i - y||_T >= ||u_i||_T   for y in span{u_j : j != i}.
```

This is the only real moving part. Once this estimate is confirmed,
Hahn-Banach gives the Auerbach functionals immediately.
