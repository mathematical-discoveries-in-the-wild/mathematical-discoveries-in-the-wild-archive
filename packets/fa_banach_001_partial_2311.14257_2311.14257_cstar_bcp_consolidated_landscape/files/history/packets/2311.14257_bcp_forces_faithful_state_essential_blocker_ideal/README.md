# BCP Forces a Faithful State and an Essential Blocker Ideal

Status: likely valid partial result; serious bridge analysis for the
remaining `BCP => UBCP` question.

Source target: arXiv:2311.14257, Question 3.

## Result

This packet proves two C*-specific necessary consequences of BCP.

1. If a C*-algebra `A` has BCP, then `A` admits a faithful state.
2. If `C` is a countable BCP witness from the exact
   BCP/countable-capture equivalence, then the closed two-sided ideal
   generated by `C` is essential in `A`.

Together with the previous packet
`2311.14257_separable_essential_ideal_ubcp_reduction`, this narrows the full
`BCP => UBCP` problem to one precise gap:

```text
Can an essential ideal generated by a countable BCP blocker fail to contain
enough separable/local countable structure to yield UBCP?
```

## Key Point

The tempting proof that BCP gives a separable essential ideal is false as
stated: an ideal generated by countably many elements inside a larger
nonseparable C*-algebra need not be separable. A rank-one projection in
`B(H)` for nonseparable `H` generates `K(H)`, which is nonseparable.

So the remaining issue is not essentiality; we get that. The issue is whether
the *BCP blocker property* gives more than ordinary countable ideal generation.
