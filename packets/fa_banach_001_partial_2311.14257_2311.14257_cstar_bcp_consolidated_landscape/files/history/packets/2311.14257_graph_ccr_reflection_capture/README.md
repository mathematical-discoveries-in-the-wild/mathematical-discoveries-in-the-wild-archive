# Graph CCR Reflection Capture for BCP

Status: likely valid partial result / new obstruction mechanism.

Source target: arXiv:2311.14257, Question 3, asking for a
noncommutative C*-algebra version of the commutative BCP/UBCP criterion.

## Result

Let `A` be a C*-algebra generated by an uncountable family of norm-one
elements `(u_v)_{v in V}` with two properties:

- every countable subset of `A` is contained in the C*-subalgebra generated
  by some countable subfamily of the `u_v`;
- for every `v in V`, there is an isometric automorphism `alpha_v` of `A`
  fixing all the other generators and sending `u_v` to `-u_v`.

Then `A` has countable capture. In particular `A` fails BCP.

The key application is Farah's graph/CCR algebra `B(G)`: for every
uncountable graph `G`, `B(G)` has countable capture and fails BCP.

## Stress-Test Application

Farah constructs a simple nuclear AM C*-algebra `B(G(N,A))` of density
continuum which has irreducible representations on both separable and
nonseparable Hilbert spaces. This algebra is a graph/CCR algebra with
uncountably many generators, so the reflection-capture theorem proves that it
fails BCP.

This is important for the remaining lane-19 wall: it handles a hard
separably represented, simple, nuclear, density-continuum test case that is
not covered by primitive topology, von Neumann methods, reduced groups, or a
literal unused tensor coordinate.

## Scope

This does not prove the full arbitrary C*-algebra classification. The
remaining simple density-continuum wall is now narrower: a positive or
negative resolution must avoid even this reflection mechanism, not only
literal tensor coordinates and saturation.
