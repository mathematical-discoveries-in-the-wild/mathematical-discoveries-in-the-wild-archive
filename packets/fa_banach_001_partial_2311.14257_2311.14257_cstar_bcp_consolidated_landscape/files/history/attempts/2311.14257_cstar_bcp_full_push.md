# Attempt: arXiv:2311.14257 C*-algebra BCP full-push

status: promoted partial result; full arbitrary C*-algebra characterization not solved

target: Question 3 in Huang--Kudaybergenov--Liu, asking for a noncommutative
version of the commutative C*-algebra criterion for BCP/UBCP.

## Starting point

The previous lane-19 packet proves the sharp trivial-field theorem:
`C_0(K,X)` has BCP iff `K` has a countable pi-basis and `X` has BCP, and
`C_0(K,X)` has UBCP when `K` has a countable pi-basis and `X` is separable.
In particular, for every nonzero separable C*-algebra `A`,
`C_0(K,A)` has BCP/UBCP iff `K` has a countable pi-basis.

## Literature calibration

Searches for the exact C*-algebra characterization still do not show a direct
answer.  The later paper arXiv:2412.07137 is highly relevant but not a full
answer to the 2311.14257 question: it proves that Calkin-type quotient Banach
algebras `B(X)/K(X)` fail BCP for `X` with a shrinking 1-unconditional basis,
in particular `B(H)/K(H)` for separable infinite-dimensional Hilbert `H`.

This creates a useful stress test for any proposed criterion:

- `B(H)` has UBCP by the cited operator-space theorem used in arXiv:2311.14257.
- `B(H)/K(H)` fails BCP by arXiv:2412.07137.
- A primitive-spectrum-only criterion cannot distinguish them, since both are
  simple in the separable infinite-dimensional Hilbert case.

## Route A: pure-state pi-basis

Candidate: `A` has BCP/UBCP iff the weak* pure state space `P(A)` has a
countable pi-basis.

This candidate passes the obvious coarse tests:

- For commutative unital `A=C(K)`, it is exactly Corollary 3.9 of the source
  paper.
- For separable C*-algebras, the state space is weak* metrizable, hence `P(A)`
  is second countable, and separability already gives UBCP.
- It is plausible for atomic separably acting von Neumann algebras and should
  fail for diffuse von Neumann algebras, matching Theorem 3.8.
- It can plausibly distinguish `B(H)` from the Calkin algebra, but this still
  needs proof.

Main obstruction: the commutative proof uses scalar bump functions supported
inside a chosen open set.  The direct noncommutative replacement

> every nonempty weak* open set `W` in `P(A)` contains the pure-state cozero
> support `{phi in P(A): phi(h)>0}` of some nonzero positive `h in A`

is false already for `A=M_n`.  In `M_n`, a nonzero positive `h` has cozero
set consisting of all vector states not orthogonal to `ran(h)`, which cannot
be placed inside an arbitrary small projective-neighborhood.  So symmetry of
the pure-state topology alone cannot reproduce the `C(K)` proof.

The more realistic positive mechanism would need both:

1. a countable pi-basis in `P(A)`, and
2. countably many local Hilbert-space directions inside the relevant
   irreducible representations, so that Kadison transitivity can manufacture
   centers matching the local action of a given `a in S_A`.

This may still be equivalent to a good noncommutative pi-basis notion, but the
plain pure-state pi-basis condition is not yet proved sufficient.

## Route B: extension/corona obstruction

arXiv:2412.07137 proves `B(H)/K(H)` fails BCP by recursively choosing tail
blocks where each proposed center still has quotient norm.  This is a genuine
noncommutative obstruction that is invisible to primitive spectra.

Possible abstraction: a corona algebra `M(I)/I` should fail BCP whenever `I`
has a countable approximate-unit decomposition with enough almost orthogonal
corners that every quotient norm can be recovered along infinitely many
disjoint corners.  This would extend the Calkin theorem from basis-tail
operators to sigma-unital stable ideals with rich quasicentral block
decompositions.  I do not yet have a clean proof for arbitrary C*-coronas.

## Route C: continuous C_0(X)-algebras

The negative half of the previous `C_0(K,X)` theorem extends cleanly to
continuous `C_0(X)`-algebras.

Let `A` be a continuous `C_0(X)`-algebra over a locally compact Hausdorff
space `X`.  Assume every nonempty open `U subset X` is locally nonzero in the
sense that `C_0(U)A != 0` (for example, all fibers are nonzero).  If `X` has
no countable pi-basis, then `A` cannot have BCP.

Proof idea: for countably many proposed centers `a_n`, the fiber-norm functions
`x -> ||a_n(x)||` are continuous, so the near-norm sets
`U_{n,k}={x: ||a_n(x)||>||a_n||-1/k}` are open.  Choose nonempty open `W`
containing none of them.  Pick `b in C_0(W)A` with `||b||=1`.  For each `n,k`,
some point outside `W` nearly norms `a_n`, and at that point `b` vanishes.
Hence `||b-a_n|| >= ||a_n||-1/k`; let `k -> infinity`.

Positive half likely needs a countable fiberwise sphere-net of global sections.
This is strictly stronger than arbitrary `C_0(X)`-algebra structure but covers
trivial fields and many continuous fields with a countable fundamental family.

Important correction: the same proof does not work for arbitrary
upper-semicontinuous `C_0(X)`-algebras, because superlevel near-norm sets of
upper-semicontinuous fiber norms need not be open.
