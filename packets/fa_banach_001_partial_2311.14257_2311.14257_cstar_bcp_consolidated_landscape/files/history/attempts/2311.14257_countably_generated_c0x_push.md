# Attempt: arXiv:2311.14257 countably generated continuous C0(X)-algebras

status: promoted partial result

target: Question 3 in Huang--Kudaybergenov--Liu, asking for a
noncommutative characterization of C*-algebras with BCP/UBCP.

## Candidate theorem

Let `A` be a continuous `C_0(X)`-algebra over a locally compact Hausdorff
space `X`, all fibers nonzero.  If `A` is countably generated as a
`C_0(X)`-algebra, then the following are equivalent:

1. `A` has BCP;
2. `A` has UBCP;
3. `X` has a countable pi-basis.

## Proof mechanism

The previous base obstruction gives `(1) -> (3)`.  For `(3) -> (2)`, reduce to
the fiber-net packet by constructing a countable localizable `1/64`-fiber-net.

Let `{a_n}` generate `A` as a `C_0(X)`-algebra.  Let `D` be the countable set
of noncommutative *-polynomials in the `a_n` with rational complex
coefficients.  For each `x`, the set `{d(x): d in D}` is dense in `A_x`
because the fiber is generated by `{a_n(x)}`.

For `d in D`, define a bounded section

`sigma_d(x)=d(x)/max(||d(x)||,1)`.

Continuity of `A` makes `x -> max(||d(x)||,1)^(-1)` continuous and bounded.
For `h in C_c(X)`, `h sigma_d = h g_d d in A`; so `sigma_d` is localizable.
For `a in A`, `(x,s)->||a(x)-s sigma_d(x)||` is locally continuous because
near any point one may localize `g_d d` by a scalar cutoff equal to `g_d` on a
small compact neighborhood.

If `u in S_{A_x}`, choose `d in D` with `||d(x)-u||<1/128`.  Then
`sigma_d(x)` is within `1/64` of `u`: if `||d(x)||<=1`, it is immediate; if
`||d(x)||>1`, radial normalization moves by at most another `1/128`.

This gives the countable localizable fiber-net needed for the previous packet.
