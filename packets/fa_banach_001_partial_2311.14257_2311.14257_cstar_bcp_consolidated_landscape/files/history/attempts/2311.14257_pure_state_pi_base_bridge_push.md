## Candidate

The attractive full classification candidate was:

```text
A has BCP/UBCP  iff  the weak* pure state space P(A) has a countable pi-base.
```

It matches the commutative theorem, is compatible with the separable and

## Why the Pure-State Bridge Did Not Close

The commutative proof relies on scalar bump functions supported inside a
chosen open set.  The direct noncommutative replacement remains false:

```text
Every nonempty weak* open W in P(A) contains {phi: phi(h)>0}
for some nonzero positive h in A.
```

Already in `M_n`, a nonzero positive `h` has cozero set all vector states not
orthogonal to `ran(h)`, so it cannot lie inside an arbitrary small projective
neighborhood.

Pure-state excision gives a better substitute: for a pure state `phi`, one can
choose positive contractions whose high-level states are forced into a given
basic neighborhood.  This repairs the topological part of the naive argument.
The remaining obstruction is directional:

- to cover an element `a`, one needs countably many local directions
  approximating `a e` or the GNS vector `pi_phi(a)xi_phi`;
- pure-state pi-base alone has not yet been shown to make the corresponding
  right ideals `cl(A e)` separable;
- Kadison transitivity gives finite-dimensional control in an irreducible
  representation, but not a countable global right-support net by itself.

Thus the current sharp missing implication is:

```text
pure-state countable pi-base
  => countable separable right-support net
  => UBCP.
```

Only the second arrow is proved in the promoted packet.

## Necessity Route Status

The reverse direction, BCP implying a pure-state countable pi-base, also
remains open.  From a BCP cover one gets countably many centers and norming
pure states for their `c_n^* c_n`, but turning those norming sets into a
pi-base again requires a noncommutative Urysohn/excision argument strong
enough to construct an element supported in a prescribed pure-state open set.
The high-level excision version is promising but did not yield the exact
escape inequality for arbitrary centers.

## Best Next Push

The best next target is not the raw pure-state topology, but the intermediate
support-net implication:

```text
Does pure-state countable pi-base force a countable family of pure-state
excisers e_n with separable right ideals cl(A e_n) and sup_n ||a e_n|| = ||a||?
```

A positive answer would combine immediately with the new right-support packet
to prove the desired UBCP direction.  A negative answer would likely produce
a counterexample to the plain pure-state pi-base criterion and identify the
right strengthened invariant.
