## Why This Is the Right Invariant

Pure states alone see `a* a`, but a covering center for a non-self-adjoint
element needs a direction.  The correct directional object is a transition
coefficient

```text
a |-> <pi(a) xi, eta>
```

or abstractly an extreme point of the dual unit ball.  This is why the
rank-one coefficient estimate in the Fell/type-I packets works.

The transition invariant now passes two boundary tests:

- commutative: `T(C(K)) ~= T x K`, so it is exactly the source pi-base
  condition;
- von Neumann: negative source cases have no countable transition pi-base.

## Failed Full Routes From This Push

### 1. Pure-State Excision Alone

Pure-state excision can localize a pure state against a finite set, but it
does not give a pre-countable family of centers independent of the element
being covered.  It also only gives diagonal information.  For general
non-self-adjoint `a`, the needed datum is a transition direction
`<pi(a)xi,eta>`, not only a state value on `a* a`.

### 2. Separable Support Nets

This route is already dead by the split-interval counterexample:

```text
C(K) has pure-state and transition countable pi-base and has UBCP,
but no nonzero positive e has separable cl(C(K)e).
```

So the positive proof cannot ask for separable principal right ideals.

### 3. Truncating Arbitrary BCP Centers

I tried to turn an arbitrary BCP cover `B(c_n, ||c_n||)` into bounded centers
by scaling or truncating the `c_n` through C*-polar/functional calculus.  The
basic obstruction is quantitative: from

```text
||u - c|| < ||c||
```

one gets a strict half-space type inequality, but the gap may go to zero with
`u` and `c`.  Scaling preserves membership in many model cases but does not
create a uniform off-origin gap.  I did not find a C*-specific truncation
which turns this into UBCP.

### 4. "No Transition Pi-Base Implies Capture"

The desired negative theorem would be:

```text
T(A) has no countable pi-base => A has countable capture => A fails BCP.
```

This is true in the primitive-ideal proof because an open primitive set gives
a genuine ideal element vanishing outside it.  For arbitrary weak-star open
sets in `T(A)`, the missing step is a noncommutative Urysohn theorem:

```text
given a transition-open W, find nonzero u in A whose transition coefficients
vanish off W.
```

This is true in commutative and continuous-trace/rank-one chart settings, but
not available for arbitrary C*-algebras.

## Sharp Remaining Lemma

The most precise next lemma is:

```text
Local transition bump lemma.
Assume T(A) has a countable pi-base by transition-open sets W_j with
localizable coefficient bumps v_j in A: outside W_j the bump is invisible,
and on W_j it behaves like a scalar/rank-one coefficient direction.
Then A has UBCP.
```

This is not just a rephrasing of UBCP if the bump condition is stated in
C*-terms:

- in `C(K)`, the bumps are scalar functions supported in open subsets of K;
- in Fell/type-I charts, the bumps are rank-one compact-operator sections;
- in separable-essential-ideal cases, the bumps are dense ideal centers;
- in von Neumann atomless or nonseparable atomic cases, the failure of such
  localizable patches is witnessed by singular/common-projection or
  disjoint-open arguments.
