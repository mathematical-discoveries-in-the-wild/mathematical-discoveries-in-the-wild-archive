## Breakthrough

The non-norming case is not a new covering problem.  It is exactly the
annihilator summand.

For an extension
\[
        0\to I\to A\to F\to0
\]
let
\[
        J=\operatorname{Ann}_A(I)=\{a\in A:aI=Ia=0\}.
\]
The canonical action of `A` on `I` has kernel `J`, and the Busby map
\[
        \tau:F\to Q(I)
\]
has kernel `q(J)`.  Since `F` is finite dimensional, `J` is finite dimensional.
Moreover `J` is a unital ideal with central unit, so it splits off
orthogonally.

Now enlarge the ideal from `I` to
\[
        I^\sharp=I\oplus J.
\]
This is still continuous trace with separable fibres, and it has countable
primitive pi-base iff `I` does.  The quotient
\[
        A/I^\sharp \cong F/q(J)
\]
has injective Busby map into `Q(I)`.  Because the quotient is finite
dimensional, injectivity gives a uniform lower bound
\[
        \|\tau(y)\|\ge m\|y\|.
\]
This implies that `I^\sharp` is uniformly norming in `A`.

Therefore the previous norming finite-quotient theorem applies to
`I^\sharp \triangleleft A`, and `A` has UBCP.
