## Result

For a discrete group `G`,

```text
C*_r(G) has BCP <=> C*_r(G) has UBCP <=> G is countable.
```

## Proof Sketch

If `G` is countable, then `C*_r(G)` is separable, so it has UBCP.

If `G` is uncountable, let `(a_n)` be countably many proposed centers.
Each element of `C*_r(G)` lies in `C*_r(H)` for some countable subgroup `H`:
approximate it by finitely supported group-algebra elements and take the
subgroup generated by the countable union of supports.  For countably many
centers, one countable subgroup `H` contains all of them.

Choose `g in G \ H`, and set `u=lambda_g`.  The canonical conditional
expectation

```text
E_H : C*_r(G) -> C*_r(H)
```

satisfies `E_H(u)=0` and `E_H(a_n)=a_n`.  Since `E_H` is contractive,

```text
||u-a_n|| >= ||E_H(u-a_n)|| = ||a_n||
```

for every `n`.  Thus the unitary `u` escapes every ball
`B(a_n,||a_n||)`.

## Why It Matters

For uncountable groups with simple reduced C*-algebra, this gives simple
one-point-primitive non-Type-I C*-algebras failing BCP, independent of the
von Neumann factor obstruction.  It is a clean ambient-norm visibility theorem:
the conditional expectation makes a countable generated subalgebra visible.
