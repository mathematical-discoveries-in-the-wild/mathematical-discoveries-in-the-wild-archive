# Compact-line extension problem answered by arXiv:2403.19327

status: `literature_already_answered (compact-line subproblem)`

## Source question

Maciej Korpalski and Grzegorz Plebanek, *Countable discrete extensions of
compact lines*, arXiv:2305.04565; published in *Fundamenta Mathematicae* 265
(2024), 75-93.

On PDF page 16, Section 7, Problem 7.1 asks whether it is relatively consistent
that every countable discrete extension `L` of every separable compact space
`K` of weight `omega_1` admits a bounded extension operator
`C(K) -> C(L)`.  The next sentence explicitly asks for the compact-line class.

## Answering paper

Antonio Aviles and Maciej Korpalski, *Barely alternating real almost chains and
extension operators for compact lines*, arXiv:2403.19327; published in
*RACSAM* 118 (2024), Paper 148,
<https://doi.org/10.1007/s13398-024-01651-7>.

Theorem 1.4 on PDF page 2 proves that under `MA(kappa)`, if `K` is a separable
compact line of weight `kappa`, then every countable discrete extension `L`
admits an extension operator `E:C(K)->C(L)` with norm at most three.  The
paragraph immediately following the theorem explicitly says that it solves
the problem posed in the source paper.

Thus `MA(omega_1)` supplies the requested relative consistency for all
separable compact lines of weight `omega_1`.

## Scope

The later paper answers the compact-line subproblem completely.  It explicitly
leaves open the broader Problem 7.1 for arbitrary separable compact spaces.
