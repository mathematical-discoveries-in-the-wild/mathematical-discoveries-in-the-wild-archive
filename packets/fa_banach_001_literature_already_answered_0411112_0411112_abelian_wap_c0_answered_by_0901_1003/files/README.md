# Literature Already Answered: Abelian WAP Groups

Status: `literature_already_answered`

## Source Question

- Source paper: Michael Megrelishvili, "Fragmentability and representations of flows", arXiv:math/0411112.
- Location: Section "Reflexive representations of (semi)groups", page 13 of the source PDF; TeX source near the paragraph beginning "It is also easy to show that every non-Archimedean..."
- Question: whether every abelian Hausdorff topological group, for example the additive group of a Banach space, is WAP.

## Supporting Answer

- Supporting paper: Itai Ben Yaacov, Alexander Berenstein, and Stefano Ferri, "Reflexive representability and stable metrics", arXiv:0901.1003, Math. Z. 267 (2011), 129--138.
- Location: Section 4, page 8 of the supporting PDF, especially Theorem 4.1.

Ben Yaacov--Berenstein--Ferri explicitly recall the Megrelishvili/Glasner--Megrelishvili problem asking for an abelian group that is not reflexively representable. They then prove Theorem 4.1: the additive group of the Banach space `c_0` is not reflexively representable.

In the terminology of the source question, WAP topological groups are exactly the groups whose topology is induced by WAP functions, equivalently the reflexively representable groups. Therefore the additive group of `c_0`, with its norm topology, is an abelian Hausdorff topological group that is not WAP. This gives a negative literature answer to the source question.

## Scope

This packet records a later literature answer, not a new proof from this run. It answers the abelian-Hausdorff WAP question negatively. It does not address unrelated fragmentability, Asplund-flow, or representation questions elsewhere in the source paper.
