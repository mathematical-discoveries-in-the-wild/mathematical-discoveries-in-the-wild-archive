# Full Result: Block-Measure Criterion for `c0` in Adequate-Family Spaces

## Source Question

- Source paper: Trond A. Abrahamsen, Vegard Lima, and Andre Martiny, *Delta-points in Banach spaces generated by adequate families*, arXiv:2012.00406.
- Location: final section, after the authors note that `h_{A,p}` contains an isometric copy of `c0` if there is an infinite set `E` with `|A cap E| <= 1` for all `A in mathcal A`.
- Question: "What is a natural condition on `mathcal A` such that `h_A` (or `h_{A,p}`) does not contain `c0`. That is, when does `h_{A,p}` have a boundedly complete basis?"

## Answer

For every adequate family `mathcal A` and every `1 <= p < infinity`, `h_{A,p}` contains `c0` if and only if `mathcal A` admits a uniformly bounded successive sequence of normalized finitely supported block measures.

Equivalently, `h_{A,p}` avoids `c0`, and hence its canonical unconditional basis is boundedly complete, exactly when every such normalized block-measure sequence has unbounded accumulated `mathcal A`-mass.

This recovers the paper's singleton sufficient condition as the special case `mu_n = delta_{i_n}`. The criterion is stronger because it also detects weighted block obstructions.

## Review Focus

Review should check two standard Banach-space inputs:

1. A copy of `c0` in a Banach space with a Schauder basis yields, after passing to a subsequence and a gliding-hump perturbation, a normalized block basic sequence equivalent to the `c0` basis.
2. For an unconditional basis, absence of `c0` is equivalent to bounded completeness of the basis.

Once those are accepted, the measure criterion follows by direct norm computation in `h_{A,p}`.

## Scope

This packet answers the `c0`/boundedly-complete-basis question from the source paper. It does not answer the later delta-point question asking whether `(V)`- or `(I)`-polyhedrality of `h_A` forces `h_A^*` to fail to have delta-points.
