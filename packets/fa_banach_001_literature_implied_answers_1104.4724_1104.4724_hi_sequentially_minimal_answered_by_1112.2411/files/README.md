# HI Sequentially Minimal Example Answered by 1112.2411

Status: `literature_implied_answer (final Problem (a) / type-(2) example only)`

Original source: Valentin Ferenczi and Christian Rosendal, "Banach spaces without minimal subspaces - Examples", arXiv:1104.4724.

Supporting source: Valentin Ferenczi and Thomas Schlumprecht, "Subsequential minimality in Gowers and Maurey spaces", arXiv:1112.2411.

## Identification

In the final "Problems and comments" section of arXiv:1104.4724, the authors ask for examples filling missing classes in the Ferenczi-Rosendal classification. In particular, Problem (a) asks:

```text
Find a HI space which is sequentially minimal.
```

This is the missing type-(2) row in the chart: "HI, tight, sequentially minimal".

## Supporting Answer

The abstract of arXiv:1112.2411 states that Ferenczi and Schlumprecht construct block sequences in every block subspace of a variant of the Gowers-Maurey space and that this implies the existence of a subsequentially minimal HI space. In the introduction they state explicitly:

```text
There exists a tight, HI, sequentially minimal space X_GM.
```

Thus arXiv:1112.2411 answers the arXiv:1104.4724 final Problem (a) and supplies the missing type-(2) example.

## Scope

This packet does not answer the other final problems in arXiv:1104.4724, including type (5b), type (5d), tight-with-constants uniformly inhomogeneous examples, or the exact type of the Argyros-Haydon space.
