\section*{Source question}

The source paper is K. Mahesh Krishna and P. Sam Johnson, \emph{Expansion of approximate Bessel sequences to approximate Schauder frames for Banach spaces}, arXiv:2102.03288.  After defining \(p\)-approximate Bessel sequences, the authors ask Question~2.12:

\begin{quote}
For which classes of Banach spaces, \(p\)-approximate Bessel sequences can be expanded to a \(p\)-ASF?
\end{quote}

\begin{center}
\includegraphics[width=.92\linewidth]{figures/open_problem_crop.png}
\end{center}

Here \(1\le p<\infty\).  A \(p\)-approximate Bessel sequence, abbreviated \(p\)-ABS below, is a pair \((\{f_n\},\{\tau_n\})\subseteq \X^*\times \X\) for which the analysis and synthesis maps
\[
 \theta_f:\X\to \lp,\qquad \theta_f x=(f_n(x))_n,
 \quad\text{and}\quad
 \theta_\tau:\lp\to \X,\qquad \theta_\tau(a_n)=\sum_{n=1}^{\infty}a_n\tau_n
\]
are bounded.  It is a \(p\)-approximate Schauder frame, abbreviated \(p\)-ASF, when additionally the frame operator \(S_{f,\tau}=\theta_\tau\theta_f\) is invertible.

\section*{Proof intuition}

There are two separate issues in the source question.  First, if the space already has one \(p\)-ASF, then any \(p\)-ABS can be completed: use a Parseval \(p\)-ASF \((a_n,\omega_n)\) for the identity, and add the vectors \((I-S_{f,\tau})\omega_n\) with the same functionals \(a_n\).  The added frame operator is exactly \(I-S_{f,\tau}\), so the enlarged pair has frame operator \(I\).

Second, the existence of one \(p\)-ASF is already known to be an operator-factorization condition through \(\ell^p\).  The characterization theorem of Krishna--Johnson, arXiv:2010.10514, says a \(p\)-ASF is the same as bounded maps \(U:\X\to\lp\) and \(V:\lp\to\X\) with \(VU\) invertible.  That is equivalent to saying that \(\X\) is isomorphic to a complemented subspace of \(\ell^p\).

\includegraphics[width=.92\linewidth]{figures/supporting_p_asf_characterization_crop.png}
\end{center}

\section*{The full answer}

\begin{theorem}\label{thm:classification}
Let \(1\le p<\infty\) and let \(\X\) be a Banach space.  The following are equivalent.
\begin{enumerate}
\item Every \(p\)-approximate Bessel sequence for \(\X\) can be expanded to a \(p\)-approximate Schauder frame for \(\X\).
\item \(\X\) admits a \(p\)-approximate Schauder frame.
\item \(\X\) is isomorphic to a complemented subspace of \(\lp\).
\end{enumerate}
Thus the answer to Question~2.12 in arXiv:2102.03288 is: exactly the Banach spaces isomorphic to complemented subspaces of \(\ell^p(\mathbb N)\).
\end{theorem}

\begin{proof}
We first prove \((1)\Leftrightarrow(2)\).

Assume (1).  The zero pair \(f_n=0\), \(\tau_n=0\) is a \(p\)-ABS for \(\X\).  By (1), it can be expanded to a \(p\)-ASF, so \(\X\) admits a \(p\)-ASF.

Conversely, assume that \(\X\) admits a \(p\)-ASF.  Replacing it by its canonical Parseval version, we may choose a \(p\)-ASF \((\{a_n\},\{\omega_n\})\) whose frame operator is \(I_\X\).  Explicitly, if \((\{g_n\},\{\omega_n\})\) has frame operator \(R\), then \((\{g_nR^{-1}\},\{\omega_n\})\) is a Parseval \(p\)-ASF.

Let \((\{f_n\},\{\tau_n\})\) be an arbitrary \(p\)-ABS for \(\X\), with bounded frame operator
\[
S:=S_{f,\tau}=\theta_\tau\theta_f.
\]
Define
\[
 h_n:=a_n,\qquad \rho_n:=(I_\X-S)\omega_n,\qquad n\in\mathbb N.
\]
The added analysis map is \(\theta_a:\X\to\lp\), which is bounded.  The added synthesis map is
\[
\theta_\rho=(I_\X-S)\theta_\omega:\lp\to\X,
\]
which is bounded because \(\theta_\omega\) and \(I_\X-S\) are bounded.  Therefore the enlarged pair is still a \(p\)-ABS.  Its frame operator is
\[
S_{(f,h),(\tau,\rho)}
=S_{f,\tau}+S_{h,\rho}
=S+(I_\X-S)S_{a,\omega}
=S+(I_\X-S)I_\X
=I_\X.
\]
Hence the enlarged pair is a Parseval \(p\)-ASF.  This proves (1).

It remains to prove \((2)\Leftrightarrow(3)\).  This is the standard operator characterization of \(p\)-ASFs from Krishna--Johnson, arXiv:2010.10514, Theorem~2.5: a pair is a \(p\)-ASF for \(\X\) if and only if there are bounded operators
\[
 U:\X\to\lp,\qquad V:\lp\to\X
\]
such that \(VU\) is bounded invertible, with \(f_n=h_nU\) and \(\tau_n=Ve_n\), where \((e_n)\) is the unit vector basis of \(\lp\) and \((h_n)\) is its coordinate functional sequence.

If \(\X\) has a \(p\)-ASF, choose such \(U,V\), and put \(A=VU\).  Then
\[
J:=UA^{-1}:\X\to\lp
\]
is an isomorphic embedding and \(VJ=I_\X\).  Consequently \(JV:\lp\to\lp\) is a bounded projection onto \(J(\X)\), so \(J(\X)\) is complemented in \(\lp\), and \(\X\) is isomorphic to that complemented subspace.

Conversely, suppose \(\X\) is isomorphic to a complemented subspace of \(\lp\).  Then there are bounded operators \(J:\X\to\lp\) and \(V:\lp\to\X\) such that \(J\) is an isomorphic embedding and \(VJ=I_\X\).  Define
\[
 f_n=h_nJ,\qquad \tau_n=Ve_n,\qquad n\in\mathbb N.
\]
The analysis map is \(J\), the synthesis map is \(V\), and the frame operator is \(VJ=I_\X\).  Thus \((\{f_n\},\{\tau_n\})\) is a Parseval \(p\)-ASF for \(\X\).  This proves (2), and completes the classification.
\end{proof}

\section*{Verification and limitations}

The expansion construction is a direct operator computation.  The only external theorem used is the p-ASF characterization from arXiv:2010.10514, Theorem~2.5, which is cited and included as `supporting\_paper\_2010.10514.pdf'.  The proof also shows that whenever expansion is possible, it can be done so that the enlarged \(p\)-ASF is Parseval.

This packet answers Question~2.12 as stated.  It does not address the separate earlier source question about finite tight expansion of translation ASFs in \(L^p(\mathbb R^d)\).
