# Literature-Implied Partial Subcase: star-\(n\)-finite ball coverings of \(\ell_1\)

Status: `literature_implied_answer (partial strengthened subcase)`

Source paper: Carlo Alberto De Bernardi, Jacopo Somaglia, and Libor Vesely,
"Star-finite coverings of Banach spaces", arXiv:2002.04308.

Supporting paper: Carlo Alberto De Bernardi, Tommaso Russo, Seyda Sezgek, and
Jacopo Somaglia, "Tilings and coverings by balls in \(\ell_1\)",
arXiv:2604.15092.

## Identification

The source paper asks in Problem 3.21 whether there exists a separable Banach
space admitting a star-finite covering by closed balls. It also asks related
questions for infinite-dimensional Frechet smooth spaces and spaces \(C(K)\).

The supporting paper does not solve the star-finite problem. It proves a
strictly stronger-finiteness negative result for a natural separable test
space: Theorem A(ii) states that if \(\kappa<2^{\aleph_0}\) is infinite and
\(n\in\mathbb N\), then \(\ell_1(\kappa)\) does not admit star-\(n\)-finite
coverings by balls of positive radius. In particular, \(\ell_1\) admits no
star-\(n\)-finite ball covering for any fixed \(n\).

## Scope

This is not a new result from this run and not a full answer to Problem 3.21.
A star-finite covering allows the number of intersecting neighbours to vary
from ball to ball, so the supporting theorem leaves open whether \(\ell_1\), or
any separable infinite-dimensional Banach space, admits a genuinely
star-finite covering by closed balls.
