\section*{Status}

This is a likely-valid partial structural packet for the questions in
Gonzalez--Salas-Brown, \emph{Two classes of operators related to the
perturbation classes problem}, arXiv:2302.04506.  It does not prove the
global equalities \(\SSing=\PhiS\) or \(\SCos=\PhiC\).  It gives two
structural advances and two sharp localization theorems:
\[
\begin{gathered}
\text{reflexive duality turns the source paper's one-way adjoint results into equivalences,}\\
Y\times Y\hookrightarrow Y \text{ forces }\PhiS(X,Y)=P\Phi_+(X,Y),\\
X\twoheadrightarrow X\times X \text{ forces }\PhiC(X,Y)=P\Phi_-(X,Y),\\
\text{subprojective/superprojective operators cannot witness strictness of }
\SSing\subset\PhiS
\text{ or }\SCos\subset\PhiC,\\
\text{extension/lifting properties force the remaining perturbation classes back to }
\SSing\text{ and }\SCos.
\end{gathered}
\]

\section*{Source Question}

The source paper asks whether the new classes \(\PhiS\) and \(\PhiC\) are
subspaces, operator ideals, or satisfy the same multiplication stability as
the perturbation classes, and asks whether the inclusions
\(\SSing(X,Y)\subset \PhiS(X,Y)\) and \(\SCos(X,Y)\subset \PhiC(X,Y)\) can be
strict.

\section*{Definitions}

For \(S,K\in \LL(X,Y)\), write
\[
(S,K)x=(Sx,Kx)\in Y\times Y,\qquad
[S,K](x_1,x_2)=Sx_1+Kx_2\in Y.
\]
Assume the corresponding semi-Fredholm classes are nonempty.  An operator
\(K\in\LL(X,Y)\) is \(\Phi\)-singular, \(K\in\PhiS(X,Y)\), if
\[
(S,K)\in \PhiPlus(X,Y\times Y)\quad\Longrightarrow\quad S\in\PhiPlus(X,Y)
\]
for every \(S\in\LL(X,Y)\).  It is \(\Phi\)-cosingular,
\(K\in\PhiC(X,Y)\), if
\[
[S,K]\in \PhiMinus(X\times X,Y)\quad\Longrightarrow\quad S\in\PhiMinus(X,Y)
\]
for every \(S\in\LL(X,Y)\).

We also use the operator versions of subprojectivity and superprojectivity
from \cite{GP24}.  An operator \(T\in\LL(X,Y)\) is
subprojective if every closed infinite-dimensional subspace \(M\subset X\) on
which \(T\) is an isomorphism contains a closed infinite-dimensional subspace
\(N\) such that \(T(N)\) is complemented in \(Y\).  Dually, \(T\) is
superprojective if every closed infinite-codimensional subspace \(M\subset Y\)
for which \(Q_MT\) is onto is contained in a closed infinite-codimensional
subspace \(N\subset Y\) such that \(T^{-1}(N)\) is complemented in \(X\).

We say that the pair \((Y,Y)\) has the qualitative linear extension property
if every bounded operator \(R:E\to Y\), defined on a closed subspace
\(E\subset Y\), has a bounded extension \(\widetilde R:Y\to Y\).  This is the
qualitative version of the standard linear extension property for pairs of
Banach spaces \cite{JS11}.  We say that \(X\) has the lifting property if,
for every quotient map \(q:E\to F\) and every \(A\in\LL(X,F)\), there is
\(T\in\LL(X,E)\) such that \(qT=A\).

\section*{Proof Intuition}

There are two mechanisms.

First, the source proof dualizes \([S,K]\) and \((S,K)\).  The only
obstruction to reversing it is that an arbitrary operator \(Y^*\to X^*\) need
not be an adjoint.  Reflexivity removes exactly that obstruction: every
operator \(A:Y^*\to X^*\) is \(S^*\) for \(S=A^*:X\to Y\), after the canonical
identifications \(X=X^{**}\) and \(Y=Y^{**}\).

Second, the square hypotheses let the new classes collapse to the usual
perturbation classes.  For \(\PhiS\), an embedding
\(U:Y\times Y\to Y\) converts a decoupled test \((S,K)\) into a single
operator \(VS+WK\).  If \(K\) belongs to the perturbation class \(P\Phi_+\),
then subtracting the perturbation \(WK\) leaves \(VS\) upper semi-Fredholm,
and the injectivity of \(V\) gives \(S\in\Phi_+\).  For \(\PhiC\), a quotient
map \(Q:X\to X\times X\) pulls a decoupled lower semi-Fredholm test
\([S,K]\) back to a single operator \(SU_1+KU_2\), and subtracting \(KU_2\)
forces \(S\) lower semi-Fredholm because \(U_1\) is onto.

Third, subprojective and superprojective operators remove the main local
escape route for a strict \(\PhiS\) or \(\PhiC\) counterexample.  If a
non-strictly singular operator is subprojective, then on some further
infinite-dimensional subspace its image is complemented.  This lets one
subtract a bounded multiple of \(K\) from a fixed upper semi-Fredholm operator
and create an infinite-dimensional kernel.  The lower semi-Fredholm dual
argument uses a complemented preimage of a large quotient kernel.

Fourth, the same subtraction trick works without complemented ranges when the
ambient space itself supplies extensions or liftings.  A perturbation-class
operator \(K\) which is not strictly singular would make
\(A(K|_M)^{-1}:K(M)\to Y\) nonextendable for every upper semi-Fredholm \(A\).
Dually, a perturbation-class operator which is not strictly cosingular would
force every lower semi-Fredholm test \(B\) to fail to lift through the quotient
map \(Q_MK\).  Thus injective-type codomains and projective-type domains are
too flexible to support strict witnesses.

\section*{Theorem 1: Reflexive Duality}

\begin{theorem}
Let \(X\) and \(Y\) be reflexive Banach spaces.  Whenever the displayed
classes are defined, for every \(K\in\LL(X,Y)\) one has
\[
K\in\PhiC(X,Y)\quad\Longleftrightarrow\quad K^*\in\PhiS(Y^*,X^*)
\]
and
\[
K\in\PhiS(X,Y)\quad\Longleftrightarrow\quad K^*\in\PhiC(Y^*,X^*).
\]
\end{theorem}

\begin{proof}
The forward implications from the dual classes to the original classes are
Proposition 4.9 of Gonzalez--Salas-Brown.  We prove the converses.

Assume first that \(K\in\PhiC(X,Y)\).  Let \(A\in\LL(Y^*,X^*)\) and suppose
\[
(A,K^*)\in\PhiPlus(Y^*,X^*\times X^*).
\]
Since \(X\) and \(Y\) are reflexive, there is an operator \(S\in\LL(X,Y)\)
such that \(A=S^*\).  Under the standard identification
\([S,K]^*=(S^*,K^*)\), the preceding hypothesis says
\([S,K]^*\in\PhiPlus\).  For any bounded operator \(T\), \(T^*\in\PhiPlus\)
if and only if \(T\in\PhiMinus\).  Hence
\([S,K]\in\PhiMinus(X\times X,Y)\).  Since \(K\in\PhiC(X,Y)\), it follows
that \(S\in\PhiMinus(X,Y)\).  Dualizing again gives
\(A=S^*\in\PhiPlus(Y^*,X^*)\).  Therefore \(K^*\in\PhiS(Y^*,X^*)\).

Now assume that \(K\in\PhiS(X,Y)\).  Let \(A\in\LL(Y^*,X^*)\) and suppose
\[
[A,K^*]\in\PhiMinus(Y^*\times Y^*,X^*).
\]
Again write \(A=S^*\) for some \(S\in\LL(X,Y)\).  Since
\((S,K)^*=[S^*,K^*]=[A,K^*]\), the semi-Fredholm duality
\(T\in\PhiPlus\) if and only if \(T^*\in\PhiMinus\) gives
\((S,K)\in\PhiPlus(X,Y\times Y)\).  Since \(K\in\PhiS(X,Y)\), we have
\(S\in\PhiPlus(X,Y)\).  Hence \(A=S^*\in\PhiMinus(Y^*,X^*)\), proving
\(K^*\in\PhiC(Y^*,X^*)\).
\end{proof}

\begin{corollary}
For reflexive \(X\) and \(Y\), the structural questions for
\(\PhiS(X,Y)\) are dual to the corresponding structural questions for
\(\PhiC(Y^*,X^*)\).  In particular, \(\PhiS(X,Y)\) is a linear subspace if
and only if \(\PhiC(Y^*,X^*)\) is a linear subspace, and the strictness of
\(\SSing(X,Y)\subset \PhiS(X,Y)\) is equivalent, under adjoints, to the
corresponding strictness question for \(\PhiC(Y^*,X^*)\) whenever the usual
strictly singular/strictly cosingular adjoint duality applies in the spaces
under consideration.
\end{corollary}

\section*{Theorem 2: One-Sided Square Characterization}

\begin{theorem}
Let \(X\) and \(Y\) be Banach spaces, and assume the relevant classes are
defined.
\begin{enumerate}
\item If \(Y\times Y\) is isomorphic to a subspace of \(Y\), then
\[
\PhiS(X,Y)=P\Phi_+(X,Y).
\]

\item If \(X\times X\) is isomorphic to a quotient of \(X\), then
\[
\PhiC(X,Y)=P\Phi_-(X,Y).
\]
\end{enumerate}
\end{theorem}

\begin{proof}
The inclusions \(\PhiS(X,Y)\subset P\Phi_+(X,Y)\) and
\(\PhiC(X,Y)\subset P\Phi_-(X,Y)\) are Proposition 3.2 and Proposition 4.2 of
the source paper.  It remains to prove the reverse inclusions.

Suppose \(Y\times Y\) embeds into \(Y\).  Let
\(U:Y\times Y\to Y\) be an into isomorphism, and write
\[
U(y_1,y_2)=Vy_1+Wy_2
\]
with \(V,W\in\LL(Y)\).  Let \(K\in P\Phi_+(X,Y)\), and assume
\((S,K)\in\PhiPlus(X,Y\times Y)\).  Since \(U\) is an into isomorphism,
\[
U(S,K)=VS+WK\in\PhiPlus(X,Y).
\]
By Proposition 2.2 of the source paper, \(WK\in P\Phi_+(X,Y)\).  The class
\(P\Phi_+(X,Y)\) is a linear subspace, so \(-WK\in P\Phi_+(X,Y)\).  Applying
the perturbation property to the upper semi-Fredholm operator \(VS+WK\) gives
\[
VS=(VS+WK)-WK\in\PhiPlus(X,Y).
\]
The operator \(V:y\mapsto U(y,0)\) is an into isomorphism, hence \(VS\) is
upper semi-Fredholm if and only if \(S\) is upper semi-Fredholm.  Therefore
\(K\in\PhiS(X,Y)\).

Now suppose there is a quotient map \(Q:X\to X\times X\), and write
\(Qx=(U_1x,U_2x)\).  Since \(Q\) is onto, \(U_1:X\to X\) is onto.  Let
\(K\in P\Phi_-(X,Y)\), and assume
\([S,K]\in\PhiMinus(X\times X,Y)\).  Then
\[
[S,K]Q=SU_1+KU_2\in\PhiMinus(X,Y).
\]
Here precomposition with a surjection preserves the range.  By Proposition
2.2, \(KU_2\in P\Phi_-(X,Y)\), and hence
\[
SU_1=(SU_1+KU_2)-KU_2\in\PhiMinus(X,Y).
\]
Since \(U_1\) is onto, \(SU_1(X)=S(X)\); therefore \(S\) has
finite-codimensional closed range exactly when \(SU_1\) does.  Thus
\(S\in\PhiMinus(X,Y)\), and \(K\in\PhiC(X,Y)\).
\end{proof}

\begin{corollary}
Under the hypotheses of Theorem 2, the source paper's subspace and
same-component multiplication-stability questions have positive answers.  More
precisely, \(\PhiS(X,Y)\) is a closed linear subspace and satisfies the
componentwise stability of Proposition 2.2 whenever \(Y\times Y\hookrightarrow
Y\), while \(\PhiC(X,Y)\) has the analogous properties whenever
\(X\twoheadrightarrow X\times X\).
\end{corollary}

\begin{corollary}
In particular, the \(\PhiS\) characterization applies to every codomain
\(Y\) which is isomorphic to its square, including the classical examples
listed in the source paper such as \(\ell_p\), \(L_p(\mu)\), \(c_0\), and
\(C[0,1]\).  The formulation also applies in the weaker situation where
\(Y\) merely contains a closed subspace isomorphic to \(Y\times Y\).  Dually,
the \(\PhiC\) characterization applies whenever \(X\) has a quotient
isomorphic to \(X\times X\), in particular whenever \(X\simeq X\times X\).
\end{corollary}

\section*{Theorem 3: Projective-Operator Barrier}

\begin{theorem}
Let \(X\) and \(Y\) be Banach spaces.
\begin{enumerate}
\item Suppose \(\Phi_+(X,Y)\neq\emptyset\).  If \(K\in\LL(X,Y)\) is a
subprojective operator, then
\[
K\in P\Phi_+(X,Y)\quad\Longleftrightarrow\quad
K\in\PhiS(X,Y)\quad\Longleftrightarrow\quad K\in\SSing(X,Y).
\]

\item Suppose \(\Phi_-(X,Y)\neq\emptyset\).  If \(K\in\LL(X,Y)\) is a
superprojective operator, then
\[
K\in P\Phi_-(X,Y)\quad\Longleftrightarrow\quad
K\in\PhiC(X,Y)\quad\Longleftrightarrow\quad K\in\SCos(X,Y).
\]
\end{enumerate}
\end{theorem}

\begin{proof}
We prove the first assertion.  The inclusions
\(\SSing(X,Y)\subset\PhiS(X,Y)\subset P\Phi_+(X,Y)\) are from the source
paper, so it is enough to show that a subprojective \(K\in P\Phi_+(X,Y)\) is
strictly singular.

Assume, to the contrary, that \(K\) is not strictly singular.  There is a
closed infinite-dimensional subspace \(M\subset X\) such that \(K|_M\) is an
isomorphism.  Since \(K\) is subprojective, \(M\) contains a closed
infinite-dimensional subspace \(N\) such that \(K(N)\) is complemented in
\(Y\).  Let \(P:Y\to K(N)\) be a bounded projection, and let
\[
R=(K|_N)^{-1}:K(N)\to N.
\]
Choose \(A\in\Phi_+(X,Y)\), and define \(L=ARP\in\LL(Y)\).  For every
\(n\in N\), \(LKn=An\).  Hence
\[
S=A-LK
\]
vanishes on \(N\), so \(S\notin\Phi_+(X,Y)\).

On the other hand, if \(K\in P\Phi_+(X,Y)\), then Proposition 2.2 of the
source paper gives \(LK\in P\Phi_+(X,Y)\), and since the perturbation class is
a linear subspace, also \(-LK\in P\Phi_+(X,Y)\).  Applying this perturbation
to the upper semi-Fredholm operator \(A\) gives
\[
A-LK=S\in\Phi_+(X,Y),
\]
a contradiction.  Therefore a subprojective \(K\in P\Phi_+(X,Y)\) must be
strictly singular.

For the second assertion, it is enough to show that a superprojective
\(K\in P\Phi_-(X,Y)\) is strictly cosingular.  Suppose not.  Then there is a
closed infinite-codimensional subspace \(M\subset Y\) such that
\(Q_MK:X\to Y/M\) is onto.  Since \(K\) is superprojective, there is a closed
infinite-codimensional subspace \(N\subset Y\), with \(M\subset N\), such that
\[
E=K^{-1}(N)
\]
is complemented in \(X\).  The map \(Q_NK:X\to Y/N\) is onto and has kernel
\(E\).  Because \(E\) is complemented, \(Q_NK\) has a bounded right inverse
\(R:Y/N\to X\).

Choose \(B\in\Phi_-(X,Y)\), and set
\[
T=R Q_N B\in\LL(X).
\]
Then \(Q_NKT=Q_NB\), so \(S=B-KT\) has range contained in \(N\).  Since \(N\)
has infinite codimension in \(Y\), \(S\notin\Phi_-(X,Y)\).  But if
\(K\in P\Phi_-(X,Y)\), Proposition 2.2 gives \(KT\in P\Phi_-(X,Y)\), and
hence \(-KT\in P\Phi_-(X,Y)\).  Applying this perturbation to
\(B\in\Phi_-(X,Y)\) gives
\[
B-KT=S\in\Phi_-(X,Y),
\]
again a contradiction.  Thus a superprojective \(K\in P\Phi_-(X,Y)\) must be
strictly cosingular.
\end{proof}

\begin{corollary}
Any counterexample to \(\SSing(X,Y)=\PhiS(X,Y)\) must be a
\(\Phi\)-singular operator which is not subprojective in the sense of
Gonzalez--Pello.  Any counterexample to \(\SCos(X,Y)=\PhiC(X,Y)\) must be a
\(\Phi\)-cosingular operator which is not superprojective.

Consequently, \(\SSing(X,Y)=\PhiS(X,Y)\) whenever every operator
\(X\to Y\) is subprojective, for instance when \(Y\) is subprojective
\cite[Corollary 2.4]{GP24}.  Similarly,
\(\SCos(X,Y)=\PhiC(X,Y)\) whenever every operator \(X\to Y\) is
superprojective, for instance when \(X\) is superprojective
\cite[Corollary 2.4]{GP24}.
\end{corollary}

\section*{Theorem 4: Extension--Lifting Obstruction}

\begin{theorem}
Let \(X\) and \(Y\) be Banach spaces.
\begin{enumerate}
\item Suppose \(\Phi_+(X,Y)\neq\emptyset\).  Let \(K\in P\Phi_+(X,Y)\).
If \(K\) is not strictly singular, then for every closed
infinite-dimensional subspace \(M\subset X\) on which \(K|_M\) is an
isomorphism and every \(A\in\Phi_+(X,Y)\), the operator
\[
A|_M (K|_M)^{-1}:K(M)\longrightarrow Y
\]
has no bounded extension to an operator \(Y\to Y\).  Consequently, if the
pair \((Y,Y)\) has the qualitative linear extension property, then
\[
P\Phi_+(X,Y)=\PhiS(X,Y)=\SSing(X,Y).
\]

\item Suppose \(\Phi_-(X,Y)\neq\emptyset\).  Let \(K\in P\Phi_-(X,Y)\).
If \(K\) is not strictly cosingular, then for every closed
infinite-codimensional subspace \(M\subset Y\) for which
\(Q_MK:X\to Y/M\) is onto and every \(B\in\Phi_-(X,Y)\), the operator
\[
Q_MB:X\longrightarrow Y/M
\]
does not lift through \(Q_MK\); equivalently, there is no \(T\in\LL(X)\) with
\[
Q_MKT=Q_MB.
\]
Consequently, if \(X\) has the lifting property, then
\[
P\Phi_-(X,Y)=\PhiC(X,Y)=\SCos(X,Y).
\]
\end{enumerate}
\end{theorem}

\begin{proof}
We prove the upper semi-Fredholm assertion first.  Let \(K\in P\Phi_+(X,Y)\),
and suppose that \(M\subset X\) is closed infinite-dimensional and that
\(K|_M\) is an isomorphism.  Then \(K(M)\) is closed in \(Y\).  Choose
\(A\in\Phi_+(X,Y)\).  If the operator
\[
R=A|_M(K|_M)^{-1}:K(M)\to Y
\]
extended to some \(L\in\LL(Y)\), then
\[
S=A-LK
\]
would vanish on \(M\).  Hence \(S\notin\Phi_+(X,Y)\), because its kernel is
infinite-dimensional.  On the other hand, Proposition 2.2 of the source paper
gives \(LK\in P\Phi_+(X,Y)\); since the perturbation class is a linear
subspace, \(-LK\in P\Phi_+(X,Y)\).  Perturbing the upper semi-Fredholm
operator \(A\) by \(-LK\) gives \(A-LK=S\in\Phi_+(X,Y)\), a contradiction.

If \((Y,Y)\) has the qualitative linear extension property and
\(K\in P\Phi_+(X,Y)\) is not strictly singular, then such an \(M\) exists and
the preceding obstruction is impossible.  Thus
\(P\Phi_+(X,Y)\subset\SSing(X,Y)\).  The reverse chain
\[
\SSing(X,Y)\subset\PhiS(X,Y)\subset P\Phi_+(X,Y)
\]
is Proposition 3.2 of the source paper, so equality follows.

For the lower semi-Fredholm assertion, let \(K\in P\Phi_-(X,Y)\), and suppose
that \(M\subset Y\) is closed infinite-codimensional and that
\(q=Q_MK:X\to Y/M\) is onto.  Choose \(B\in\Phi_-(X,Y)\).  If there were
\(T\in\LL(X)\) with \(qT=Q_MB\), then
\[
S=B-KT
\]
would have range contained in \(M\).  Since \(M\) has infinite codimension,
\(S\notin\Phi_-(X,Y)\).  But Proposition 2.2 gives \(KT\in P\Phi_-(X,Y)\),
and hence \(-KT\in P\Phi_-(X,Y)\).  Perturbing \(B\in\Phi_-(X,Y)\) by
\(-KT\) gives \(B-KT=S\in\Phi_-(X,Y)\), a contradiction.

If \(X\) has the lifting property and \(K\in P\Phi_-(X,Y)\) is not strictly
cosingular, then a quotient \(q=Q_MK\) as above exists; applying the lifting
property to the map \(Q_MB:X\to Y/M\) contradicts the obstruction just proved.
Thus \(P\Phi_-(X,Y)\subset\SCos(X,Y)\), and equality follows from the source
paper's chain
\[
\SCos(X,Y)\subset\PhiC(X,Y)\subset P\Phi_-(X,Y).
\]
\end{proof}

\begin{corollary}
The strictness problem has the following additional localization.  A strict
witness in \(\PhiS(X,Y)\setminus\SSing(X,Y)\) must force nonextension of the
operators \(A|_M(K|_M)^{-1}:K(M)\to Y\) described above.  A strict witness in
\(\PhiC(X,Y)\setminus\SCos(X,Y)\) must force nonlifting of the quotient maps
\(Q_MB\) through \(Q_MK\).

In particular, injective codomains, and more generally codomains \(Y\) for
which the pair \((Y,Y)\) has the qualitative linear extension property, have
\[
\SSing(X,Y)=\PhiS(X,Y)
\]
for every \(X\) with \(\Phi_+(X,Y)\neq\emptyset\).  Dually, projective
domains, and more generally domains with the lifting property, have
\[
\SCos(X,Y)=\PhiC(X,Y)
\]
for every \(Y\) with \(\Phi_-(X,Y)\neq\emptyset\).
\end{corollary}

\section*{Limitations and Review Notes}

The reflexivity assumption is essential for this proof, because it is used to
represent an arbitrary \(A:Y^*\to X^*\) as an adjoint \(S^*\).  Without such a
pre-adjoint representation, Proposition 4.9 need not reverse by this argument.

These theorems do not settle whether \(\SSing=\PhiS\) or whether
\(\SCos=\PhiC\).  Theorem 2 isolates the one-sided square mechanism already
visible in the source paper's combined perturbation-class arguments and
promotes it to explicit characterizations of the intermediate classes
\(\PhiS\) and \(\PhiC\).  In particular it strengthens the source paper's
square-isomorphism characterizations of these intermediate classes to
one-sided square hypotheses, and gives full positive answers to the source
subspace and Proposition 2.2 stability questions in those cases.  The global
operator-ideal question, with arbitrary outside domains and codomains, remains
beyond this packet.  Theorem 3 is a sharp projectivity obstruction theorem: it
does not rule out strictness, but it says that any strictness witness must
avoid the subprojective/superprojective operator classes introduced in
\cite{GP24}.  Theorem 4 adds an independent extension/lifting obstruction:
even a non-subprojective or non-superprojective witness must also defeat the
explicit extension or lifting tests in that theorem.

Bounded novelty check: the run indexes had no prior entry for arXiv
2302.04506 before this packet.  Web searches for exact \(\PhiS/\PhiC\)
duality and stability phrases found only the source arXiv paper.  Targeted
searches for \(\PhiS/\PhiC\) together with subprojective or superprojective
operators found only the Gonzalez--Pello paper introducing those operator
classes, not the triangular perturbation consequence above.  Crossref
metadata was checked for the older perturbation-class papers of Gonzalez
\cite{G03} and Weis \cite{W81}; no later source surfaced that states these
isolated one-sided square characterizations for \(\PhiS\) and \(\PhiC\).
A final search for \(\Phi\)-singular/\(\Phi\)-cosingular operators together
with extension, lifting, injective, and projective Banach-space terminology
found the source paper and background extension-property literature, but no
statement of the extension--lifting obstruction in Theorem 4.  The result is a
structural partial answer pending human review.
