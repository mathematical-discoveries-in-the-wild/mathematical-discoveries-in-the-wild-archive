# Exact extreme rays of the `K_3 P K_3` cone

Status: `partial_solution_likely_valid`

This packet answers the `n=3` case of the all-dimensions matrix-cone question
on printed page 38 of Alexander Mueller-Hermes, *Decomposable Pauli diagonal
maps and Tensor Squares of Qubit Maps*, arXiv:2006.14543.

For

\[
K_3=\frac23E_3-I_3,\qquad
\mathcal C_3=\{P\in M_3(\mathbb R):P\geq_{\rm ew}0,\ K_3PK_3\geq_{\rm ew}0\},
\]

the cone has exactly **78 extreme rays**. Under the symmetries generated by
independent row and column permutations, transposition, and
`P -> K_3 P K_3`, these split into three orbits of sizes `36`, `36`, and `6`.
Integral representatives are displayed and proved exhaustive in the packet.

The proof is a finite exact polyhedral argument. The verifier enumerates all
`binom(18,8) = 43,758` possible active-facet bases using SymPy rational
arithmetic. No floating-point decision is used.

Files:

- `solution_packet.pdf`: human-readable theorem and proof
- `main.tex`: packet source
- `source_paper.pdf`: original arXiv paper
- `figures/open_problem_crop.png`: source question on printed page 38
- `code/verify_n3_extreme_rays.py`: exact exhaustive verifier

Run the verifier from this directory with:

```bash
conda run --no-capture-output -n sandbox python code/verify_n3_extreme_rays.py
```

The general problem for arbitrary `n` remains open.
