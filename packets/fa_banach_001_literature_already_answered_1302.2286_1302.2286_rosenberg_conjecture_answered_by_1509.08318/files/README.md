# Rosenberg conjecture status in 1302.2286

Status: literature_already_answered.

Source/open-problem paper: Ben Hayes, "An `l^p`-Version of von Neumann
Dimension for Banach Space Representations of Sofic Groups II",
arXiv:1302.2286.

Supporting answer paper: Aaron Tikuisis, Stuart White, Wilhelm Winter,
"Quasidiagonality of nuclear C*-algebras", arXiv:1509.08318.

## Identification

On source PDF page 32, Hayes writes that if every amenable group were in the
class `C`, then `C^*_lambda(Gamma)` would be quasidiagonal for every amenable
group, and states that this is Rosenberg's conjecture, then open.

Tikuisis--White--Winter later explicitly state in the abstract of
arXiv:1509.08318 that they confirm Rosenberg's conjecture: discrete amenable
groups have quasidiagonal C*-algebras.

## Scope

This packet records only the later answer to the Rosenberg-conjecture status
remark inside the source paper. It does not claim to settle Hayes's closing
Conjectures 4.1--4.3 on `l^p`-dimension.
