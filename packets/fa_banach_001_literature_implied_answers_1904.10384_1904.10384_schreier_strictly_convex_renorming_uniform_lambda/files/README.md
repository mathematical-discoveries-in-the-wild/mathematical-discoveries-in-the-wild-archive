# Schreier space can be renormed to have the uniform lambda-property

Status: `literature_implied_answer (full answer to source Question II)`.

Source paper: Kevin Beanland and Hung Viet Chu, *The Schreier Space Does Not Have the Uniform $\lambda$-property*, arXiv:1904.10384.

Source question: Section 5, Future Research, item (II): "Can $X$ be renormed to have the uniform $\lambda$-property?"

Answer: yes. Since the Schreier space is separable, it admits an equivalent strictly convex norm. Under any strictly convex norm every unit vector is an extreme point of the unit ball, and every point of the unit ball has a lambda-decomposition with coefficient at least `1/2`.

Provenance note: this is not claimed as a new deep Schreier-specific theorem. It is the direct application of the standard separable strictly-convex renorming construction; the packet includes the short construction for completeness.
