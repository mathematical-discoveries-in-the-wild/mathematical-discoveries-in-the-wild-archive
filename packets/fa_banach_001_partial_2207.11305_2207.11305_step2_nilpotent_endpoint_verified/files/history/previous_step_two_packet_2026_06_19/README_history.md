# Previous Step-Two Packet

This folder preserves the active packet that was superseded on 2026-06-19.

The previous packet proved the complete nilpotency-step-two endpoint theorem for
connected, simply connected, step-two nilpotent Lie groups with arbitrary
bracket-generating left-invariant Carnot--Caratheodory structure.

The new active packet keeps that theorem and adds Lorentz propagation, weak-seed
reductions, Cartan-group reductions, and higher-step structured examples. The
general endpoint conjecture remains open in the active packet.
