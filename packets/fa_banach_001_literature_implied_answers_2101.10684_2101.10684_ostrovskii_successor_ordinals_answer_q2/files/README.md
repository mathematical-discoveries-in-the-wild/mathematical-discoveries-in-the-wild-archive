# Literature-Implied Answer to Silber's Quasi-Reflexive Order Question

Status: `literature_implied_answer (source Question 2; source Question 1 remains open)`.

Source paper: Zdenek Silber, *Weak$^*$ derived sets of convex sets in duals of non-reflexive spaces*, arXiv:2101.10684, J. Funct. Anal. 281 (2021), no. 12, Paper No. 109259.

Supporting paper: Mikhail I. Ostrovskii, *Weak$^*$ closures and derived sets for convex sets in dual Banach spaces*, arXiv:2112.04670.

## Identification

Silber's second question in Section 3 asks whether a non-reflexive quasi-reflexive Banach space can have convex subsets of the dual with order higher than `omega + 1`.

Ostrovskii's later theorem applies to every non-reflexive Banach space and produces convex subsets whose weak-star derived-set order is any countable successor ordinal. Since non-reflexive quasi-reflexive spaces are included, choosing a successor ordinal above `omega + 1` gives a positive answer to Silber's second question.

This packet is not an original solution from the run. It records that the question is answered by later literature after direct identification.

## Scope

The first source question, asking whether a convex subset of the dual of a separable Banach space can have limit ordinal order, is not answered by this packet. Ostrovskii's paper explicitly records that this limit-order question remains unanswered.
