# Literature Already Answered: Diestel's LWCG Versus WCG Question

Source: A. Aviles, A. J. Guirao, S. Lajara, J. Rodriguez, and P. Tradacete, *Weakly compactly generated Banach lattices*, arXiv:1512.08628.

Supporting paper: A. Aviles, J. Rodriguez, and P. Tradacete, *The free Banach lattice generated by a Banach space*, arXiv:1706.08147.

Status: `literature_already_answered`.

## Identification

The source records Diestel's question: whether every lattice weakly compactly generated Banach lattice is weakly compactly generated as a Banach space.

The supporting paper explicitly restates this as Problem 1.1 and gives a negative answer. Corollary 5.5 says that, for any uncountable set `Gamma` and `1 < p <= 2`, `FBL[ell_p(Gamma)]` is LWCG but is not isomorphic to a subspace of any WCG Banach space. Hence it is not WCG.

## Scope

This settles the general LWCG = WCG question negatively. It does not settle every related structural question in the source, such as whether an LWCG Banach lattice can contain a sublattice isomorphic to `ell_infty`.
