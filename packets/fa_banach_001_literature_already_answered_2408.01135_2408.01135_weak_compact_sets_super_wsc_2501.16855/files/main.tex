\section*{Original question}

Silber asks whether there exists a Banach space in which every relatively
weakly compact set is relatively super weakly compact, but which is not weakly
sequentially complete \cite{Silber2024}.

\section*{Idea of the proof}

This packet does not claim an original proof. The identification is direct:
the source asks whether the ``weakly compact sets are super weakly compact''
property can occur without weak sequential completeness, while the supporting
paper proves that the property forces weak sequential completeness.

The supporting proof starts from a nontrivial weakly Cauchy sequence. A
Rosenthal summing subsequence produces uniformly bounded functionals with a
triangular pattern. Differences of the weakly Cauchy sequence form a relatively
weakly compact convex set, and the triangular functionals witness the finite
tree obstruction from the Lancien--Raja characterization of super weak
compactness. Thus any failure of weak sequential completeness manufactures a
relatively weakly compact set that is not relatively super weakly compact
\cite{Silber2025}.

\section*{Theorem}

\begin{theorem}
There is no Banach space in which every relatively weakly compact set is
relatively super weakly compact but which is not weakly sequentially complete.
\end{theorem}

\begin{proof}
This is exactly the contrapositive of Theorem 2.8 in arXiv:2501.16855: if
relatively weakly compact sets in a Banach space $X$ are relatively super
weakly compact, then $X$ is weakly sequentially complete \cite{Silber2025}.
\end{proof}
