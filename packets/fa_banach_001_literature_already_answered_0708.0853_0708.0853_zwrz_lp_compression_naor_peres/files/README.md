# Literature Already Answered: `L_p` Compression of `Z wr Z`

Status: `literature_already_answered`

## Source Question

- Source paper: Assaf Naor and Yuval Peres, "Embeddings of discrete groups and the speed of random walks", arXiv:0708.0853.
- Location: Question 7.9, page 21 of the source PDF.
- Question: Evaluate `alpha_p^*(Z wr Z)` for `1 < p < 2`.

## Supporting Answer

- Supporting paper: Assaf Naor and Yuval Peres, "`L_p` compression, traveling salesmen, and stable walks", arXiv:0904.4728, Duke Math. J. 157 (2011), 53-108.
- The abstract and introduction state

```text
alpha_p^*(Z wr Z) = max{ p/(2p-1), 2/3 }.
```

For `1 < p < 2`, the maximum is `p/(2p-1)`, with equality at `p=2`.
Thus the supporting paper gives the requested evaluation for the full range
asked in Question 7.9.

answered in later literature by the same authors. Other questions from the
