# Partial result: uniform epsilon-plus center for arbitrary Banach lattices

Status: `partial_result_likely_valid`

Source paper: Timur Oikhberg and Pedro Tradacete, *Almost band preservers*,
arXiv:1610.02557.

## Target question

The source paper asks whether every `epsilon`-band preserving bounded operator
on a Banach lattice belongs to the `epsilon`-center. Equivalently, does the
local domination constant from source Theorem 2.10 admit one finite choice
uniformly over the unit ball?

## Result

This packet proves the endpoint question up to an arbitrarily small loss:

```text
If T in B(X) is epsilon-BP, then for every eta > 0,
T belongs to (epsilon + eta)-Z(X).
```

More quantitatively, if `M = ||T|| > 0`, then one may take

```text
lambda >= 8 M^2 (M+1) / eta^2.
```

Thus the only remaining obstruction to the full converse is endpoint
attainment: whether the required uniform `lambda` can be chosen at `eta = 0`.

## Proof idea

The source theorem already proves that an exactly disjoint functional cannot
see `Tx` by more than `epsilon`. The new point is a quantitative version:
if a positive norm-one functional satisfies

```text
<x*, |x|> <= alpha,
```

then

```text
<x*, |Tx|> <= epsilon + O(sqrt(alpha)).
```

If a large positive part
`(|Tx| - lambda |x|)_+` survived above `epsilon + eta`, a supporting
functional can be restricted to the part where `|Tx| > lambda |x|`. On that
part it almost annihilates `|x|`, with `alpha <= ||T||/lambda`. Taking
`lambda` of order `eta^{-2}` contradicts the quantitative disjoint-functional
bound.

## Relationship to the previous packet

The earlier packet
`1610.02557_c0_al_epsilon_center_converse` proves the exact endpoint
`epsilon-Z` conclusion for `C_0(K)` spaces and AL-spaces. This packet is
orthogonal and broader: it works for every Banach lattice, but with
`epsilon + eta` rather than exact `epsilon`.

## Limitations

This is not a full solution of the source problem. The proof gives no bound
on `lambda` as `eta -> 0`; indeed the displayed quantitative bound blows up
like `eta^{-2}`. A full result would need an endpoint compactness/attainment
argument or a different method.
