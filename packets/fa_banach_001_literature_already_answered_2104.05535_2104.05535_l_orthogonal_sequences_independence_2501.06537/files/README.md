# Literature answer: L-orthogonal sequences versus L-orthogonal elements

## Status

`literature_already_answered`

The remaining headline question from arXiv:2104.05535 is answered by the
separate later paper arXiv:2501.06537.

## Original source

Antonio Aviles, Gonzalo Martinez-Cervantes, Abraham Rueda Zoca,
*L-orthogonal elements and L-orthogonal sequences*, arXiv:2104.05535.

Question 1.1 asks whether every Banach space with an `L`-orthogonal sequence
has an `L`-orthogonal element in its bidual. The same source proves positive
consistency results and negative consistency for the weak-star-closure variant,
but leaves open whether a negative answer to Question 1.1 is consistent.

## Supporting answer

Antonio Aviles, Gonzalo Martinez-Cervantes, Alejandro Poveda, Luis Saenz,
*A Banach space with L-orthogonal sequences but without L-orthogonal elements*,
arXiv:2501.06537.

The abstract says the paper gives a definitive answer to Question 1.1 of
Aviles--Martinez-Cervantes--Rueda. Its Main Theorem states that the existence
of a Banach space with an `L`-orthogonal sequence but without `L`-orthogonal
elements in its bidual is independent of ZFC.

## Scope

This is not a new proof packet. It records that the original question is
already settled in the later literature: in some models every such sequence
has an `L`-orthogonal bidual element, while in other models there is a Banach
space counterexample.

The separate existing run packet for arXiv:2501.06537 concerns that later
paper's quantitative Question 2 about epsilon-strong Q-measures; it does not
record the headline answer to arXiv:2104.05535.
