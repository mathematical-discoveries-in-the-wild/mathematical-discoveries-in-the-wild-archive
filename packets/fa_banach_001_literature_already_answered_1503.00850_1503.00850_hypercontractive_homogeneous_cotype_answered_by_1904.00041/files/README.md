# Hypercontractive homogeneous cotype conjecture answered by Carando--Marceca--Sevilla-Peris

Status: `literature_already_answered`

## Source Question

Daniel Carando, Andreas Defant, and Pablo Sevilla-Peris, *Some polynomial versions of cotype and applications*, arXiv:1503.00850.

The source introduces hypercontractive homogeneous cotype and conjectures that it is equivalent to ordinary cotype: a Banach space has hypercontractive homogeneous cotype `q` if and only if it has cotype `q`.

## Supporting Answer

Daniel Carando, Felipe Marceca, and Pablo Sevilla-Peris, *Hausdorff-Young type inequalities for vector-valued Dirichlet series*, arXiv:1904.00041.

The introduction explicitly says that the paper proves Rademacher cotype is equivalent to the hypercontractive homogeneous cotype defined in arXiv:1503.00850. The exact result is Theorem `cotimplicapolcot` in Section 3: for `2 <= q < infinity`, `X` has cotype `q` if and only if a polynomial cotype inequality holds with constants `C^m` for all degree `m` polynomials. Its Remark `expo` says the `L_q` norm may be replaced by other `L_r` norms with only exponential change in the constants. Restricting to homogeneous polynomials and taking `r=2` gives precisely hypercontractive homogeneous cotype `q`.

Thus the source conjecture is answered affirmatively in the later paper.
