# Rapid+ P+ extension of the sliding-jump theorem

Status: partial result, likely valid.

Source paper: Paolo Leonetti, "Regular matrices of unbounded linear operators", arXiv:2201.13059.

Open question addressed: In the closing remarks on page 24, the source asks whether Theorem 4.1 holds for the larger class of rapid+ P+-ideals, and whether that condition characterizes the latter class; it also asks an analogue for Theorem 3.6.

Result: The first extension question has a positive answer. Theorem 4.1 remains valid when the countably generated hypothesis on the ideal `J` is replaced by `J` being rapid+ and P+.

Proof mechanism: the published proof uses countable generation only to choose indices `s_n` so that `{s_n}` is `J`-positive and the nth selected index lies in the nth good set. For rapid+ P+ ideals, P+ gives a `J`-positive pseudointersection of the decreasing good sets, and rapid+ thins it so that its nth element lies beyond the finite exceptional set for the nth good set. The original sliding-jump block construction then runs unchanged.

Scope: This packet does not prove the characterization of rapid+ P+-ideals, the Lorentz-Macphail analogue for Theorem 3.6, or the `Z`-ideal matrix-class characterization from the same closing paragraph.
