## Summary

In arXiv:0901.3029, Q8 asks whether an Asplund Banach space with a
Markushevich basis whose functionals form a norming subspace must be weakly
compactly generated.

Hajek--Russo--Somaglia--Todorcevic, arXiv:2007.14201, explicitly restate the
same Godefroy problem as Problem 1.1 and immediately state that their main
result is a negative answer: there exists an Asplund space with a 1-norming
M-basis that is not WCG.
