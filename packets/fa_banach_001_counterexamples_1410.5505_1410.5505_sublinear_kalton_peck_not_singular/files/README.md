# Sublinear Kalton-Peck centralizers are not singular

Status: **candidate full negative answer / counterexample; human review required**.

This packet answers the open question at the end of Section 6, page 25, of arXiv:1410.5505. For every `0 < r < 1`, the Kalton-Peck centralizer associated with

`phi_r(t)=t` for `0 <= t <= 1` and `phi_r(t)=t^r` for `t>1`

is **not singular** on `ell_2`. An explicit disjoint block subspace `Y` is constructed on which the centralizer differs from a linear diagonal map by a bounded `ell_2`-valued map. Consequently, the quotient map of the associated twisted sum is an isomorphism over an infinite-dimensional closed subspace.
