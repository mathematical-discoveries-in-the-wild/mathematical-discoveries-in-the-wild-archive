# Full literature answer to the superreflexive lineability problem

Status: `literature_already_answered`.

Source target: G. Botelho, D. Diniz, and D. Pellegrino, *Lineability
of the set of bounded linear non-absolutely summing operators*,
arXiv:0811.0092v5 / JMAA 357 (2009), 171-175, Problem 1.1.

Decisive later paper: D. Kitson and R. M. Timoney, *Operator ranges
and spaceability*, JMAA 378 (2011), 680-686,
DOI `10.1016/j.jmaa.2010.12.061`, Theorem 4.1.

The source asks whether, for every infinite-dimensional superreflexive Banach
space `E`, every infinite-dimensional Banach space `F`, and every `p>=1`,
the complement `L(E,F) \ Pi_p(E,F)` is lineable. The source paper proves this
when either `E` has a complemented unconditional basic subspace or `F`
contains an unconditional basic sequence, leaving the simultaneous
pathological case.

Kitson--Timoney explicitly state that their Theorem 4.1 answers the original
problem. They prove the stronger simultaneous statement

`K(E,F) \ union_{1<=p<infinity} Pi_p(E,F)` is spaceable.

Thus one closed infinite-dimensional subspace works for every finite `p`, and
all exceptional cases left by arXiv:0811.0092 are covered.
