# Supporting-paper access record

Decisive paper:

- Derek Kitson and Richard M. Timoney, *Operator ranges and spaceability*,
  Journal of Mathematical Analysis and Applications 378 (2011), no. 2,
  680-686.
- DOI: https://doi.org/10.1016/j.jmaa.2010.12.061
- Accepted manuscript:
  https://www.tara.tcd.ie/tara8/server/api/core/bitstreams/85cf04e3-66ac-423a-a7eb-b2c42cad6414/content
- Exact result: Section 4, Theorem 4.1 (accepted-manuscript pages 7-8).

The accepted manuscript was opened and its full text inspected on
2026-07-22. The paragraph immediately before Theorem 4.1 explicitly says
that the result answers Puglisi--Seoane-Sepulveda Problem 2.4. The theorem
states that, for infinite-dimensional Banach spaces `X,Y` with `X`
superreflexive,

`K(X,Y) \ union_{1<=p<infinity} Pi_p(X,Y)`

is spaceable in `K(X,Y)`.

Two local download attempts returned HTML anti-bot challenge pages rather
than a PDF. Those invalid artifacts were discarded. This access note is
kept instead so no HTML file is misrepresented as the supporting article.
