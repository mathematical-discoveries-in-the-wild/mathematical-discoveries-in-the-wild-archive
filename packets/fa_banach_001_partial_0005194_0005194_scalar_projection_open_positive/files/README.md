# Scalar finite-rank projection subcase of Akemann--Shell--Weaver

Status: partial result, likely valid.

Source target: arXiv:0005194, C. A. Akemann, G. C. Shell, and
N. Weaver, "Locally nonconical convexity".

Open problem transcribed from the source: let \(Q\) be a closed, bounded,
strictly convex subset of a Hilbert space \(X\), and let
\(T:X\to \mathbb R^n\) be a bounded linear map. Is
\(T|_Q:Q\to T(Q)\) necessarily open?

Result recorded here: the answer is yes for \(n=1\), and in fact strict
convexity and closedness are not needed. If \(Q\) is any bounded convex
subset of a normed space and \(T:X\to\mathbb R\) is linear, then
\(T|_Q:Q\to T(Q)\) is open. More generally, for finite-dimensional targets
the same elementary argument gives openness at all image points \(y\) for
which a fixed homothetic copy \((1-\alpha)y+\alpha T(Q)\) is a relative
neighborhood of \(y\) in \(T(Q)\).

What remains open: the source problem for \(n\ge 2\), especially at
non-conical/cuspidal boundary points of the finite-dimensional image
\(T(Q)\).
