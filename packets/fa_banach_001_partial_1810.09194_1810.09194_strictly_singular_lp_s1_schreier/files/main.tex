\section*{Definitions}

The first Schreier family is
\[
  \Sone=\{F\subset\mathbb N:\ F\text{ finite and } |F|\leq\min F\}.
\]
An operator $T:X\to Y$ is $\Sone$-strictly singular if for every normalized
basic sequence $(x_n)$ in $X$ and every $\varepsilon>0$ there exist
$F\in\Sone$ and $0\ne x\in[x_n]_{n\in F}$ such that
\[
  \|Tx\|\leq\varepsilon\|x\|.
\]
Androulakis, Dodos, Sirotkin, and Troitsky introduced this hierarchy of
Schreier strictly singular operators in \cite{ADST}.

\section*{Proof Intuition}

The Kadec-Pelczynski dichotomy says that a seminormalized weakly null sequence
in $L_p$ has a subsequence modeled either on $\ell_p$ or on $\ell_2$.  Thus,
after passing to subsequences, both a candidate domain sequence and its image
under a bounded operator have only two possible spreading geometries.

If a strictly singular $T$ were not $\Sone$-strictly singular, then on some
basic sequence all Schreier-admissible blocks would stay bounded below.  After
the Kadec-Pelczynski reductions, the domain looks like $\ell_a$ and the image
looks like $\ell_b$, with $a,b\in\{p,2\}$.  If $a=b$, $T$ is an isomorphism on
the generated infinite-dimensional span, contradicting strict singularity.  If
$a\ne b$, boundedness forces $a<b$, and then long averages over
$\{N+1,\ldots,2N\}\in\Sone$ have image/domain ratio of order
$N^{1/b-1/a}\to0$.

\section*{Main Theorem}

\begin{theorem}\label{thm:lp-s1}
Let $1<p<\infty$.  Every strictly singular operator
$T:\Lp\to\Lp$ is $\Sone$-strictly singular.
\end{theorem}

\begin{proof}
The case $p=2$ is immediate because strictly singular operators on Hilbert
space are compact, hence finitely strictly singular and therefore
$\Sone$-strictly singular.  Assume $p\ne2$.

Let $(u_n)$ be a normalized basic sequence in $\Lp$ and fix
$\varepsilon>0$.  Since $\Lp$ is reflexive, by passing to a subsequence we may
assume that $(u_n)$ is weakly convergent.  If its weak limit is zero, set
$v_n=u_n$.  Otherwise pass to a further subsequence and set
$v_n=u_{2n+1}-u_{2n}$.  The sequence $(v_n)$ is weakly null and
seminormalized: normalized basic sequences are separated in norm.  After
normalizing and passing to a subsequence, the Kadec-Pelczynski dichotomy
\cite{KP} gives an exponent $a\in\{p,2\}$ such that
\[
  (x_n):=\left(v_n/\|v_n\|\right)
\]
is equivalent to the unit vector basis of $\ell_a$.

If a subsequence of $(Tx_n)$ tends to zero in norm, then a singleton Schreier
set gives the desired vector.  Thus, after passing to a subsequence, assume
that $(Tx_n)$ is seminormalized.  Since $(x_n)$ is weakly null, so is
$(Tx_n)$.  Applying Kadec-Pelczynski again, pass to a further subsequence so
that $(Tx_n)$ is equivalent to the unit vector basis of $\ell_b$ for some
$b\in\{p,2\}$.

We claim first that $a\ne b$.  If $a=b$, then for all finitely supported
scalar families $(\alpha_n)$ the two equivalences give constants $c,C>0$ with
\[
  \left\|T\sum \alpha_n x_n\right\|
  =\left\|\sum \alpha_n Tx_n\right\|
  \geq c\|(\alpha_n)\|_{\ell_a}
  \geq C^{-1}\left\|\sum \alpha_n x_n\right\|.
\]
Thus $T$ is bounded below on $[x_n]$, contradicting strict singularity.

Next, boundedness of $T$ forces $a<b$.  Indeed, the equivalences and
boundedness imply
\[
  \|(\alpha_n)\|_{\ell_b}
  \lesssim
  \left\|\sum \alpha_n Tx_n\right\|
  \leq
  \|T\|\left\|\sum \alpha_n x_n\right\|
  \lesssim
  \|(\alpha_n)\|_{\ell_a}
\]
for all finitely supported $(\alpha_n)$, which is possible only when
$a\leq b$.  Since $a\ne b$, we have $a<b$.

Choose $N$ so large that the constants in the two basis equivalences give
\[
  \frac{\left\|T\sum_{n=N+1}^{2N} x_n\right\|}
       {\left\|\sum_{n=N+1}^{2N} x_n\right\|}
  \leq \varepsilon .
\]
This is possible because the numerator is $O(N^{1/b})$, the denominator is
$\Omega(N^{1/a})$, and $1/b-1/a<0$.  The set
$F=\{N+1,\ldots,2N\}$ belongs to $\Sone$, so this gives the required Schreier
small vector for the normalized weakly null sequence $(x_n)$.

It remains only to transfer this vector back to the original basic sequence
$(u_n)$.  If $v_n=u_n$, this is immediate, using that $\Sone$ is spreading
under subsequences.  If $v_n=u_{2n+1}-u_{2n}$, then a Schreier set
$F\in\Sone$ in the difference sequence uses at most the original coordinates
\[
  \{2n,2n+1:\ n\in F\},
\]
after the preceding subsequence selection.  This set has cardinal at most
$2|F|$ and minimum at least $2\min F$, hence it is again in $\Sone$ after
passing back to the original indices.  Therefore the required vector lies in
the span of an $\Sone$-admissible subset of the original sequence.  Since
$(u_n)$ and $\varepsilon$ were arbitrary, $T$ is $\Sone$-strictly singular.
\end{proof}

\begin{corollary}\label{cor:no-dependency}
There is no operator
\[
  T\in\SSing(L_p[0,1])\setminus \Sone(L_p[0,1])
\]
for $1<p<\infty$.
\end{corollary}

\begin{proof}
This is exactly Theorem \ref{thm:lp-s1}.
\end{proof}

\section*{Relation to the Approximate-Factorization Problem}

The preceding conditional packet proved that the finite-dimensional-sum
factorizations in Remark 6.11, in the reflexive exponent range
$1<r<s<\infty$, force $\Sone$-strict singularity.  Corollary
\ref{cor:no-dependency} shows that this necessary condition is automatic for
strictly singular operators on $L_p[0,1]$.  Hence the Schreier-rank obstruction
cannot yield a counterexample to Remark 6.11 on classical $L_p$.

The original approximate-factorization question remains open from this
argument: $\Sone$-strict singularity is only a necessary condition, not proved
here to be sufficient for approximate $(\ell_r,\ell_s)$ factorization.
