# Literature Status: `1309.3501` Problems 1 and 2

Status: `literature_already_answered` for the intended graph formulation of
Problem 1 and the algebraic characterization problem.

Source paper: Agelos Georgakopoulos, Sebastian Haeseler, Matthias Keller,
Daniel Lenz, Radoslaw K. Wojciechowski, *Graphs of finite measure*,
arXiv:1309.3501.

Supporting paper: Simon Puchert, *Characterizations of canonically
compactifiable graphs via intrinsic metrics and algebraic properties*,
arXiv:1803.01942.

## Identification

The source paper asks in Section 8.3, page 53:

- Problem 1: whether condition `(D)` implies condition `(C)` in general.
- Problem 2: whether `\widetilde D` being an algebra implies canonical
  compactifiability.

Puchert's abstract states that the paper answers questions from the
Georgakopoulos--Haeseler--Keller--Lenz--Wojciechowski paper. Theorem 1 and
Corollary 1 prove that a graph is canonically compactifiable if and only if
the underlying set is totally bounded with respect to every finite-measure
intrinsic metric. Theorem 2 and Corollary 2 prove that a graph is canonically
compactifiable if and only if the finite-energy space is an algebra.

## Scope Note

Puchert formulates the answer for graphs `G=(X,b)`, i.e. the standard
zero-killing graph setting used in the metric characterization. This is the
formulation Puchert explicitly identifies as answering the questions from the
source paper. The packet does not claim to settle every possible reading with
an arbitrary killing term `c`.
