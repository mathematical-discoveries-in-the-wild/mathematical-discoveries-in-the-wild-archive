# Literature-Implied Negative Answer: `c_0` Spreading Models Versus Asymptotic-`c_0`

Run: `fa_banach_001`

Source paper: D. Freeman, E. Odell, B. Sari, and B. Zheng,
*On spreading sequences and asymptotic structures*, arXiv:1607.03587.

Supporting paper: S. A. Argyros and P. Motakis,
*On the complete separation of asymptotic structures in Banach spaces*,
arXiv:1902.10092.

Result type: `literature_implied_answer (negative answer to first question)`.

Status: literature-implied partial answer, pending human review.

## Original Problem

Problem 4.5 of arXiv:1607.03587 asks: if `ell_1` does not embed into `X` and
every spreading model generated by weakly null normalized sequences in `X` is
equivalent to the unit vector basis of `c_0`, must `X` contain an
asymptotic-`c_0` subspace? It also asks whether `X` must contain a subspace
`Y` with `Y^*` separable.

## Literature-Implied Answer

The first question has a negative answer by the Argyros--Motakis space
`Z = X_iw^*` from arXiv:1902.10092.

Their Theorem A constructs `X_iw` reflexive, hence `Z` is reflexive and
contains no copy of `ell_1`. Their Theorem C says every normalized weakly null
sequence in `Z` has a subsequence generating a `c_0` spreading model. Therefore
every spreading model generated by a weakly null normalized sequence in `Z` is
`c_0`: once a sequence generates a spreading model, all of its subsequences
generate the same spreading model.

Argyros--Motakis also state explicitly that `X_iw^*` has no asymptotic-`c_0`
subspace. Thus `Z` satisfies the hypotheses of Problem 4.5 but fails the
asymptotic-`c_0` subspace conclusion.

This does not answer the separable-dual-subspace question. In fact the example
is reflexive with a separable predual/dual pair, so it is not a negative answer
to that second question.
