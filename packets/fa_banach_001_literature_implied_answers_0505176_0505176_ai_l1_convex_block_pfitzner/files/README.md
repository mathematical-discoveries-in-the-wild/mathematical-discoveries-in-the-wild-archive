# Literature-implied answer: asymptotically isometric \(\ell^1\) and convex block compactness

Status: `literature_implied_answer (ZFC variant of Question 3)`.

Source paper: Marianne Morillon, *A new proof of James' sup theorem*, arXiv:math/0505176.

Source question: Question 3 on PDF page 4 asks whether Valdivia/Bourgain-Diestel/Bourgain-Fremlin-Talagrand's result persists, in `ZF` or `ZFC`, when "does not contain an isomorphic copy of \(\ell^1(\mathbb N)\)" is weakened to "does not contain asymptotically isometric copies of \(\ell^1(\mathbb N)\)".

## Answer Scope

The `ZFC` version is answered affirmatively by Pfitzner's Proposition 11 in *Boundaries for Banach spaces determine weak compactness*, Invent. Math. 182 (2010), 585-604.

A later arXiv paper, Jimenez-Sevilla and Moreno, *A note on James' theorem for weak compactness*, arXiv:1508.00496, records this exact consequence in its final remarks: Pfitzner proved that if a Banach space \(E\) does not contain asymptotically isometric copies of \(\ell^1\), then \((B_{E^*},w^*)\) is convex block compact.

Thus Morillon's Question 3 has a positive `ZFC` answer after this literature identification.

## Limitations

This packet does not settle the fully choiceless `ZF` version of Question 3. It also does not answer Morillon's Question 2 about whether weak* block compactness and weak* convex block compactness differ in general; arXiv:1508.00496 explicitly still calls that open.
