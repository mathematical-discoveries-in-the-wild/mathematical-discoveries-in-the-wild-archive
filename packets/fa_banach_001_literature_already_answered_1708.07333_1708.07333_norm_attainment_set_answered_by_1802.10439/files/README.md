# Literature Already Answered: norm-attainment-set characterization

Status: literature_already_answered.

Source/open-problem paper: Debmalya Sain, "On extreme contractions and the
norm attainment set of a bounded linear operator", arXiv:1708.07333.

Supporting answer paper: Debmalya Sain, "On the norm attainment set of a
bounded linear operator and semi-inner-products in normed spaces",
arXiv:1802.10439.

## Identification

The source paper proves a Hilbert-space characterization of the norm
attainment set \(M_T\), then states in Remark 2.1 that for bounded linear
operators between general Banach spaces, a complete characterization of
\(M_T\) remains open.

The later supporting paper explicitly says that it answers the open question
from Sain's earlier norm-attainment-set paper. Its abstract says it obtains a
complete characterization of the norm attainment set of a bounded linear
operator between any two real normed spaces. Its introduction restates the
question and says the result gives a complete solution, and Theorem 2.3 gives
the semi-inner-product characterization.

## Scope

This clears the real-normed-space version of the source question. The source
paper was stated over real or complex scalars; the supporting theorem is stated
for real normed spaces. A complex analogue may require separate checking.
