# Verification Code

`rectangle_union_smoke.py` documents an exploratory finite-translate search for possible counterexamples to the full Remark 4.16(1) question using axis-aligned rectangles for `A` and finite point sets for `B,C`.

The search is not used in the proof and does not constitute evidence for the theorem beyond a sanity check. The packet proof is the fiberwise reduction to the one-dimensional Proposition 4.15 from the source paper.
