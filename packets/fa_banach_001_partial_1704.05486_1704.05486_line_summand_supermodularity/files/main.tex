\section*{Question Restated}

The source paper proves that
\[
  \vol_n(A+B+C)+\vol_n(A)\ge \vol_n(A+B)+\vol_n(A+C)
\]
for compact convex \(A,B,C\subset\R^n\), and also proves the one-dimensional
variant
\[
  \vol_1(A+B+C)+\vol_1(\conv A)\ge
  \vol_1(A+B)+\vol_1(A+C)
\]
for arbitrary compact \(A,B,C\subset\R\). Remark 4.16(1) asks whether, in
higher dimension, the first inequality remains true when only \(A\) is assumed
convex and \(B,C\) are arbitrary compact sets.

\section*{Proof Intuition}

The one-dimensional proof works because, after translating \(B\) and \(C\), an
intersection point of \(A+B\) and \(A+C\) is forced to lie between two points of
\(A\), hence in \(\conv A\). In higher dimension this order argument is lost.
However, if \(B\) and \(C\) both lie on the same line, the missing order
reappears on every fiber parallel to that line. A convex set has interval
fibers in every direction, so the source paper's one-dimensional inequality can
be applied fiber by fiber and then integrated by Fubini.

\section*{Main Result}

\begin{theorem}[fiber-convex line-summand supermodularity]
\label{thm:fiber}
Let \(E\subset\R^n\) be a one-dimensional linear subspace. Let
\(A\subset\R^n\) be compact and assume that every fiber of \(A\) parallel to
\(E\) is either empty or a compact interval. Let \(B,C\) be compact subsets of
affine lines parallel to \(E\). Then
\[
  \vol_n(A+B+C)+\vol_n(A)\ge \vol_n(A+B)+\vol_n(A+C).
\]
\end{theorem}

\begin{corollary}
In particular, Remark 4.16(1) of \cite{FMMZ2018} has an affirmative answer in
all dimensions whenever the arbitrary compact summands \(B\) and \(C\) are
contained in a common affine line. Namely, if \(A\subset\R^n\) is compact and
convex and \(B,C\) are compact subsets of one affine line, then
\[
  \vol_n(A+B+C)+\vol_n(A)\ge \vol_n(A+B)+\vol_n(A+C).
\]
\end{corollary}

\section*{Proof}

We first record the one-dimensional input, which is Proposition 4.15 of
\cite{FMMZ2018} in the special case where the first set is already an interval.

\begin{lemma}
\label{lem:oned}
Let \(I\subset\R\) be a compact interval, possibly empty, and let
\(S,T\subset\R\) be compact. Then
\[
  \vol_1(I+S+T)+\vol_1(I)
  \ge \vol_1(I+S)+\vol_1(I+T).
\]
\end{lemma}

\begin{proof}
If \(I=\emptyset\), the assertion is trivial. Otherwise \(\conv I=I\), so this
is exactly Proposition 4.15 of \cite{FMMZ2018}. The source proof is elementary:
translate \(S,T\) so that \(\max S=0=\min T\), use
\((I+S)\cup(I+T)\subset I+S+T\), and observe that
\((I+S)\cap(I+T)\subset I\).
\end{proof}

We now prove Theorem \ref{thm:fiber}. By translating \(B\) and \(C\)
independently, we may assume that \(B,C\subset E\); all four volumes in the
desired inequality are unchanged by such translations. Choose an orthogonal
decomposition
\[
  \R^n=H\oplus E,
\]
where \(H=E^\perp\), and identify each fiber parallel to \(E\) with a subset of
\(\R\). For \(y\in H\), write
\[
  A_y=\{t\in E:y+t\in A\}.
\]
By hypothesis, \(A_y\) is empty or a compact interval. Since \(B,C\subset E\),
the corresponding fibers satisfy
\[
  (A+B)_y=A_y+B,\qquad
  (A+C)_y=A_y+C,\qquad
  (A+B+C)_y=A_y+B+C.
\]
Applying Lemma \ref{lem:oned} to the one-dimensional compact interval \(A_y\)
for every \(y\in H\) gives
\[
  \vol_1((A+B+C)_y)+\vol_1(A_y)
  \ge
  \vol_1((A+B)_y)+\vol_1((A+C)_y).
\]
The involved sets are compact, hence Lebesgue measurable, and the section
functions are measurable. Integrating this pointwise inequality over
\(H\) and using Fubini's theorem yields
\[
  \vol_n(A+B+C)+\vol_n(A)\ge
  \vol_n(A+B)+\vol_n(A+C).
\]
This proves the theorem. If \(A\) is convex, then every line-parallel fiber of
\(A\) is an interval, giving the stated corollary.
\(\square\)

\section*{Limitations and Novelty Check}

The theorem does not address arbitrary higher-dimensional compact summands
\(B,C\). It also does not address the separate fractional-power inequality
\(\vol_n^{1/n}\) in Remark 4.16(2), which the source already disproves by an
explicit convex-body example.

Bounded novelty checks were performed against the run's lightweight indexes,
the source TeX, and web searches for the exact supermodularity expression. The
searches also used phrases such as ``volume of Minkowski sums supermodular''
and ``convexification effect Minkowski summation Remark 4.16.'' These checks
found the original EMS/arXiv paper and later work on star-shaped sets and
fractional Brunn--Minkowski inequalities, but no exact statement of this
line-summand fiber theorem. Novelty confidence is therefore bounded rather than
exhaustive.

of Remark 4.16(1).
