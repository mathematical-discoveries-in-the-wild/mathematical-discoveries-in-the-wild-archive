# 0406264 Infinite Equilateral-Set Problem Answered

Status: `literature_already_answered` for the infinite-dimensional final-remarks problem only.

Source paper: K. J. Swanepoel, *Equilateral sets in finite-dimensional normed spaces*, arXiv:math/0406264.

Supporting paper: E. Glakousakis and S. Mercourakis, *Examples of infinite dimensional Banach spaces without infinite equilateral sets*, arXiv:1502.02500.

## Identification

In the final remarks, Swanepoel asks:

> Does there exist an infinite equilateral set in any separable infinite-dimensional normed space?

This packet records the negative literature answer to that universal question.

## Answer Evidence

Glakousakis and Mercourakis give a separable infinite-dimensional Banach space with no infinite equilateral sets. Their abstract states that they construct an infinite-dimensional separable Banach space, not isomorphic to a subspace of `l_1`, with no infinite equilateral sets. Their final corollary obtains such an example by taking the summands `X_n = l_p^n`, `p>2`, in their general construction.

They also note in the introduction that the broader question whether every infinite-dimensional Banach space contains an infinite equilateral set had already been answered negatively by Terenzi via an equivalent norm on `l_1`.

Since every Banach space is a normed space, and the constructed example is separable and infinite dimensional, it is a counterexample to the exact universal assertion in Swanepoel's problem.

## Scope

This packet does not address the finite-dimensional problems in Swanepoel's paper, including Kusner's questions for `l_p^n`, the strict convex upper-bound questions, Petty's example, or the conjecture that every `n`-dimensional normed space has an equilateral set of size at least `n+1`.
