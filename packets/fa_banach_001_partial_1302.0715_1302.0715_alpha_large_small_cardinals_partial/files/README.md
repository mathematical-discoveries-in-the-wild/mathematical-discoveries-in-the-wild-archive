# Partial Result: Alpha-Large Families Below the Continuum

Status: `partial`

## Source Problem

- Source paper: Spiros A. Argyros and Pavlos Motakis, "$\alpha$-Large Families and Applications to Banach Space Theory", arXiv:1302.0715.
- Location: page 14 of the source PDF, after Proposition 2.24.
- Question: If `kappa` is an infinite cardinal which is not omega-Erdos and `alpha < omega_1` is countable, does there exist an `alpha`-large, hereditary, compact family of finite subsets of `kappa`?

## Result

For every infinite cardinal `kappa <= 2^{aleph_0}` and every countable ordinal `alpha`, there exists an `alpha`-large, hereditary, compact family of finite subsets of `kappa`.

The proof restricts Argyros--Motakis's continuum family `G_alpha` on the Cantor set to an arbitrary subset of cardinality `kappa`. Heredity, compactness, and `alpha`-largeness all pass to the restriction.

## Scope

This is a small-cardinal partial answer. It proves the question for every infinite `kappa <= 2^{aleph_0}`. It does not address cardinals above the continuum, where the original question remains open from this argument.
