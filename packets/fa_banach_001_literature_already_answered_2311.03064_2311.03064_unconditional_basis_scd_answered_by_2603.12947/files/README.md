# Literature Already Answered: unconditional basis SCD question

Status: `literature_already_answered`.

Source/open-problem paper:

- Johann Langemets, Marcus Loo, Miguel Martin, and Abraham Rueda Zoca, "Slicely countably determined points in Banach spaces", arXiv:2311.03064.

Supporting answer paper:

- Marcus Loo and Yoel Perreau, "A Banach space with an unconditional basis which is not slicely countably determined", arXiv:2603.12947.

Identification:

The source paper states in its introduction that it is an open question whether every Banach space with an unconditional basis is an SCD space. The later paper explicitly restates this as Question 1.2 and then says it will provide a negative answer. Its Theorem 1.3 states that the positive unit ball of the binary tree space is not SCD and, in particular, that the binary tree space has an unconditional basis and fails to be SCD.

Scope:

This answers the unconditional-basis SCD-space question negatively. It is not a new result from this run; it is a later-literature answer recorded to prevent rediscovery work on arXiv:2311.03064.
