# Scattered Orbit Closures Give Invariant Strict Convex Renormings

Status: `partial_result_likely_valid`

Source paper: Michal Doucha, *Invariant strictly convex renormings*, arXiv:2508.16182v1.

Targeted source questions: Questions 7.1 and 7.2 ask whether a single surjective linear isometry, equivalently a `Z`-action, or more generally a countable amenable group action, on a separable Banach space must admit an invariant strictly convex renorming.

Result proved here: if a group acts on a compact metrizable space `K` and every point orbit closure is scattered, then the induced action on `C(K)` admits an invariant strictly convex renorming. In particular, every single homeomorphism of `[0,1]` induces a strictly-convex-renormable action on `C[0,1]`.

Why this is useful: Doucha proves `C(K)` cases when orbit closures are minimal and amenability supplies fully supported invariant measures. This packet covers the opposite, scattered-orbit side using Lancien's invariant LUR theorem for separable RNP spaces. It rules out a natural `C[0,1]` single-isometry counterexample route to Question 7.1.
