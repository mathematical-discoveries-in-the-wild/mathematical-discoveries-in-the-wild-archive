# Finite one-point rigidification for Lipschitz-free spaces

- Run: `fa_banach_001`
- Lane: 18
- Agent: `agent_lane_18`

- Source: Marek Cuth, Michal Doucha, and Tamas Titkos, *Isometries of Lipschitz-free Banach spaces*, arXiv:2402.08266.
- Source target: Question 3 in Section 8.2, asking whether every metric space embeds isometrically into a Lipschitz-free rigid space with only one additional point.
- Status: `partial_result_likely_valid`

## Result

Every finite metric space embeds isometrically into a Lipschitz-free rigid metric
space after adding exactly one point.

The construction is explicit up to a generic choice.  Given a finite metric
space `M`, choose a large `R` and distances `r_x` from the new point `p` to
each `x in M` inside a very short interval `(R,R+eta)`, algebraically
independent over the field generated by the old distances.  This gives a
metric extension.  The preserved-extreme graph of the extension is the old
preserved-extreme graph plus the full star from `p`.

For each old preserved edge `{x,y}`, the three edges `{p,x}`, `{x,y}`, `{y,p}`
form a cone triangle.  The source paper's admissibility axioms force any
linear isometry of the free space to induce a cycle-preserving edge bijection
with constant length ratios on each triangle.  Algebraic independence prevents
such a bijection from moving a radial edge to an old edge or to a different
radial edge.  Hence every undirected preserved-extreme edge is fixed.  Directed
cycle preservation then forces a single global orientation sign, so the only
linear isometries are `Id` and `-Id`, both induced by the identity metric
dilation with sign.

## Scope

This strictly strengthens the earlier packet
`solutions/partial/2402.08266_one_point_cone_rigidification/`, which covered
finite spaces whose preserved-extreme graph is 2-connected.  The new result
covers all finite metric spaces.

It still does not settle the full Question 3 for arbitrary infinite metric
spaces.  The proof uses a finite algebraic-independence choice over the finite
old distance field and the finite no-betweenness characterization of preserved
extreme molecules.
