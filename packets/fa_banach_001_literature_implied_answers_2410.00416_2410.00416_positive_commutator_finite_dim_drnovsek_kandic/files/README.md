# Literature-implied answer: finite-dimensional positive commutator subcase

Status: `literature_implied_answer (partial subcase)`

Source problem: arXiv:2410.00416, Madou--Remizov--Vafadar, *Open problems in one-parameter operator semigroups theory*, Problem "Positive commutator problem" in the positivity section.

Supporting theorem: Roman Drnovsek and Marko Kandic, *Positive operators as commutators of positive operators*, arXiv:1707.00882 / Studia Math. 245 (2019), Corollary 3.2.

## Identification

The source asks whether every positive quasinilpotent compact operator `C` on a Banach lattice can be written as `C = AB - BA` with `A,B` positive and one of `A,B` compact.

Drnovsek--Kandic prove that a positive matrix `C` is a commutator of positive matrices if and only if `C` is nilpotent. In a finite-dimensional Banach lattice, positive operators are positive matrices after choosing the lattice atoms, every operator is compact, and quasinilpotent is equivalent to nilpotent. Hence the source problem has an affirmative answer for finite-dimensional Banach lattices.

## Scope

This is not a new partial proof and not a full answer to the source problem. It is a provenance-only identification of a finite-dimensional subcase already implied by the supporting literature. The infinite-dimensional compact/quasinilpotent Banach-lattice problem remains open except for the classes covered by Drnovsek--Kandic and the other literature cited in the source comments.
