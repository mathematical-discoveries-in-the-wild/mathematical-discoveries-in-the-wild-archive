# Literature Answer: `c_0` Does Not Embed Into `F(ell_2)`

Status: `literature_already_answered`

Source problem: Marek Cuth, Michal Doucha, and Przemyslaw Wojtaszczyk,
*On the structure of Lipschitz-free spaces*, arXiv:1505.07209. In Section
`Open problems`, after proving that `F(M)` is weakly sequentially complete
for every subset `M` of finite-dimensional Euclidean space, the authors ask:

```text
Is it true that c_0 -> F(ell_2)?
```

Supporting answer: Ramon J. Aliaga, Camille Nous, Colin Petitjean, and
Antonin Prochazka, *Compact reduction in Lipschitz free spaces*,
arXiv:2004.14250. Corollary 2.2 proves that `F(X)` is weakly sequentially
complete whenever `X` is superreflexive. Immediately after the corollary the
authors state that, in particular, `F(ell_2)` is weakly sequentially complete
and that this gives a negative answer to Question 3 of Cuth--Doucha--
Wojtaszczyk.

Identification: If `c_0` were linearly isomorphic to a subspace of
`F(ell_2)`, then the weak sequential completeness of `F(ell_2)` would pass to
that subspace. This is impossible because `c_0` is not weakly sequentially
complete. Thus the answer to the source `ell_2` question is negative.
