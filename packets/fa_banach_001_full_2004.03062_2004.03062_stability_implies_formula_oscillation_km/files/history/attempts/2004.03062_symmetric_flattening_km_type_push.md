## Target

This push tried to promote the normalized-spreading symmetry lemma into a full
positive answer to Hanson Question 4.3:

> If `T` is a stable Banach theory, is every unary formula oscillation stable?

The active route was:

1. stabilize a bad formula on a block sequence;
2. use stability of all normalized linear-combination pullbacks to make the
   spreading color symmetric;
3. prove that such symmetric stable spreading colors can be flattened on a
   further block subspace.

The push did not close step 3. It did, however, rule out the simplest proposed
flattening lemma and isolate a more plausible replacement route.

## New Obstruction: Symmetry Plus Uniform Continuity Is Not Enough

The previous note left open the tempting lemma:

> A uniformly continuous finite-permutation-invariant function on the unit
> sphere of `ell_2` is almost constant on vectors with sufficiently small
> `ell_infty` norm.

This lemma is false.

Let `u_n` be the positive flat vector of support size `n` in `ell_2`,

```text
u_n = n^{-1/2}(1,...,1,0,...).
```

Let

```text
A = union_k { finite coordinate permutations and sign changes of u_{2^k} }.
```

Define

```text
F(x) = dist(x,A),   x in S_{ell_2}.
```

Then `F` is 1-Lipschitz and invariant under finite coordinate permutations
(indeed also under signs). Also `||u_n||_infty -> 0`.

But

```text
F(u_{2^k}) = 0
```

while the flat vectors of support size `3*2^k` stay a fixed positive distance
from `A`. Indeed the closest flat power-of-two supports have sizes
`2^{k+1}` and `2^{k+2}`, so

```text
dist(u_{3*2^k}, u_{2^{k+1}})
  = sqrt(2 - 2 sqrt(2/3)),

dist(u_{3*2^k}, u_{2^{k+2}})
  = sqrt(2 - 2 sqrt(3/4)).
```

Thus `F(u_{3*2^k})` is bounded below by the smaller of these two constants,
even though `||u_{3*2^k}||_infty -> 0`.

Conclusion: the desired flattening theorem cannot be a pure consequence of
small coordinates, symmetry, and uniform continuity. The proof must use the
stability of the normalized pullbacks in a genuinely stronger way.

## Why This Does Not Yet Give a Counterexample

The function above only kills the naive lemma. It is not a counterexample to
Hanson Question 4.3.

The reason is that nothing in the construction makes the Banach-language
pullbacks

```text
(x_1,...,x_m) -> F(sigma_c(x_1,...,x_m))
```

stable. In fact the construction is deliberately sensitive to support-scale
arithmetic. That is exactly the kind of sensitivity that stable double-limit
arguments should try to forbid.

This distinction is important. The normalized-spreading symmetry lemma proved
earlier shows that stable pullbacks remove order sensitivity. The present
example shows that order symmetry alone still permits scale sensitivity.

## Replacement Route: Formula-Aware Krivine-Maurey Types

The strongest route found in this push is to stop trying to prove a standalone
topological flattening theorem and instead adapt the Krivine-Maurey type
machinery with the formula value included in the type space.

The classical input is:

- Krivine-Maurey: every stable Banach space contains almost isometric copies
  of `ell_p` for some `1 <= p < infinity`.
- Braga-Swift's exposition records the type-space/conic-class proof: stable
  types, dilation, convolution, minimal closed admissible conic classes, and
  a common continuity point for all finite scalar convolutions.
- Hanson/Shelah-Usvyatsov supply the model-theoretic analogue for wide types:
  minimal wide types have Hilbert Morley spines in saturated extensions.

The proposed strengthened theorem is:

> Formula-aware KM lemma. Let `T` be a stable Banach theory, `M models T`,
> `phi(x,a)` a unary formula, and `Y` an infinite-dimensional subspace of
> `M`. Then, for every `eps>0`, one can find a normalized basic sequence
> `(y_n)` in `Y` whose closed span `Z` satisfies
> `osc(phi, S_Z) < eps`.

A possible proof architecture:

1. Work inside `Y`, and build a type space whose points record not only norm
   types `sigma_b(x)=||x+b||`, but also the values of all formula pullbacks
   `phi(sigma_c(x_1,...,x_m),a)`.
2. Stability of `T` gives the same double-limit interchange needed to make
   convolution of these enriched types well behaved.
3. Run the Krivine-Maurey minimal conic-class/continuity-point argument in the
   enriched type space.
4. At the selected type `p`, scalar convolutions should satisfy an `ell_p`
   law. Because `phi` is part of the enriched type, the formula coordinate of
   every normalized scalar convolution should be forced to equal the formula
   coordinate of `p`.
5. Push the resulting enriched type back to an actual basic sequence in `Y`,
   as in the Krivine-Maurey passage from type to almost isometric `ell_p`
   copies. The same approximation should keep the `phi`-oscillation below
   `eps`.

This would promote the conditional and attempt material into a full positive
answer to Hanson Question 4.3.

## Where The Proof Still Needs Work

Two technical points remain unverified.

1. **Enriched type-space compactness and continuity.** The classical
   Krivine-Maurey type space is built for norm types. One needs to check that
   adding the formula coordinates preserves the compactness and Baire
   continuity-point argument, and that all scalar convolution maps remain
   continuous at the selected point.

2. **Constancy of the formula coordinate.** Even if the norm coordinate
   satisfies the `ell_p` law, it must be proved that the `phi` coordinate of
   every normalized convolution

   ```text
   alpha_1*p * ... * alpha_m*p,   sum |alpha_i|^p = 1,
   ```

   is independent of the scalar tuple. This is the real replacement for the
   false small-`ell_infty` flattening lemma.

The normalized-spreading symmetry lemma is evidence for item 2 in the special
case of block permutations, but it does not by itself imply invariance under
splitting a coefficient into many equal pieces or under arbitrary scalar
convolutions.
