## Motivation

The previous push isolated a stronger route than the midpoint-WAP lemma.
Hanson's Banach language contains normalized linear-combination maps
`sigma_c`. Therefore, if `phi(x,a)` is a unary formula in a stable Banach
theory, then every pullback

```text
theta_c(x_1,...,x_k,a) = phi(sigma_c(x_1,...,x_k),a)
```

is again stable.

This note proves the first structural consequence of that observation:
after passing to a stabilized spreading sequence, stable normalized pullbacks
force the limiting color to be invariant under finite block permutations.

## Spreading Setup

Let `M` be a Banach structure, let `phi(x,a)` be a unary formula instance, and
let `(e_n)` be a bounded sequence in `M`.

Say that `(e_n)` is `phi`-spreading for normalized linear combinations if, for
every `k` and every scalar vector `c=(c_1,...,c_k)`, the limit

```text
Lambda(c)
  = lim_{n_1<...<n_k, n_1 -> infinity}
      phi(sigma_c(e_{n_1},...,e_{n_k}),a)
```

exists. In applications one obtains such a sequence by the usual diagonal
Ramsey/spreading-model stabilization over a countable dense set of coefficient
vectors, then extends by uniform continuity of formulas.

## Lemma: Stability Forces Block-Symmetric Spreading Limits

**Lemma.** Suppose `T=Th(M)` is stable. If `(e_n)` is `phi`-spreading for
normalized linear combinations, then the limiting functional `Lambda` is
invariant under finite block permutations of the coefficient vector. In
particular, for any two finite scalar blocks `b` and `c`,

```text
Lambda(b,c) = Lambda(c,b).
```

Consequently `Lambda` is invariant under finite coordinate permutations.

### Proof

Fix two finite scalar blocks

```text
b=(b_1,...,b_r),   c=(c_1,...,c_s).
```

Consider the formula

```text
theta(u_1,...,u_r, v_1,...,v_s, a)
  = phi(sigma_(b,c)(u_1,...,u_r,v_1,...,v_s),a).
```

Since `sigma_(b,c)` is a Banach-language function symbol and `T` is stable,
`theta` is stable. Equivalently, partitioning the variables as
`u=(u_1,...,u_r)` and `v=(v_1,...,v_s)`, the double-limit property holds for
the two-variable formula `theta(u,v,a)`.

Choose consecutive disjoint blocks of the sequence `(e_n)`:

```text
U_n = (e_{p_n+1},...,e_{p_n+r}),
V_m = (e_{q_m+1},...,e_{q_m+s}),
```

with `p_n+r < q_m+1` whenever `n` is fixed and `m` is sufficiently large.
Then for each fixed `n`, as `m -> infinity`, the block `U_n` occurs before
`V_m`, so the spreading limit gives

```text
lim_m theta(U_n,V_m,a) = Lambda(b,c).
```

Taking `n -> infinity` leaves the same value. Thus

```text
lim_n lim_m theta(U_n,V_m,a) = Lambda(b,c).
```

On the other hand, for each fixed `m`, as `n -> infinity`, the block `V_m`
occurs before `U_n` in the original sequence. The same vector is now seen by
the spreading limit with the two coefficient blocks reversed:

```text
lim_n theta(U_n,V_m,a) = Lambda(c,b).
```

Therefore

```text
lim_m lim_n theta(U_n,V_m,a) = Lambda(c,b).
```

Stability of `theta` makes the two iterated limits equal, hence
`Lambda(b,c)=Lambda(c,b)`. Adjacent block swaps generate all finite block
permutations, and coordinate permutations are the special case of one-element
blocks.

## Consequence For Hanson Q4.3

Any counterexample to Hanson Question 4.3 must survive this symmetry
constraint. More concretely, if `phi(x,a)` is a stable unary formula and a
subspace fails oscillation stability, then every sufficiently stabilized
spreading limit extracted from that subspace is forced to be a symmetric
coloring of the associated spreading model.

This rules out the easiest way a Maurey/Odell-Schlumprecht coloring could
appear: it cannot produce an asymmetric order-sensitive spreading limit. The
remaining obstruction is subtler:

> Are all uniformly continuous symmetric spreading colorings that arise from
> stable Banach formula pullbacks block-flattenable?

For Hilbert spreading models, the fourth-power test

```text
Lambda(x)=sum_i |x_i|^4
```

is symmetric and nonconstant, but it is block-flattenable by passing to vectors
with small `ell_infty` coordinates. Thus the desired conclusion is not
"symmetry implies constancy"; it must be a genuine block-flattening theorem for
symmetric spreading colorings.

## Remaining Gap

The lemma does not yet prove oscillation stability. Two additional ingredients
are still missing.

1. **Uniform spreading-to-subspace transfer.** One needs a form of
   stabilization strong enough to pass from finite spreading limits back to an
   actual infinite-dimensional subspace on which the original formula is
   controlled.
2. **Symmetric spreading flattening.** One needs to prove, or find the exact
   counterexample to, the claim that the symmetric spreading colorings allowed
   by stable normalized pullbacks admit an almost-monochromatic block subspace.

The first item is where Gowers/Rosendal block games may be needed. The second
item is the real analytic heart of the current route. It may be easier in
specific spreading models such as `ell_p`, and Krivine--Maurey theory suggests
that stable Banach spaces force `ell_p`-type spreading behavior in every
subspace.

## Current Status

This push did not crack Hanson Question 4.3. It did, however, replace the vague
"derive an order pattern from non-oscillation" target with a sharper
structural statement:

```text
stable normalized pullbacks => symmetric spreading limits.
```

Future work should try to prove the two missing bullets above, first for
Hilbert and `ell_p` spreading models and then in the full stable Banach
