## Goal Of This Push

The previous push showed that the naive Hilbert flattening lemma is false:
there are Lipschitz symmetric sphere colorings, such as distance to the
sign/permutation orbit of flat vectors with support sizes `2^k`, that do not
flatten merely because `||x||_infty` is small.

This push therefore tried to use stability more directly. The target was to
turn Hanson's finite-dimensional Dvoretzky-Milman/indiscernible-subspace
compactness into an actual in-model subspace by adding a subspace-relative
finite-satisfiability condition.

## Why The Obvious Compactness Shortcut Fails

For any uniformly continuous sphere coloring `F` and every finite `k`,
Dvoretzky-Milman gives a `k`-dimensional almost-monochromatic subspace. Taking
an ultraproduct gives an infinite-dimensional subspace in an elementary
extension on which `F` is almost constant.

This is not enough. If finite representability of that ultrapower subspace
could be pulled back directly to the original model, every uniformly
continuous Hilbert-sphere coloring would be oscillation stable, contradicting
Maurey/Odell-Schlumprecht phenomena recalled by Hanson.

The missing ingredient is coherence. Finite-dimensional good subspaces can be
chosen independently for each `k`; they need not form a nested or asymptotic
branch inside the original subspace. Stability must be used exactly to produce
that coherent branch.

There is a slightly more sophisticated version of the same false shortcut.
One can put all of `Y` into the parameter set and, by compactness over triples
`(finite Delta over Y, finite-codimensional tail Y_0, dimension k)`, obtain in
an ultrapower an infinite-dimensional subspace whose unit sphere has a single
complete type over `Y` and on which `phi` is almost constant. This still does
not give an in-model subspace. The joint finite tuple types are satisfiable in
tails of `Y`, but their realizations need not be nested: the first vector in a
good `k`-tuple may have to be replaced when one passes to a good
`(k+1)`-tuple. This is precisely the infinitary Dvoretzky-Milman obstruction
in type language.

## A More Exact Candidate Lemma

Fix a stable Banach theory `T`, a model `M`, a unary formula instance
`phi(x,a)`, an infinite-dimensional closed subspace `Y <= M`, and `eps>0`.

The route would be completed by the following lemma.

> **Tail-minimal wide type lemma.** There is a real `r` and a wide partial type
> `p(x)` over parameters including `a`, containing
>
> ```text
> ||x|| = 1,    |phi(x,a)-r| <= eps,
> ```
>
> such that `p` is finitely satisfiable in every finite-codimensional tail of
> `Y`, and such that its wide nonforking extensions over finite tuples from
> `Y` preserve this tail finite satisfiability.

If this lemma holds, the proof of Hanson Q4.3 is essentially formal:

1. Choose a tail-finitely-satisfiable minimal wide extension of the
   Dvoretzky-Milman partial type.
2. By Shelah-Usvyatsov/Hanson, a Morley sequence in the corresponding minimal
   wide type is the orthonormal basis of an indiscernible subspace, and the
   entire unit sphere of its span realizes the same wide type.
3. Tail finite satisfiability lets one approximate that Morley sequence by a
   normalized block/basic sequence inside `Y`.
4. Since all normalized finite linear combinations in the Morley span satisfy
   `|phi-r| <= eps`, a standard diagonal gliding-hump approximation gives an
   actual infinite-dimensional subspace `Z <= Y` on whose unit sphere `phi`
   has oscillation `O(eps)`.

This is the cleanest current formulation of the missing "full lemma".

## What Was Verified

The finite-dimensional source of the partial type is solid. For every
finite-codimensional `Y_0 <= Y`, every finite set of formulas `Delta`, and
every dimension `k`, Hanson's use of the Milman theorem gives a
`k`-dimensional subspace of `Y_0` on which the formulas in `Delta` are
approximately determined by Hilbert congruence. In particular, for the unary
formula `phi(x,a)`, all unit vectors in that finite-dimensional subspace have
nearly the same `phi` value.

Using a compactness/ultrafilter over triples `(Y_0, Delta, k)` selects a
cluster color `r` and yields a partial type that is wide and tail-finitely
satisfiable in the above sense.

This part uses only Dvoretzky-Milman and Hanson's finite-dimensional
indiscernibility construction; it does not use stability.

## The Remaining Gap

The unproved point is preservation of tail finite satisfiability under the
minimal wide extension process.

Shelah-Usvyatsov prove that in a stable Banach theory every wide type has a
minimal wide extension, and Hanson proves that Morley sequences in minimal wide
types are exactly orthonormal bases of indiscernible subspaces. However, the
known statement does not explicitly say that if the starting wide type is
finitely satisfiable in every finite-codimensional tail of a fixed in-model
subspace `Y`, then the selected minimal wide extension can be chosen to retain
that property over the finite tuples needed for the Morley construction.

This is exactly the in-model pullback issue. If it is true, the full positive
answer follows. If it is false, a counterexample probably has to exploit this
failure: every finite tail contains arbitrarily good finite-dimensional
monochromatic pieces, but no tail-finitely-satisfiable minimal wide branch can
be made coherent in `Y`.

## Relation To The Formula-Aware KM Route

The formula-aware Krivine-Maurey route and this tail-minimal-wide-type route
now look like two languages for the same obstruction.

- In KM language, one wants a minimal conic continuity type whose normalized
  scalar convolutions keep the `phi` coordinate fixed.
- In stable model-theoretic language, one wants a minimal wide type whose
  Morley/Hilbert span remains approximable inside the original subspace `Y`.

The second formulation seems sharper because Hanson's theorem already supplies
the "unit sphere of the Morley span realizes the type" conclusion once the
right minimal wide type is available.

## Block/Ramsey Route Checked

The Garcia-Ferreira--Hernandez-Soto block asymptotic model paper
arXiv:2009.03247 was rechecked. It gives a useful language for block
asymptotic models indexed by barriers and makes clear why finite-dimensional
Ramsey stabilization alone is insufficient: arbitrary normalized basic
sequences have many possible block asymptotic models, and these models need
not collapse to a single spreading behavior.

For the present problem, stability should force equality or compatibility of
the formula coordinates of those barrier/block asymptotic models. I did not
manage to turn this into a proof. It remains another version of the same
coherence problem.

## Current Verdict

No full proof was obtained in this push.

The progress is a substantially sharper final target:

```text
prove tail finite satisfiability survives the stable minimal-wide extension
needed to build the Morley/Hilbert spine.
```

That lemma would convert the existing finite-dimensional Hanson machinery into
an actual oscillation-stable subspace of the original `Y`, and would therefore
settle Question 4.3 positively.
