# Four-direction square counterexample to first-order magnitude sharpness

Status: counterexample_likely_valid

Source: Mark W. Meckes, *Magnitude and Holmes--Thompson intrinsic
volumes of convex bodies*, arXiv:2206.02600; Canadian Mathematical Bulletin
66 (2023), 854--867. The target is Conjecture 1.7 on page 4 (source label
C:mu1-conj).

## Result

The packet disproves Conjecture 1.7. Let

    u1 = (1,0),  u2 = (0,1),
    u3 = (1,1)/sqrt(2),  u4 = (1,-1)/sqrt(2),
    ||x||_E = sum_j |<x,u_j>|,
    K = [-1,1]^2.

This is a two-dimensional hypermetric norm generated by an even measure on
the four direction pairs. The conjecture predicts

    lim_{t -> 0+} (Mag(tK)-1)/t = mu_1^E(K)/4 = 2 + 2 sqrt(2).

An explicit squared-Hilbert embedding of this metric puts the image of K on a
sphere. The quadratic relation

    u3 tensor u3 + u4 tensor u4 = u1 tensor u1 + u2 tensor u2

allows the center to be shifted and the squared radius to be reduced by

    3 / (4(1+2 sqrt(2))).

Expanding the resulting Gaussian kernel into nonnegative Hilbert tensor
squares gives the rigorous bound

    limsup_{t -> 0+} (Mag(tK)-1)/t
     <= 2 + 2 sqrt(2) - 3/(2(1+2 sqrt(2)))
     <  mu_1^E(K)/4.

Thus the conjectured equality is impossible, regardless of whether the
one-sided limit exists.

Recommended verdict: likely valid counterexample. The most important checks
are the Gaussian tensor-series bound for signed measures, the radius shift
using the rank-one tensor relation, and the factor c_1=2 in Proposition 2.3
