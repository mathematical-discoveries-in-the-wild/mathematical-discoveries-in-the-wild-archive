\section*{Status}

This packet gives a likely valid counterexample to the broad polynomial-fiber
covering question raised in Ortega Castillo--Prieto~\cite{OP}.  It does not
settle the polynomial cluster value theorem itself.  It shows that, for the
standard choice \(X=\ell_2\) and \(H(\Ball)=\Hinfty(\Ball_X)\), the polynomial
fibers over points of \(\overline{\Ball}_{X^{**}}\) need not cover the maximal
ideal space of \(H(\Ball)\).

\section*{Source Problem Evidence}

The source passage is in Section 3 of~\cite{OP}.  In the notation of the
source paper, it asks whether
\[
  \bigcup_{x_0^{**}\in \overline{\Ball}^{**}}
  \M_{x_0^{**}}^{\PP}(H(\Ball))
  =
  \M_{H(\Ball)} .
\]

\section*{Definitions and Scope}

Let \(X\) be a complex Banach space and \(\Ball=\Ball_X\).  The algebra
\(\Au(\Ball)\) is the uniform algebra of bounded holomorphic functions on
\(\Ball\) that are uniformly continuous on \(\Ball\).  For a uniform algebra
\(H(\Ball)\) with
\[
  \Au(\Ball)\subset H(\Ball)\subset \Hinfty(\Ball),
\]
the source paper defines the polynomial fiber over
\(x^{**}\in \overline{\Ball}^{**}\) by
\[
  \M_{x^{**}}^{\PP}(H(\Ball))
  =
  \{ \tau\in \M_{H(\Ball)}:\tau|_{\Au(\Ball)}=\delta_{x^{**}}\},
\]
where \(\delta_{x^{**}}\) denotes evaluation at \(x^{**}\) through
Aron--Berner extension on \(\Au(\Ball)\).

Thus an element of \(\M_{H(\Ball)}\) lies in the union of the polynomial fibers
only if its restriction to \(\Au(\Ball)\) is one of these evaluation
characters.

\section*{Proof Intuition}

The obstruction is already visible on the Hilbert ball.  Take evaluations at
the points \(r e_n\) in \(\Ball_{\ell_2}\), with \(0<r<1\).  Any cluster point
of these evaluations on \(\Hinfty(\Ball_{\ell_2})\) is a character.  On all
linear functionals it looks like evaluation at \(0\), because \(e_n\) is weakly
null.  But the continuous quadratic polynomial
\[
  P(x)=\sum_{n=1}^{\infty} x_n^2
\]
has value \(r^2\) at every \(r e_n\), while \(P(0)=0\).  Therefore the
restriction of the cluster character to \(\Au(\Ball_{\ell_2})\) cannot be any
evaluation character.  Consequently the cluster character cannot belong to any
polynomial fiber.

\begin{theorem}
Let \(X=\ell_2\), let \(\Ball=\Ball_X\), and let
\(H(\Ball)=\Hinfty(\Ball)\).  Then
\[
  \bigcup_{x\in \overline{\Ball}_{\ell_2}}
  \M_x^{\PP}(\Hinfty(\Ball))
  \subsetneq
  \M_{\Hinfty(\Ball)} .
\]
In particular, the polynomial fibers over points of
\(\overline{\Ball}_{X^{**}}\) do not necessarily cover the maximal ideal space
of \(H(\Ball)\).
\end{theorem}

\begin{proof}
Fix \(0<r<1\), and let \(e_n\) be the standard unit vector basis of
\(\ell_2\).  For every \(n\), point evaluation
\(\delta_{r e_n}\) is an element of \(\M_{\Hinfty(\Ball)}\).  Since
\(\M_{\Hinfty(\Ball)}\) is compact in the weak-star topology, the net (or,
equivalently, a subnet of the sequence) \((\delta_{r e_n})\) has a cluster
point; call it \(\tau\in \M_{\Hinfty(\Ball)}\).

Let \(\varphi=\tau|_{\Au(\Ball)}\).  We claim that \(\varphi\) is not an
evaluation character on \(\Au(\Ball)\).  First, for every continuous linear
functional \(x^*\in \ell_2^*\), writing \(x^*(e_n)=a_n\) with
\((a_n)\in \ell_2\), we have \(a_n\to 0\).  Hence
\[
  \varphi(x^*)
  =
  \lim \delta_{r e_n}(x^*)
  =
  \lim r x^*(e_n)
  =
  0.
\]
Therefore, if \(\varphi=\delta_x\) for some \(x\in \overline{\Ball}_{\ell_2}\),
then every linear functional vanishes at \(x\), so \(x=0\).

Now define
\[
  P(x)=\sum_{j=1}^{\infty} x_j^2,\qquad x=(x_j)\in \ell_2 .
\]
This series is absolutely convergent and
\[
  |P(x)|\leq \sum_j |x_j|^2=\|x\|_2^2.
\]
Moreover,
\[
  |P(x)-P(y)|
  =
  \left|\sum_j (x_j-y_j)(x_j+y_j)\right|
  \leq
  \|x-y\|_2\,\|x+y\|_2 ,
\]
so \(P\) is a continuous 2-homogeneous polynomial and belongs to
\(\Au(\Ball)\).  Along the chosen evaluations,
\[
  P(r e_n)=r^2 \quad\text{for all } n.
\]
Thus
\[
  \varphi(P)=\tau(P)=r^2,
\]
while \(\delta_0(P)=P(0)=0\).  This contradicts the only possible evaluation
case identified above.  Hence \(\varphi\) is not equal to \(\delta_x\) for any
\(x\in \overline{\Ball}_{\ell_2}\).

Because \(\ell_2\) is reflexive, \(\overline{\Ball}_{X^{**}}\) is identified
with \(\overline{\Ball}_{\ell_2}\).  If \(\tau\) belonged to a polynomial fiber
\(\M_x^{\PP}(\Hinfty(\Ball))\), then by definition its restriction to
\(\Au(\Ball)\) would be \(\delta_x\).  But \(\tau|_{\Au(\Ball)}=\varphi\) is
not any evaluation character.  Therefore
\[
  \tau\notin
  \bigcup_{x\in \overline{\Ball}_{\ell_2}}
  \M_x^{\PP}(\Hinfty(\Ball)).
\]
This proves the strict inclusion.
\end{proof}

\section*{Verification Notes}

The proof uses only:
\begin{itemize}
  \item compactness of the maximal ideal space of a unital commutative Banach
  algebra;
  \item weak nullity of the standard basis of \(\ell_2\);
  \item the explicit continuous polynomial \(P(x)=\sum_j x_j^2\);
  \item the definition of polynomial fiber as a restriction fiber over
  evaluation on \(\Au(\Ball)\).
\end{itemize}
No computational verification is needed.

\section*{Limitations}

This is not a counterexample to the polynomial cluster value theorem
\[
  Cl_{\Ball}^{\PP}(f,x^{**})=\widehat f(\M_{x^{**}}^{\PP})
\]
for a fixed fiber.  It only disproves the broader structural hope that the
polynomial fibers cover all of \(\M_{H(\Ball)}\).  It also does not resolve the
more specialized \(C(K)\) questions pursued in Section 3 of~\cite{OP}.

\section*{Novelty and Literature Check}

Before promotion, I searched the local run indexes and parsed arXiv sources
for the arXiv id \(1702.06471\), the title phrase ``polynomial cluster value
problem'', and the phrases ``union of the polynomial fibers'', ``polynomial
fibers'', and ``range of \(\pi^{\PP}\)''.  I also searched the web for exact
phrases around the covering question.  I found the source paper and later
related literature on large fibers of \(\M(\Au(B_{\ell_p}))\), notably
Dimant--Lassalle--Maestre~\cite{DLM}, but I did not find an exact statement of
the \(H^\infty(B_{\ell_2})\) polynomial-fiber covering counterexample above.

The construction is closely related in spirit to the known phenomenon that
maximal ideal spaces of \(\Au(B_X)\) can contain many non-evaluation
characters.  For that reason, the novelty confidence should be treated as
moderate: the packet gives a direct, self-contained counterexample to the
specific covering question, but it uses standard compactness and basis-vector
ideas rather than a new technical method.

\section*{Human Review Recommendation}

Review the definition of \(\M_x^{\PP}(H(\Ball))\) against the source paper and
check that the restriction of the constructed \(\tau\) to \(\Au(\Ball)\) is
indeed not an evaluation character.  The key audit line is the simultaneous
use of all linear functionals to force a hypothetical evaluation point to be
0, followed by the quadratic polynomial \(P\) separating the character from
\(\delta_0\).
