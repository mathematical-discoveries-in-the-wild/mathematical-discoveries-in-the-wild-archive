# Property (A) plus Property (C) does not imply semi-greediness

Status: `literature_already_answered` (negative answer to Question 3 only).

The source paper, arXiv:1803.05052, asks on PDF page 19 whether a Schauder
basis with Properties (A) and (C) must be semi-greedy.  Pablo M. Berná's
arXiv:1804.05730 answers this negatively on PDF page 7.

The supporting paper proves in Theorem 1.10 that every semi-greedy Schauder
basis is quasi-greedy.  Its final remark then invokes the basis from
Berná--Blasco--Garrigós, Subsection 5.5: that basis has Property (A) and the
inequality defining Property (C), but is not quasi-greedy.  Consequently it
cannot be semi-greedy.
