# Literature Status: Question 3 of arXiv:1509.01499

Status: `literature_already_answered`

Source/open-problem paper: Julio Flores, Francisco L. Hernandez, and Pedro Tradacete, "Disjointly homogeneous Banach lattices and applications", arXiv:1509.01499.

Target: Final-section Question 3 asks:

> Is there a reflexive `p`-DH rearrangement invariant space on `[0,1]` whose dual is not DH?

Supporting answer: Sergey V. Astashkin, "Duality problem for disjointly homogeneous rearrangement invariant spaces", arXiv:1805.00691.

Astashkin explicitly identifies the problem as Question 3 from the survey and proves in Theorem 2 that for every `1<p<infinity` there exists a `p`-DH reflexive r.i. space `X_p` on `[0,1]` such that `X_p^*` is not DH. This gives a positive answer to the source question.
