# Finite-dimensional Hilbert-valued `C_0(L,H)` case answered by later theorem

Status: `literature_implied_answer`

Original source: Jarno Talponen, *Convex-transitivity and function spaces*,
arXiv:0711.3768v2.

Supporting source: Fernando Rambla-Barreno and Jarno Talponen, *Uniformly
convex-transitive function spaces*, arXiv:0902.0640v2.

## Identification

In the original paper, Section 2, subsection `Convex-transitivity of
C_0(L,X)`, the author proves that if `H` is an infinite-dimensional Hilbert
space and scalar `C_0(L)` is convex-transitive, then `C_0(L,H)` is
convex-transitive.  Immediately after the theorem the author writes that the
argument relies strongly on `dim(H)=infty` and that it is not known whether
this assumption can be removed.

The 2009 supporting paper proves a stronger theorem in Section 4, Theorem 4.3:
if real scalar `C_0^R(L)` is convex-transitive and `X` is a convex-transitive
Banach space over `K`, then `C_0^K(L,X)` is convex-transitive.

## Consequence

Every finite-dimensional Hilbert space is convex-transitive, and the
one-dimensional case is just the scalar case.  Therefore Theorem 4.3 of
arXiv:0902.0640 removes the infinite-dimensional Hilbert restriction from the
source theorem.  The implication is agent-identified; the supporting theorem is
broader and does not appear to single out the exact 2007 sentence.

## Scope

This packet records a literature-implied answer only.  It does not claim a new
proof.  It answers the specific source question about whether the
`dim(H)=infty` assumption can be removed from the Hilbert-valued `C_0(L,H)`
result.
