# Literature packet: Schlumprecht Q1 answered by Argyros-Haydon

Status: `literature_already_answered`

Source/open-problem paper: Th. Schlumprecht, "How many operators do there exist on a Banach space?", arXiv:math/0208055.

Source question: item (Q1) in the introduction asks whether every infinite-dimensional Banach space admits a bounded operator not of the form `lambda Id + C` with `C` compact.

Supporting paper: S. A. Argyros and R. G. Haydon, "A hereditarily indecomposable L_infty-space that solves the scalar-plus-compact problem", arXiv:0903.3921.

Answer: Negative. Argyros-Haydon construct a Banach space `X_K` such that every bounded operator on `X_K` is `lambda I + K` with `K` compact. Their title, abstract, introduction, and final theorem of the `Operators` section explicitly identify this as solving the scalar-plus-compact problem.

Scope limitation: This records the literature answer to Schlumprecht's Q1 only. It does not answer Schlumprecht's Q2/Q3 subspace/operator problem, Q4 Class 1/Class 2 dichotomy, Q5 partial-unconditionality question, or the small remark asking for an elementary proof that Class 1 spaces contain unconditional basic sequences.
