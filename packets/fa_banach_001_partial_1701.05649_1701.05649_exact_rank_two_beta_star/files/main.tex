\section*{Status}

\textbf{Partial result, likely valid; human verification recommended.}
The original question asks for every nonzero countable ordinal.  This packet
proves the exact value $2$ for every allowed Baire level $\xi\geq2$; it does
not settle arbitrary successor ordinals.

\section*{Source Question}

Leung, Ng, and Tang ask the following after Proposition 3.6 on page 11 of
their paper \cite{LNT}.

\begin{center}
  \includegraphics[width=0.98\linewidth]{figures/open_problem_crop.png}
\end{center}

\begin{quote}
Let $(X,\tau)$ be an uncountable Polish space and let $\xi$ be a countable
ordinal $\geq2$.  Is it true that for any nonzero countable ordinal $\zeta$,
there exists $f\in\B_\xi(\tau)$ such that $\beta^*_\xi(f)=\zeta$?
\end{quote}

The source proves that every nonzero countable limit ordinal occurs and that,
for every nonzero $\zeta$, some characteristic function has rank strictly
between $\zeta$ and $\zeta+2$ \cite[Propositions 3.4 and 3.6]{LNT}.

\section*{Definitions}

For $f\colon X\to\mathbb R$, a closed set $F$, and $\varepsilon>0$, let
$D_\sigma^1(f,\varepsilon,F)$ be the set of points $x\in F$ such that every
$\sigma$-neighborhood of $x$ contains $x_1,x_2\in F$ with
$|f(x_1)-f(x_2)|\geq\varepsilon$.  Iterate this derivation transfinitely.
The oscillation rank $\beta_\sigma(f)$ is the supremum over $\varepsilon>0$
of the first stage at which the iterated derivative is empty.

If $f\in\B_\xi(X,\tau)$, the extended rank is
\[
  \beta^*_\xi(f)=
  \min\bigl\{\beta_\sigma(f):
  \tau\subseteq\sigma\subseteq\borelSigma^0_\xi(\tau),\
  \sigma\text{ Polish, and }f\in\B_1(X,\sigma)\bigr\}.
\]

\section*{Proof Intuition}

Rank one is exactly potential continuity under an admissible refinement.
For a characteristic function, potential continuity would make its support
clopen.  Since every open set of an admissible topology is already
$\borelSigma^0_\xi$ in the original topology, rank one is possible only for
supports in $\borelDelta^0_\xi$.

Strictness of the Borel hierarchy supplies a support
$A\in\borelPi^0_\xi\setminus\borelSigma^0_\xi$.  It cannot become clopen, so
rank one is excluded.  On the other hand, its complement is
$\borelSigma^0_\xi$, and the topology-refinement lemma from the source paper
can make that complement open, hence make $A$ closed.  The indicator of a
closed set has rank at most two: after one oscillation derivative only points
of $A$ remain, and the function is constant there.

\begin{theorem}[Exact value two]
Let $(X,\tau)$ be an uncountable Polish space and let
$2\leq\xi<\omega_1$.  There exists $f\in\B_\xi(X,\tau)$ such that
\[
  \beta^*_\xi(f)=2.
\]
The function $f$ may be chosen to be characteristic.
\end{theorem}

\begin{proof}
By strictness of the Borel hierarchy on uncountable Polish spaces
\cite[Section 22]{Kechris}, choose
\[
  A\in\borelPi^0_\xi(\tau)\setminus\borelSigma^0_\xi(\tau).
\]
In particular $A\in\borelDelta^0_{\xi+1}(\tau)$, so the
Lebesgue--Hausdorff characterization gives
$f=\chi_A\in\B_\xi(X,\tau)$.

Set $G=X\setminus A\in\borelSigma^0_\xi(\tau)$.  Apply the topology-refinement
lemma \cite[Lemma 3.1]{LNT} to $G$ and the original topology.  It gives a
Polish topology $\sigma$ such that
\[
  \tau\subseteq\sigma\subseteq\borelSigma^0_\xi(\tau)
  \quad\text{and}\quad G\in\sigma.
\]
Thus $A$ is $\sigma$-closed.  The indicator of a closed set is Baire class
one in a metrizable topology, so $\sigma$ is admissible for the minimum
defining $\beta^*_\xi(f)$.

We claim that $\beta_\sigma(f)\leq2$.  If $\varepsilon>1$, the first
derivative is empty.  If $0<\varepsilon\leq1$, every point of the open set
$G$ has a $\sigma$-neighborhood on which $f$ is constantly zero.  Hence
\[
  D_\sigma^1(f,\varepsilon,X)\subseteq A.
\]
On this derived closed set $f$ is constantly one, so
\[
  D_\sigma^2(f,\varepsilon,X)=\varnothing.
\]
Therefore $\beta^*_\xi(f)\leq\beta_\sigma(f)\leq2$.

It remains to exclude rank one.  Suppose $\beta^*_\xi(f)=1$.  Then an
admissible Polish topology $\rho$ satisfies $\beta_\rho(f)=1$.  Vanishing of
every first oscillation derivative is equivalent to continuity of $f$.
Consequently
\[
  A=f^{-1}((1/2,\infty))
  \quad\text{and}\quad
  X\setminus A=f^{-1}(({-}\infty,1/2))
\]
are both $\rho$-open.  Since
$\rho\subseteq\borelSigma^0_\xi(\tau)$, both sets belong to
$\borelSigma^0_\xi(\tau)$, and therefore
$A\in\borelDelta^0_\xi(\tau)$.  This contradicts
$A\notin\borelSigma^0_\xi(\tau)$.  Thus
$\beta^*_\xi(f)\geq2$, completing the proof.
\end{proof}

\section*{Focused Full-Upgrade Pushes}

Three routes toward all successor ordinals were tested.

\begin{enumerate}
\item A one-point accumulation gadget can add one ordinary oscillation
derivative, but it is not refinement-resistant: a singleton is
$\borelDelta^0_2$ and can be made clopen by an admissible refinement when
$\xi\geq2$.
\item The Hausdorff difference hierarchy over $\borelPi^0_\xi$ gives a
natural lower bound, but the available comparison is
$\alpha_\xi(f)\leq2\alpha^*_\xi(f)$.  Without a construction attaining the
sharp factor-two calibration and a matching upper topology, it does not
force a prescribed successor value of $\beta^*_\xi$.
\item Nested $\borelPi^0_\xi$ supports and multilevel characteristic sums
give short upper derivative chains after refinement.  No lower bound was
found that prevents a different admissible refinement from collapsing those
chains.  Thus this route was not promoted beyond the exact-rank-two case.
\end{enumerate}

\section*{Scope and Novelty Check}

Combining the theorem with the constant-function case and
\cite[Proposition 3.6]{LNT}, the known exact range contains $1$, $2$, and
every nonzero countable limit ordinal.  Arbitrary successors, beginning with
$3$, remain unresolved by this packet.
