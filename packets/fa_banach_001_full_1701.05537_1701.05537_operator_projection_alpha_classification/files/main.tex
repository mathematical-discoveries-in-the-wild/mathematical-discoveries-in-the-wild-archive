\section*{Source Problems}

The source paper is N. Monod, \emph{Fixed points in convex cones},
arXiv:1701.05537.  Near the end of the paper, after discussing
$\ell^p$-variants of the $f$-ratio and $f$-Reiter properties, the author
introduces the following Hilbert-space analogues.  A unitary representation
$\pi$ of a group $G$ on a Hilbert space $V$ has the operator $\alpha$-ratio
property, for a continuous operator $\alpha\neq 0$, if for every finite
$S\subseteq G$ and every $\eps>0$ there is $v\in V$ such that
\[
  \|\alpha\pi(s)v\| < (1+\eps)\|\alpha v\|\qquad (s\in S).
\]
It has the stronger operator $\alpha$-Reiter property if one can choose $v$
such that
\[
  \|\alpha(\pi(s)v-v)\| < \eps\|\alpha v\|\qquad (s\in S).
\]
Problems 47 and 48 ask to clarify these properties, especially for nonzero
projections $\alpha$.

\section*{Proof Intuition}

The ratio property is much weaker than it first appears.  It only compares the
$\alpha$-size of finitely many unitary translates with the original
$\alpha$-size.  A vector almost attaining the operator norm of $\alpha$
already works, so the ratio property is automatic for every bounded nonzero
$\alpha$.

The projection-Reiter side has an equally sharp obstruction.  If the
representation is nontrivial, then some unitary operator $U=\pi(g)$ is not the
identity.  The spectral theorem gives a nonzero spectral subspace on which
$U$ stays a definite distance away from $1$.  Projecting onto that spectral
subspace turns the Reiter inequality into a uniform lower bound, so Reiter
fails for that projection.  Conversely, a trivial representation has zero
movement and hence satisfies every Reiter condition.

\begin{theorem}
\label{thm:main}
Let $\pi:G\to \mathcal U(V)$ be a unitary representation on a Hilbert space.

\begin{enumerate}
\item For every nonzero $\alpha\in B(V)$, the representation has the operator
$\alpha$-ratio property.

\item For every nonzero $\alpha\in B(V)$, the representation has the operator
$\alpha$-Reiter property if and only if, for every finite $S\subseteq G$,
\[
  \inf_{\alpha v\neq 0}
  \frac{\sum_{s\in S}\|\alpha(\pi(s)v-v)\|^2}{\|\alpha v\|^2}=0.
\]
Equivalently, for no finite $S\subseteq G$ does there exist $c>0$ such that
\[
  \sum_{s\in S}\|\alpha(\pi(s)v-v)\|^2
  \geq c\|\alpha v\|^2
  \qquad(v\in V).
\]

\item The representation has the operator $P$-Reiter property for every
nonzero orthogonal projection $P$ on $V$ if and only if $\pi$ is trivial.
\end{enumerate}
\end{theorem}

\begin{proof}
For the ratio property, fix finite $S\subseteq G$ and $\eps>0$.  Since
$\alpha\neq0$, choose a unit vector $v$ with
\[
  \|\alpha v\|>\frac{\|\alpha\|}{1+\eps}.
\]
For each $s\in S$, unitarity gives
\[
  \|\alpha\pi(s)v\|\leq \|\alpha\|<(1+\eps)\|\alpha v\|.
\]
This proves (1).

For (2), the displayed infimum is zero exactly when, for every $\delta>0$,
there exists $v$ with $\alpha v\neq0$ and
\[
  \sum_{s\in S}\|\alpha(\pi(s)v-v)\|^2
  <\delta^2\|\alpha v\|^2.
\]
Taking $\delta=\eps$ gives the defining Reiter inequalities for all
$s\in S$.  Conversely, the defining inequalities imply the displayed sum
estimate with $\delta^2=|S|\eps^2$.  The second formulation is simply the
negation of the existence of a positive lower bound for this Rayleigh
quotient.

For (3), first suppose that $\pi$ is trivial.  Then
$\alpha(\pi(s)v-v)=0$ for every operator $\alpha$, every $s$, and every $v$,
so every projection-Reiter condition holds.

Conversely, suppose $\pi$ is not trivial.  Choose $g\in G$ such that
$U=\pi(g)\neq I$.  By the spectral theorem for the unitary $U$, there is
$\delta>0$ such that the spectral projection
\[
  P=1_{\{z\in\mathbb T:\ |z-1|\geq \delta\}}(U)
\]
is nonzero.  This projection commutes with $U$, and on $PV$ the operator
$U-I$ is bounded below by $\delta$:
\[
  \|(U-I)w\|\geq \delta\|w\|\qquad(w\in PV).
\]
Therefore, for every $v\in V$,
\[
  \|P(\pi(g)v-v)\|
  =\|(U-I)Pv\|
  \geq \delta\|Pv\|.
\]
Taking the finite set $S=\{g\}$ and any $0<\eps<\delta$ shows that the
operator $P$-Reiter property fails.  Hence a representation satisfying
$P$-Reiter for every nonzero orthogonal projection must be trivial.
\end{proof}

\begin{corollary}[Problem 48]
For unitary representations on Hilbert space:
\begin{enumerate}
\item every representation satisfies the operator $P$-ratio property for all
nonzero projections $P$;
\item the representations satisfying the operator $P$-Reiter property for all
nonzero projections $P$ are exactly the trivial representations.
\end{enumerate}
The same conclusion holds if ``projection'' is read as arbitrary bounded
idempotent rather than orthogonal projection.
\end{corollary}

\begin{proof}
The ratio statement is Theorem~\ref{thm:main}(1).  For Reiter, orthogonal
projections are among arbitrary bounded idempotents, so the spectral-projection
obstruction applies under either convention.  A trivial representation has
zero displacement and therefore satisfies the Reiter inequalities for every
operator, not only projections.
\end{proof}

\begin{corollary}[Bounded-below operators]
If $\alpha$ is bounded below, then the operator $\alpha$-Reiter property is
equivalent to the ordinary existence of almost invariant vectors for $\pi$.
\end{corollary}

\begin{proof}
Choose constants $0<m\leq M<\infty$ with
\[
  m\|w\|\leq \|\alpha w\|\leq M\|w\|\qquad(w\in V).
\]
If $\pi$ has almost invariant vectors, choose $v\neq0$ with
\[
  \|\pi(s)v-v\|<\eps mM^{-1}\|v\|\qquad(s\in S).
\]
Then
\[
  \|\alpha(\pi(s)v-v)\|\leq M\|\pi(s)v-v\|
  <\eps m\|v\|\leq \eps\|\alpha v\|.
\]
Conversely, if $\alpha$-Reiter holds, choose $v$ with
\[
  \|\alpha(\pi(s)v-v)\|<\eps mM^{-1}\|\alpha v\|\qquad(s\in S).
\]
Then
\[
  m\|\pi(s)v-v\|\leq \|\alpha(\pi(s)v-v)\|
  <\eps mM^{-1}\|\alpha v\|\leq \eps m\|v\|,
\]
which gives ordinary almost invariance.
\end{proof}

\begin{corollary}[Finite-rank projections]
Let $P$ be an orthogonal projection with finite-dimensional range $W$.  Then
$\pi$ has the operator $P$-Reiter property if and only if, for every finite
$S\subseteq G$, there exists $v\in V$ such that
\[
  Pv\neq0
  \quad\text{and}\quad
  P(\pi(s)v-v)=0\qquad(s\in S).
\]
In particular, for a rank-one projection $P_\xi$ onto the line spanned by
$0\neq\xi\in V$, this is equivalent to
\[
  \xi\notin
  \spanop\{\pi(s)^{-1}\xi-\xi:\ s\in S\}
  \qquad\text{for every finite }S\subseteq G.
\]
\end{corollary}

\begin{proof}
For fixed $S$, define
\[
  L:V\to W\oplus W^S,\qquad
  L(v)=\big(Pv,\ (P(\pi(s)v-v))_{s\in S}\big).
\]
Since the target is finite-dimensional, $L(V)$ is closed.  If the exact
condition holds, then the $P$-Reiter inequalities hold with zero left hand
side.  Conversely, if $P$-Reiter holds, choose $v_n$ with $\|Pv_n\|=1$ and
\[
  \sum_{s\in S}\|P(\pi(s)v_n-v_n)\|^2\to0.
\]
After passing to a subsequence, $Pv_n$ converges to some unit vector
$w\in W$.  Hence $(w,0)$ lies in the closure of $L(V)$ and therefore in
$L(V)$, giving the exact condition.

For the rank-one statement, the equations become
\[
  \langle v,\xi\rangle\neq0,\qquad
  \langle v,\pi(s)^{-1}\xi-\xi\rangle=0\quad(s\in S),
\]
which is possible exactly when $\xi$ does not belong to the span displayed
above.
\end{proof}

\section*{Scope and Novelty Check}

The theorem gives a complete answer to Problem 48 under both common projection
conventions.  It also gives an exact spectral-gap criterion for the general
operator $\alpha$-Reiter side of Problem 47.  The broader group-theoretic
questions in the same source paper about products, supramenability, and
finitely generated exponential-growth examples with the fixed-point property
for cones are separate and are not addressed here.
