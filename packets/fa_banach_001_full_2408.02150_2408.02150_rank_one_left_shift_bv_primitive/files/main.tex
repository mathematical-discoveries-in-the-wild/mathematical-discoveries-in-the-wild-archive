\section*{Open Problem Extract}

In the source paper's example, $X=L^1([0,1])$, $(R(t))_{t\ge 0}$ is the nilpotent right-shift semigroup, and the sun-dual semigroup is the nilpotent left-shift semigroup $(L(t))_{t\ge 0}$ on
\[
  X^\sun=\{b\in C([0,1]): b(1)=0\}.
\]
The source describes the rank-one $C$-admissible controls for $(L(t))_{t\ge 0}$ as
\[
  \{C' : C\in\calL(X_1,\C),\ C'(1)=\partial b=\widpartial c
  \text{ for some } b\in X^\sun,\ c\in\BV([0,1])\}.
\]
Remark 5.4(c) asks whether this description can simply be written as
\[
  C'(1)=\partial b\quad\text{for some }b\in X^\sun\cap \BV([0,1]).
\]

\section*{Proof Intuition}

The only apparent gap between the source description and the proposed simpler one is whether the primitive $b$ must have bounded variation.  But the source already gives a bounded-variation primitive $c$ for the same distribution after passing through the $\widetilde{\partial}$ calculus.  Restricting that equality to ordinary compactly supported test functions says that the distributional derivative of the continuous function $b$ is a finite Radon measure on $(0,1)$.  In one dimension, that is exactly the distributional characterization of bounded variation.

The converse is just integration by parts.  If $b\in X^\sun\cap\BV([0,1])$, then $\partial b$ and $\widpartial b$ agree on the relevant test functions.  The endpoint term at $1$ is killed by $b(1)=0$, while the endpoint term at $0$ is killed by the definition of the $\widetilde{\calD}$ test space.

\section*{Theorem}

\begin{theorem}
With the notation of Arora--Schwenninger, for the nilpotent left-shift semigroup $(L(t))_{t\ge0}$ on
\[
  X^\sun=\{b\in C([0,1]): b(1)=0\},
\]
one has
\[
  \B_{\mathrm C}(\C,X^\sun,(L(t))_{t\ge0})
  =
  \{ C' : C\in\calL(X_1,\C),\ C'(1)=\partial b
  \text{ for some } b\in X^\sun\cap\BV([0,1])\}.
\]
Thus the simplification asked for in Remark 5.4(c) is valid.
\end{theorem}

\section*{Proof}

Let
\[
  \widetilde{\calD}
  =\{\phi\in C^\infty([0,1]):\phi(0)=0\},
  \qquad
  \calD=C_c^\infty((0,1)).
\]
For distributions on $\widetilde{\calD}$, the notation $g=\widpartial f$ means
\[
  \langle g,\phi\rangle=-\langle f,\phi'\rangle
  \qquad(\phi\in\widetilde{\calD}),
\]
whereas $\partial f$ denotes the ordinary distributional derivative on $\calD$.

The source Proposition 5.3 proves that
\[
  \B_{\mathrm C}(\C,X^\sun,(L(t))_{t\ge0})
  =
  \{ C' : C\in\calL(X_1,\C),\ C'(1)=\partial b=\widpartial c
  \text{ for some } b\in X^\sun,\ c\in\BV([0,1])\}.
  \tag{1}
\]
We show that the set on the right of (1) is exactly the simpler set in the theorem.

First suppose that $C'(1)=\partial b=\widpartial c$ as in (1).  Since $c\in\BV([0,1])$, the distribution $\widpartial c$, restricted to $\calD$, is a finite Radon measure on $(0,1)$.  Hence the ordinary distributional derivative of the continuous function $b$ is a finite Radon measure on $(0,1)$.

By the standard one-dimensional characterization of functions of bounded variation, an $L^1_{\mathrm{loc}}$ function whose distributional derivative is a finite Radon measure is a BV function, with that measure as its distributional derivative.  Applying this to $b$ on $(0,1)$ shows that $b$ has bounded variation on $(0,1)$.  Since $b$ is continuous on $[0,1]$, its one-sided endpoint limits are finite and agree with $b(0)$ and $b(1)$; adjoining the endpoints therefore does not add infinite variation.  Thus
\[
  b\in X^\sun\cap\BV([0,1]).
\]
This proves that every control in the source description has the simplified form.

Conversely, suppose that $b\in X^\sun\cap\BV([0,1])$ and $C'(1)=\partial b$.  The BV integration-by-parts formula gives, for every $\phi\in\widetilde{\calD}$,
\[
  -\int_0^1 b(t)\phi'(t)\dd t
  = \int_{[0,1]}\phi(t)\dd Db(t)-b(1)\phi(1)+b(0)\phi(0),
\]
where $Db$ is the finite signed distributional derivative measure of $b$.  Now $b(1)=0$ because $b\in X^\sun$, and $\phi(0)=0$ because $\phi\in\widetilde{\calD}$.  Hence
\[
  \langle\widpartial b,\phi\rangle
  =
  \int_{[0,1]}\phi(t)\dd Db(t).
\]
On compactly supported test functions this is exactly $\partial b$.  Therefore $\partial b=\widpartial b$ in the sense needed in Proposition 5.3.  Taking $c=b\in\BV([0,1])$ in (1), the source proposition gives
\[
  C'\in \B_{\mathrm C}(\C,X^\sun,(L(t))_{t\ge0}).
\]

The two inclusions prove the asserted equality and answer the question in Remark 5.4(c). \qed

\section*{Verification Notes}

\begin{itemize}
\item No numerical or symbolic computation is used.
\item The proof depends only on the source Proposition 5.3 and the standard distributional characterization of one-dimensional BV functions.
\item The boundary terms are the main point: $b(1)=0$ is part of the definition of $X^\sun$, and $\phi(0)=0$ is part of the definition of $\widetilde{\calD}$.
\item This packet answers only the rank-one left-shift description in Remark 5.4(c). It does not settle the broader $p=1$ zero-class-removal problem from Theorem 4.3.
\end{itemize}
