# Partial Packet: uncountable `c0(Gamma)` has no continuous inner product

Status: `partial_result_likely_valid`

Source paper: Jarno Talponen, *Directionally Euclidean Structures of Banach
Spaces*, arXiv:1005.2737.

Source target: Section 2.1 asks what kind of Banach spaces admit a continuous
non-degenerate inner product, after proving that `omega*`-separable dual is a
sufficient condition and that such an inner product gives a continuous
injection into some `c0(Gamma)`.

Result: If `Gamma` is uncountable, then `c0(Gamma)` admits no continuous
non-degenerate inner product. Equivalently, there is no bounded linear
injection from `c0(Gamma)` into any Hilbert space. More generally, every
bounded operator `T:c0(Gamma)->H` into a Hilbert space kills all but countably
many unit vectors.

Scope: This does not classify all spaces admitting continuous inner products
and does not settle the DES inheritance questions. It gives a natural
non-example and shows that the source's necessary `c0(Gamma)`-injection
condition is not sufficient.
