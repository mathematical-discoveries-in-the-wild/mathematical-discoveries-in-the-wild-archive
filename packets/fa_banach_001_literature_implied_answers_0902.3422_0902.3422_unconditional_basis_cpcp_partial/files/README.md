# Literature-Implied Partial Subcase: Unconditional-Basis CPCP Consequence

status: `literature_implied_answer (partial subcase)`

Source target: arXiv:0902.3422, Gines Lopez Perez, "Banach spaces with many boundedly complete basic sequences failing PCP."  The relevant open-ended question is Remark 2.7(iii), on page 8 of the arXiv PDF: does a Banach space satisfy CPCP whenever every semi-normalized basic sequence has a boundedly complete subsequence?

Supporting literature: arXiv:2302.11287, Cleon S. Barroso, "Fixed point property, unconditional basis and Banach spaces containing no copies of c0."  Remark 4.15, page 11, records the known implication, citing Schachermayer [52, Corollary 2.11], that a Banach space with an unconditional basis which fails CPCP contains a `c_0`-sequence.

Identification: In the unconditional-basis subcase, the source property rules out `c_0`-sequences, since the unit vector basis of `c_0` has no boundedly complete subsequence. Therefore any Banach space with an unconditional basis and the source sequential property must satisfy CPCP.

Scope limitation: This is not a full answer to Remark 2.7(iii). It only records the unconditional-basis subcase. The unrestricted question remains open from this packet's search and direct-attack pass.
