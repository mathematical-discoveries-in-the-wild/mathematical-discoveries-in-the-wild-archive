# 2007.13474 - Expected p=2 rate via an adapted Hilbert scale

Status: partial_likely_valid_strengthened

## Source

- arXiv: 2007.13474
- Title: *Variational source conditions in Lp-spaces*
- Authors: De-Han Chen and Irwin Yousept
- Open signal: Remark 4.9 says that in Theorem 4.6(iv), for `p=2`, one may
  expect the rate `||a^delta_alpha - a^\dag||_2^2 = O(delta^{2s/(s+2)})`;
  the paper obtains a weaker rate and leaves the issue as an open question.

## Result

The packet proves a partial positive result. For the elliptic coefficient
inverse problem in the `p=2` penalty case, the varying coefficients may remain
in the paper's original `L^\mathfrak p` admissible class, with
`\mathfrak p >= 2`. If the true state is bounded above and away from zero, and
one measures in `L^2` (or in any `L^\tau` controlling `L^2` on the bounded
domain), then the PDE identity gives a linear stability estimate in an adapted
`X_{-2}` Hilbert-scale norm:

```text
||a^\dag-a||_{X_{-2}} <= C ||S(a)-S(a^\dag)||_2.
```

Combining this with the standard Hilbert-scale splitting argument yields the
expected variational source condition and therefore the rate

```text
||a^delta_alpha - a^\dag||_2^2 = O(delta^{2s/(s+2)}).
```

## Scope and Limitations

This is not a full unconditional solution of the source-paper open direction.
The smoothness assumption is stated in the adapted Hilbert scale generated by
`u^\dag D(A_0)`, where `A_0=-div(kappa grad)+1` is the fixed Neumann operator.
Under a smooth positive true state, this scale is the natural Neumann/elliptic
counterpart of an `H^s` scale, but translating it under only the source paper's
minimal true-state assumptions remains the main barrier.

The strengthened packet improves the previous version by removing the
artificial `L^\infty` bound on the varying coefficients. It still assumes the
true state is a bounded multiplier with bounded reciprocal.
