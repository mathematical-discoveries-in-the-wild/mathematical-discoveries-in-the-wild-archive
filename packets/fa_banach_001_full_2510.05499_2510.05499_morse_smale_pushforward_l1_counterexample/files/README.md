# A Morse–Smale counterexample to the Axiom-A pushforward conjecture

Status: `candidate_full_likely_valid`

Source: Sergey Tikhomirov, *Generalized `(C, lambda)`-structure for
nonlinear diffeomorphisms of Banach spaces*, arXiv:2510.05499,
Conjecture 1 on page 14.

## Full negative resolution

The source conjectures that if a diffeomorphism `h` of a finite-dimensional
manifold satisfies Axiom A and the Strong Transversality Condition, then its
pushforward

`(S_h v)(h(x)) = Dh(x)v(x)`

is generalized hyperbolic on every `L^p(M,TM)`, `1 <= p <= infinity`, and on
continuous vector fields.

The packet constructs a smooth Morse–Smale diffeomorphism `h` of a closed
surface for which `S_h` is **not** generalized hyperbolic on `L^1(M,TM)`.
Since Morse–Smale diffeomorphisms satisfy Axiom A and strong transversality,
this disproves Conjecture 1.

## Counterexample mechanism

Choose `h` as the time-one map of a Morse–Smale gradient flow with a saddle
at which, in Euclidean local coordinates,

`h(x,y) = (x/2, 4y)`.

For the `L^1` pushforward, the adjoint on bounded covector fields is

`(S_h^* psi)(x) = J_h(x) Dh(x)^* psi(h(x))`.

In the first coordinate covector direction the local multiplier is exactly

`det(Dh) * (1/2) = 2 * (1/2) = 1`.

Arbitrarily long disjoint orbit towers fit inside the linear saddle chart.
Putting triangular coefficients on such a tower produces covector fields
`psi_N` satisfying

`||psi_N||_infinity = 1`,
`||(S_h^*-I)psi_N||_infinity <= 1/N`.

Thus `S_h^*-I` is not bounded below and `S_h-I` is not surjective.

On the other hand, every generalized hyperbolic isomorphism `T` makes `I-T`
surjective: if `P,Q` are its stable/unstable projections, the convergent
Green series

`G y = sum_(n>=0) T^n P y - sum_(n>=1) T^(-n) Q y`

satisfies `(I-T)G=I`. This contradiction proves the result.

## Novelty check and limitation

A bounded arXiv/web search on 2026-07-21 used the phrases `pushforward vector
fields generalized hyperbolic Lp Axiom A`, `S_h generalized hyperbolic vector
fields`, `Axiom A pushforward operator vector fields Lp shadowing`, and
`Morse-Smale pushforward operator Lp vector fields spectrum`. It found related
work on weighted operators generated by Morse–Smale maps (arXiv:1303.1293)
and generalized hyperbolicity on `L^p` (arXiv:2009.11526), but no statement of
this Jacobian-neutral saddle obstruction. Novelty confidence is moderate.

The counterexample settles only the universal conjecture: it uses `p=1` and
does not classify which Axiom-A maps or which other `p` yield generalized
hyperbolicity.

## Human review recommendation

Check the standard relative Morse–Smale realization of the prescribed local
saddle, the change-of-variables formula for `S_h^*`, and the implication
“surjective operator implies adjoint bounded below.” The rest is exact
telescoping.
