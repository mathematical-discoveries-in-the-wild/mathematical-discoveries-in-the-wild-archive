  T:\ellone\longrightarrow \ellinf,\qquad

\section*{Claim}

\begin{theorem}
Let \(T:\ellone\to\ellinf\) be defined by
\[
  T x = \Big(\sum_{k=1}^\infty x_k\Big)\one,
  \qquad \one=(1,1,\ldots).
\]
Then \(T\) is disjoint weak-star Dunford-Pettis.
\end{theorem}

\begin{proof}
Let \((x_n)\) be a norm-bounded disjoint sequence in \(\ellone\), and let
\((f_n)\subset(\ellinf)'\) satisfy \(f_n\to 0\) in the weak-star topology
\(\sigma((\ellinf)',\ellinf)\). Put
\[
  \sigma(x)=\sum_{k=1}^\infty x_k,\qquad x\in\ellone .
\]
Then \(\sigma\in(\ellone)'\), so the scalar sequence \((\sigma(x_n))\) is
bounded. Also, since \(\one\in\ellinf\) and \(f_n\to0\) weak-star,
\[
  f_n(\one)\longrightarrow 0 .
\]
Therefore
\[
  f_n(Tx_n)
  =
  f_n(\sigma(x_n)\one)
  =
  \sigma(x_n) f_n(\one)
  \longrightarrow 0 .
\]
This is precisely the definition of a disjoint weak-star Dunford-Pettis
operator.
\end{proof}

\begin{remark}
The displayed calculation in the source cannot produce a counterexample:
any weak-star null sequence in \((\ellinf)'\) must vanish on the constant
vector \(\one\). In particular, coordinate functionals \(e_n\) are not
weak-star null in \((\ellinf)'\), since \(e_n(\one)=1\) for every \(n\).
The argument above only refutes the printed assertion about this particular
rank-one operator; it does not address whether some different example can
serve the intended purpose.
\end{remark}
