# Literature-Implied Partial Subcase: Midpoint Nontriviality for Kothe Couples

Status: `literature_implied_answer (partial subcase)`

Source problem paper: W. H. G. Correa, *Type, cotype and twisted sums induced
by complex interpolation*, arXiv:1701.07084.

Supporting paper: J. M. F. Castillo, W. H. G. Correa, V. Ferenczi, and
M. Gonzalez, *Stability of the differential process generated by complex
interpolation*, arXiv:1712.09647.

## Target

In Section 5, Final Remarks, Correa asks whether
`dX_{1/2}` must be nontrivial if `dX_theta` is nontrivial for every
`theta in (0,1)`, `theta != 1/2`.

## Result

For interpolation pairs of superreflexive Kothe function spaces, the answer is
yes. This is not a new theorem of this run; it is an agent-identified
implication of Theorem 4.3 in arXiv:1712.09647.

## Identification

Correa identifies `dX_theta` with the exact sequence generated by the
derivation `Omega_theta`. The supporting Theorem 4.3 says that for a pair
`(X_0,X_1)` of superreflexive Kothe function spaces and any `0 < theta < 1`,
`Omega_theta` is trivial if and only if `X_1 = X_0(w)` up to equivalence of
norms for some weight function `w`.

The condition `X_1 = X_0(w)` is independent of `theta`. Therefore, if
`Omega_{1/2}` were trivial, Theorem 4.3 would force the same weighted-endpoint
condition and hence `Omega_theta` would be trivial for every `theta`. This
contradicts nontriviality for every `theta != 1/2`.

## Scope

This packet covers the superreflexive Kothe-pair subcase, including the
standard `ell_p`/`L_p`-type setting that motivated the source question. It does
not answer the arbitrary Banach-space interpolation-pair problem. The
supporting paper itself notes that local/global stability for general Banach
space couples remains open.
