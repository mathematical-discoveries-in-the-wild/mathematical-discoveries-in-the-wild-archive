\section*{Status}

This packet proves a partial result for the question following Corollary 6.7
of Kaminska--Popov--Spinu--Tcaciuc--Troitsky~\cite{KPSTT}.  The full question
for arbitrary strictly singular operators $\ell_p\to d_{w,p}$, $p>1$, remains
open here.

\section*{Source Question}

Let $j:\ell_p\to d_{w,p}$ be the formal identity.  After proving that
$\SSing(\ell_1,d_{w,1})$ is contained in the closure of operators factoring
through $j$, the source asks whether the same assertion remains valid for
$p>1$.

\section*{Proof Intuition}

The source proves the key column condition: if
$T:\ell_p\to d_{w,p}$ is strictly singular, then the columns $(Tf_n)$ are
almost lengthwise bounded.  For arbitrary operators this is not by itself
enough, because truncating columns does not automatically give a bounded
$\ell_p$-valued operator.  In the block-diagonal case, however, the supports
are disjoint.  The truncated columns are uniformly $N$-sparse and disjoint, so
they define a bounded operator on $\ell_p$.  The discarded tails are also
disjoint and uniformly small in $d_{w,p}$, so the Lorentz upper $p$-estimate
makes the tail operator small in norm.

\section*{Definitions}

Let $1<p<\infty$, and let $w=(w_n)$ be a Lorentz weight:
$w_1=1$, $w_n\downarrow0$, $w_n>0$, and $\sum_n w_n=\infty$.  For
$x\in c_0$, write $x^*$ for the non-increasing rearrangement of
$(|x_n|)$, and
\[
  \norm{x}_{d_{w,p}}^p=\sum_{n=1}^{\infty}w_n(x_n^*)^p.
\]
We say that $T:\ell_p\to d_{w,p}$ is block diagonal if, for the unit vector
basis $(f_n)$ of $\ell_p$, the vectors $y_n=Tf_n$ have pairwise disjoint
successive supports in the unit vector basis of $d_{w,p}$.

\begin{lemma}[Disjoint upper $p$-estimate]\label{lem:upper}
If $(x_n)$ are pairwise disjointly supported vectors in $d_{w,p}$, then for
all finitely supported scalar sequences $(a_n)$,
\[
  \norm{\sum_n a_nx_n}_{d_{w,p}}^p
  \leq
  \sum_n |a_n|^p\norm{x_n}_{d_{w,p}}^p .
\]
\end{lemma}

\begin{proof}
It is enough to consider non-negative finitely supported vectors.  Rearrange
the entries of each $x_n$ in decreasing order, and compare them with the
global decreasing rearrangement of the disjoint union
$\sum_n a_nx_n$.  If the $m$-th largest entry of a fixed block appears at
global position $k$, then $k\geq m$, hence $w_k\leq w_m$.  Summing over all
entries gives the stated inequality.
\end{proof}

\section*{Partial Theorem}

\begin{theorem}
Let $1<p<\infty$.  Let $T:\ell_p\to d_{w,p}$ be a strictly singular
block-diagonal operator.  Then
\[
  T\in \overline{\{jS:S\in L(\ell_p)\}}^{\ \norm{\cdot}_{L(\ell_p,d_{w,p})}},
\]
where $j:\ell_p\to d_{w,p}$ is the formal identity.
\end{theorem}

\begin{proof}
Put $y_n=Tf_n$.  By Corollary 6.4 of~\cite{KPSTT}, the sequence $(y_n)$ is
almost lengthwise bounded.  Thus, for each $\varepsilon>0$, there is
$N\in\N$ such that after removing the $N$ largest coordinates of each $y_n$,
the remaining vector has $d_{w,p}$-norm at most $\varepsilon$.

For each $n$, let $z_n$ be the vector obtained from $y_n$ by keeping the
$N$ largest coordinates and setting all other coordinates to zero.  Since
$T$ is block diagonal, the supports of the $z_n$ are disjoint.  Also,
$z_n$ is $N$-sparse and
\[
  \norm{z_n}_{d_{w,p}}\leq \norm{y_n}_{d_{w,p}}\leq \norm{T}.
\]
For an $N$-sparse vector $z$ one has
\[
  \norm{z}_{d_{w,p}}^p
  \geq
  w_N\norm{z}_{\ell_p}^p,
\]
because the first $N$ Lorentz weights are all at least $w_N$.  Hence
$\sup_n\norm{z_n}_{\ell_p}\leq w_N^{-1/p}\norm{T}$.

Define $S_N:\ell_p\to\ell_p$ by $S_Nf_n=z_n$.  Since the supports of the
$z_n$ are disjoint,
\[
  \norm{S_N(a_n)}_{\ell_p}^p
  =
  \sum_n |a_n|^p\norm{z_n}_{\ell_p}^p
  \leq
  w_N^{-1}\norm{T}^p\sum_n |a_n|^p,
\]
so $S_N\in L(\ell_p)$.

Now set $r_n=y_n-z_n$.  The supports of the $r_n$ are still disjoint, and
$\norm{r_n}_{d_{w,p}}\leq\varepsilon$ for every $n$.  By
Lemma~\ref{lem:upper}, for every finitely supported $(a_n)\in\ell_p$,
\[
  \norm{(T-jS_N)(a_n)}_{d_{w,p}}^p
  =
  \norm{\sum_n a_nr_n}_{d_{w,p}}^p
  \leq
  \varepsilon^p\sum_n |a_n|^p .
\]
Thus $\norm{T-jS_N}_{L(\ell_p,d_{w,p})}\leq\varepsilon$.  Since
$\varepsilon>0$ was arbitrary, $T$ belongs to the stated norm closure.
\end{proof}

\section*{Scope And Novelty Check}

This proves the source question only for block-diagonal operators.  The
general case remains open here because almost lengthwise boundedness of the
columns does not, for $p>1$, automatically make the column truncation map
bounded from $\ell_p$ to $\ell_p$.

Local run indexes were searched for arXiv:1108.6026 and for keywords around
Lorentz sequence spaces, norm-closed operator ideals, formal identity,
strictly singular operators, and $\ell_p$ factorization.  A web search on
2026-07-03 found the source paper but no exact later answer to the $p>1$
question.  The result should be reviewed as a natural partial subcase, not as
a full resolution of Conjecture 6.1 or the full question after Corollary 6.7.
