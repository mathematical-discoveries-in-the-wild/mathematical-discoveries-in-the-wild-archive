# Literature-Already-Answered: Modular Semilattice Orthoscheme Complexes

Status: `literature_already_answered`

Source paper: Miroslav Bacak, *Old and new challenges in Hadamard spaces*, arXiv:1807.01355.

Source question: In the section on submodular functions on modular lattices, the paper reproduces Chalopin--Chepoi--Hirai--Osajda Conjecture 7.3: if `L` is a modular semilattice of finite rank, is the orthoscheme complex `K(L)` a CAT(0) space? Hirai later identifies this as Bacak Problem 6.10.

Answer: Yes. Hiroshi Hirai, *A Nonpositive Curvature Property of Modular Semilattices*, arXiv:1905.01449 / Geometriae Dedicata 214 (2021), proves the conjecture affirmatively. Theorem 1.2 states that if `L` is a modular semilattice of finite rank, then `K(L)` is CAT(0).

Why this is an exact match: The source question and Hirai's theorem use the same hypotheses and conclusion: finite-rank modular semilattice, orthoscheme complex, CAT(0). Hirai's introduction explicitly says Chalopin et al. conjectured this and adds "see also [Problem 6.10]" of Bacak's survey.

Scope limitations: This only clears the modular-semilattice orthoscheme-complex problem. The other open problems in Bacak's Hadamard-space survey, including weak topologies, compact convex hulls, stochastic PPA without local compactness, and cyclic projection asymptotic regularity, are not addressed here.
