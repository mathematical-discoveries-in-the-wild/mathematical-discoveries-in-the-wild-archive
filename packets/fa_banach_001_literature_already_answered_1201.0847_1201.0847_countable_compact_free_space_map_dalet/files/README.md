# Literature Answer: Countable Compact Free Spaces Have MAP

Status: `literature_already_answered`

Source problem: Godefroy and Ozawa, *Free Banach spaces and the approximation
properties*, arXiv:1201.0847. In the closing list of questions, Question 2 asks:
"Is there a countable compact space \(K\) such that \(\mathfrak F(K)\) fails
(AP)?"

Supporting answer: Dalet, *Free Spaces over Countable Compact Metric Spaces*,
arXiv:1307.0735. The abstract and Theorem 3.1 prove that if \(K\) is a
countable compact metric space, then \(\mathcal F(K)\) has the metric
approximation property. The introduction explicitly says this gives a negative
answer to Question 2 in Godefroy--Ozawa.

Nearby subcase marker: Dalet, *Free spaces over some proper metric spaces*,
arXiv:1404.3939, Theorem 2.6 proves MAP for countable proper metric spaces.
Thus the locally finite/proper uniformly discrete subcase of Kalton's
uniformly-discrete BAP question from the same source is also covered by known
literature. The arbitrary uniformly discrete problem is not settled here; it is
still identified as open in Cuth--Kalenda, arXiv:2604.01875.
