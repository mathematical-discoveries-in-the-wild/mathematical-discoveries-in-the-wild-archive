## Claim

The global supremum-norm strengthening suggested in Remark 2.2 of arXiv:2109.13512 cannot hold for the paper's own rank-one bounded sigmoid activation
\[
\sigma(x)=\beta(\psi(x))z,
\]
where \(\psi\in X'\setminus\{0\}\), \(z\neq 0\), and \(\beta\in C_b(\mathbb R)\) has finite limits at \(+\infty\) and \(-\infty\) (in particular the paper's example with limits \(1\) and \(0\)).

For every finite network generated by this \(\sigma\), the restriction to any affine ray \(t\mapsto tv\) has a finite limit as \(t\to\infty\). The bounded continuous function
\[
f(x)=\sin(\varphi(x)^2)
\]
for a continuous linear functional \(\varphi\) with \(\varphi(v)=1\) has no such limit along the same ray. Therefore the network span is not dense in \(C_b(X;\mathbb R)\) with the supremum norm; in fact every network remains at distance at least \(1\) from this \(f\).

## Scope

This does not affect the source paper's compact-open universal approximation theorem. It is a negative result for the natural global `C_b(X)` version under the source's concrete one-coordinate sigmoid activation class.
