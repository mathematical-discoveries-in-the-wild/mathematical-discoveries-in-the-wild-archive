# fsJNP Discrete Supports Answered By arXiv:2303.03809

Status: literature_already_answered.

Source/open-problem paper: Jerzy Kakol, Damian Sobota, and Lyubomyr
Zdomskyy, *The Josefson--Nissenzweig theorem, Grothendieck property, and
finitely supported measures on compact spaces*, arXiv:2009.07552.

Source question: Question 4.23 asks whether every Tychonoff space with the
finitely supported Josefson--Nissenzweig property admits an fsJN-sequence whose
union of supports is a discrete subspace.

Supporting answer paper: Witold Marciszewski, Damian Sobota, and Lyubomyr
Zdomskyy, *On sequences of finitely supported measures related to the
Josefson--Nissenzweig theorem*, arXiv:2303.03809.

Answer: yes. Theorem 1.2 of arXiv:2303.03809 states that whenever a Tychonoff
space carries a finitely supported JN-sequence, it carries one with pairwise
disjoint supports and with discrete union of supports. The proof is assembled
from Theorem 4.3 and Theorem 5.4 in the same paper.
