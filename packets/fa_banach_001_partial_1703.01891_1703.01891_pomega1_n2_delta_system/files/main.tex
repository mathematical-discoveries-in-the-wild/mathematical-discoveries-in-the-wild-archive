\section*{Source and Target}

The source paper is Andrew Swift, \emph{Coarse embeddings into \(c_0(\Gamma)\)}, arXiv:1703.01891.  In Section 3 Swift defines \(P(\alpha)\): for every \(n\in\mathbb N\) and every coloring
\[
  \sigma:[\alpha]^n\to\cC,
\]
at least one of the following alternatives holds:
\begin{enumerate}
\item there is a sequence \((F_j)_{j=1}^\infty\) of pairwise disjoint members of \([\alpha]^n\) such that \(\sigma(F_i)=\sigma(F_j)\) for all \(i,j\);
\item there is \(F\in[\alpha]^{n-1}\) such that
\[
  \sigma\bigl(\{F\cup\{\gamma\}:\gamma\in\alpha\setminus F\}\bigr)
\]
is infinite.
\end{enumerate}
Problem 2 in Section 5 asks whether Theorem B is true for \(\omega_1\), i.e. whether \(P(\omega_1)\) holds.

\section*{Result}

This packet proves the first nontrivial arity case.  It is a partial result only: the full question requires all finite \(n\).

\begin{theorem}
The \(n=2\) instance of \(P(\omega_1)\) holds in ZFC.  Equivalently, for every set of colors \(\cC\) and every map
\[
  \sigma:[\omega_1]^2\to\cC,
\]
either there are pairwise disjoint two-element sets \(F_1,F_2,\ldots\in[\omega_1]^2\) with \(\sigma(F_i)\) constant, or there is an ordinal \(\alpha<\omega_1\) such that
\[
  \{\sigma(\{\alpha,\beta\}):\beta<\omega_1,\ \beta\ne\alpha\}
\]
is infinite.
\end{theorem}

\section*{Proof Intuition}

The obstruction to the second alternative says that every vertex of the complete graph on \(\omega_1\) sees only finitely many colors.  Attach to each vertex the finite set of colors incident to it.  Since every edge has a color visible at both of its endpoints, these finite sets are pairwise intersecting.  The Delta-system lemma turns an uncountable family of pairwise-intersecting finite sets into an uncountable family with a fixed nonempty finite root.  On the corresponding uncountable set of vertices, every edge color lies in that one finite root.  The ordinary finite-color Ramsey theorem on a countably infinite subset then gives an infinite monochromatic clique, and hence an infinite monochromatic matching.

\section*{Proof}

Assume that the second alternative fails.  Then for each \(\alpha<\omega_1\) the incident color set
\[
  C_\alpha=\{\sigma(\{\alpha,\beta\}):\beta<\omega_1,\ \beta\ne\alpha\}
\]
is finite.

For distinct \(\alpha,\beta<\omega_1\), the edge color \(\sigma(\{\alpha,\beta\})\) belongs to both \(C_\alpha\) and \(C_\beta\).  Hence
\[
  C_\alpha\cap C_\beta\ne\varnothing
\]
for all distinct \(\alpha,\beta\).

Apply the Delta-system lemma to the uncountable family \((C_\alpha)_{\alpha<\omega_1}\) of finite sets.  There are an uncountable set \(A\subset\omega_1\) and a finite root \(R\) such that
\[
  C_\alpha\cap C_\beta=R
\]
whenever \(\alpha,\beta\in A\) are distinct.  Since the family is pairwise intersecting, \(R\ne\varnothing\).  Moreover, for all distinct \(\alpha,\beta\in A\),
\[
  \sigma(\{\alpha,\beta\})\in C_\alpha\cap C_\beta=R.
\]
Thus the complete graph on \(A\) is colored using only the finite color set \(R\).

Choose a countably infinite subset \(B\subset A\).  By the infinite Ramsey theorem for finite colorings of pairs, there are an infinite subset \(B_0\subset B\) and a color \(r\in R\) such that
\[
  \sigma(\{\alpha,\beta\})=r
\]
for all distinct \(\alpha,\beta\in B_0\).  Enumerate \(B_0=\{\gamma_0,\gamma_1,\gamma_2,\ldots\}\), and set
\[
  F_j=\{\gamma_{2j},\gamma_{2j+1}\}\qquad(j=0,1,2,\ldots).
\]
The \(F_j\)'s are pairwise disjoint two-element subsets of \(\omega_1\), and all have color \(r\).  This is the first alternative of \(P(\omega_1)\) for \(n=2\).
\qed

\section*{Verification Notes}

The proof uses only two standard ZFC combinatorial facts: the Delta-system lemma for an uncountable family of finite sets and the ordinary infinite Ramsey theorem for finite colorings of pairs on a countable set.  The conclusion matches exactly the \(n=2\) specialization of Swift's property \(P(\alpha)\).

The argument does not prove the \(n\ge3\) cases.  For \(n=3\), for example, failure of the second alternative gives finite color sets on pairs, while a triple color must be compatible with all three of its pair-faces.  The simple vertex-root Delta-system reduction above no longer directly produces a finite global color set on an uncountable vertex set.
