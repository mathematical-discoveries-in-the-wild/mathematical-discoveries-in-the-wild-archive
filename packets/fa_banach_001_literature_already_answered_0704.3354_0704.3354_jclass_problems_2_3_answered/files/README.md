# 0704.3354 J-class Problems 2 and 3 answered

Status: `literature_already_answered` for Problems 2 and 3 only.

Source paper: G. Costakis and A. Manoussos, *J-class operators and hypercyclicity*, arXiv:0704.3354.

## Identification

The source paper lists five open problems in Section 6. This packet records exact later answers to:

- Problem 2: whether every non-separable infinite-dimensional Banach space supports a J-class operator.
- Problem 3: whether there is a spectral description of the norm closure of the set of J-class operators on a Hilbert space.

## Answer Evidence

Problem 2 is answered negatively by A. B. Nasseri, *On the existence of J-class operators on Banach spaces*, arXiv:1009.3461. The abstract explicitly says it answers the Costakis-Manoussos question in the negative, and Theorem 1.1 constructs a non-separable complex Banach space with no J-class operator.

Problem 3 is answered by G. Tian and B. Hou, *Limits of J-class operators*, arXiv:1010.3386. The abstract and introduction explicitly say they answer the Costakis-Manoussos third problem, and Theorem 2.10 gives the spectral description of `closure(J(H))`.

## Scope

This packet does not answer source Problems 1, 4, or 5. A bounded direct attack did not produce a proof or counterexample for those live questions.
