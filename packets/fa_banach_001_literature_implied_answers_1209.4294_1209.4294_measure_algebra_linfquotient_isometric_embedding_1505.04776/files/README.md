# Literature-implied answer: measure-algebra `C(K)` embeds isometrically into `ell_infty/c0`

Status: `literature_implied_answer (full answer to Question 6.8 subquestion)`

Source/open-problem paper: arXiv:1209.4294, Piotr Koszmider,
*Universal objects and associations between classes of Banach spaces and
classes of compact spaces*.

Supporting paper: arXiv:1505.04776, Christina Brech and Piotr Koszmider,
*An isometrically universal Banach space induced by a non-universal Boolean
algebra*.

## Identification

Question 6.8 in the source paper asks whether it is consistent that, for `K`
the Stone space of the measure algebra, `C(K)` does **not** isometrically embed
into `ell_infty/c0`.

The 2015 supporting paper states in its introduction that, although OCA can
make the measure algebra fail to embed into `P(N)/Fin`, it is well known that
`C(K_M) = L_infty` embeds isometrically into
`C(K_{P(N)/Fin}) = ell_infty/c0`, citing Amir--Arbel.

Therefore the source question has a negative ZFC answer: the proposed
consistent failure cannot occur, because the isometric embedding exists.

## Scope

This packet only answers the measure-algebra `C(K)` isometric-embedding
subquestion, Question 6.8. It does not settle the broader ZFC equivalence
questions in arXiv:1209.4294 about universal Banach spaces, isometrically
universal Banach spaces, and universal compact spaces.
